# 跨境电商运营速查

## 亚马逊运营

**COSMO算法**
亚马逊COSMO算法强调产品理解和内容质量比运营技巧更重要。优化Listing内容深度后，自然流量增长明显，纯靠黑科技打法逐渐失效。

**AMC精准营销 / RUFUS AI搜索**
AMC（Amazon Marketing Cloud）支持受众细分和归因分析，实现精准营销。RUFUS是亚马逊AI购物助手，卖家需针对AI问答场景优化Listing结构和内容。

**广告三阶段打法与ABA数据**
广告投放遵循"认知→考量→转化"三阶段：认知期用SP广泛匹配和SD拉新，考量期用SP词组匹配蓄水，转化期用SP精准匹配和SB收割。结合ABA（亚马逊品牌分析）搜索词报告，持续优化关键词和竞价策略。

**Listing优化提示词工作流**
完整流程为：确认优化需求→采集后台数据（搜索词、转化率）→VOC差评分析→竞品文案拆解→用AI生成至少3个版本→合规检查后上线测试。多版本A/B测试比单次修改更有效。

**差评痛点分类 / 违禁词替换**
将差评按"质量、功能、包装、物流、客服"五大维度分类统计，高频痛点反向驱动产品迭代。Listing文案中的违禁词（如医疗宣称、绝对化用语）需建立替换词库，避免触发审核。

**FBA库存FC Processing问题**
FBA货件长时间处于"FC Processing"状态，常见原因是危险品审核（Hazmat Review）。排查路径：后台合规性工具→危险品状态查询→提交SDS或豁免表，同时开Case催促。

**出口报关9810**
9810是跨境电商出口海外仓的监管代码，适用于FBA和第三方海外仓模式。需在单一窗口完成企业备案，货物离境后凭报关单申请出口退税。

**API安全与关联风险**
使用第三方ERP或工具时，API密钥泄露可能导致店铺关联封号。建议使用IAM角色授权而非根密钥，定期轮换密钥，不同店铺使用独立的开发者账号。

**图片像素规格**
亚马逊主图要求纯白背景（RGB 255），建议像素不低于1600×1600以支持缩放。辅图场景图建议2000×2000像素，信息图文字高度不低于16px。

**Size Value锁定解决方法**
变体Size属性被锁定无法编辑时，需先删除该变体，等待24小时系统刷新后重新上传。若仍无法解决，开Case让客服后台刷新变体主题（Variation Theme）。

## TikTok Shop

**视频播放量为0**
TikTok Shop视频发布后播放量为0，部分原因是平台服务器迁移导致数据延迟。若持续24小时无播放，需排查内容是否触发限流或账号是否有违规记录。

**GMV MAX / 智能推广 / 本地动态功能**
GMV MAX是TikTok Shop的全托管广告产品，系统自动分配预算至短视频、直播、商城等场景。智能推广支持一键投放，本地动态功能（Local Dynamic Ads）可根据用户地理位置推送附近商品。

**东南亚AIGC内容新规**
TikTok东南亚站更新政策，要求AIGC生成内容必须标注"AI生成"标签。违规未标注的账号可能面临限流或下架处理，卖家需在发布流程中主动勾选声明。

**清洁小家电美区爆发案例**
美国市场清洁类小家电（手持吸尘器、蒸汽拖把等）在TikTok Shop爆发增长。核心打法是UGC短视频种草+达人挂车分销，配合GMV MAX放量，单品月销可达数十万美金。

## 独立站

**建站门槛与全域流量布局**
AI工具大幅降低了独立站建站门槛，模板+AI填充即可快速上线。但建站容易获客难，必须做全域流量布局（SEO、社媒、付费广告、EDM、KOL），单一渠道依赖风险极高。

**品牌感打造**
品牌感=底层定位+视觉语言的一致性+AI辅助内容生成。先明确品牌调性和目标人群，再统一色彩、字体、摄影风格，用AI批量生成符合调性的文案和素材。

**Shopify建站 + AI Toolkit**
Shopify内置AI工具包可辅助生成产品描述、邮件主题和页面文案。结合第三方AI插件（如ChatGPT集成），可实现从选品到上架的全流程提效。

**WordPress + Claude批量上产品**
三种方法：①用Claude生成CSV数据表，通过WooCommerce批量导入；②写Python脚本调用Claude API批量生成产品描述后推送到WordPress；③用WP-CLI配合Claude直接操作数据库批量创建。方法一适合非技术卖家，方法二效率最高。

**EDM邮件营销体系**
核心流程：采集→分层→自动化触发→转化。用弹窗/Klaviyo采集邮箱，按购买阶段分 welcome / browse abandon / cart abandon / post-purchase 四类自动化流，复购贡献可达总营收20%-30%。

**收款方案**
主流收款方案为Stripe和连连支付。注册美国公司开Stripe费率更低、风控更宽松，优于香港公司主体。LLC注册成本约200-500美金，Delaware和Wyoming是热门注册州。

**UpPromote分销插件 / Shop App渠道限制**
UpPromote是Shopify热门分销/联盟营销插件，支持自定义佣金比例和二级分销。Shop App渠道对部分品类（如成人用品、电子烟）有限制，需提前确认合规性。

**巴基斯坦写手利弊**
巴基斯坦写手价格低（$3-8/篇），适合批量内容填充。但英语表达常有语法问题，需搭配AI润色或Editor审核，适合预算有限且对内容质量要求中等的场景。

## 选品与SEO

**去运营化趋势**
行业趋势是"去运营化"，将80%精力放在产品开发和供应链上，20%精力做运营。好的产品自带流量，运营只能锦上添花，无法拯救差产品。

**AI选品数据精准化**
结合Sorftime的数据筛选能力和卖家精灵的MCP（Model Context Protocol）接口，可实现AI驱动的自动化选品。通过自然语言描述需求，AI自动筛选符合条件的产品池并输出分析报告。

**爆款短视频拆解→脚本复刻**
方法论：找到目标类目爆款短视频→逐秒拆解（开头钩子、痛点展示、产品演示、CTA引导）→提取脚本框架→用AI生成同类目变体脚本→批量拍摄发布。关键在于复刻结构而非照搬内容。

## 平台动态

**Etsy中国店铺关停**
Etsy对中国大陆店铺进行大规模关停清理，主要涉及无实体手工制作证明的转售卖家。仍在运营的卖家需确保手工属性证明材料完整，或考虑迁移至Shopify独立站。

**海南封关与跨境**
海南自贸港2025年底封关运作后，将实施"零关税、低税率、简税制"政策。跨境电商企业可关注海南作为仓储和进出口节点的机会，降低综合物流和税务成本。

**跨境品牌案例库资源**
社群共享的跨境品牌案例库，收录了各品类DTC品牌从0到1的完整路径。包括品牌定位、流量打法、产品迭代节奏等关键信息，可作为选品和品牌建设的参考素材。

## 税务与合规

**核定征收各地不一致**
个体户核定征收政策各地执行标准不统一，部分园区综合税负率可低至1%-2%，但政策稳定性存疑。建议提前与当地税务局确认最新政策，避免追缴风险。

**税务策略：店铺拆分 vs 出口退税**
两种主流税务策略：①店铺拆分至多个小规模纳税人，享受增值税免征额；②走正规出口退税流程，退税率可达13%。选择取决于销售额规模和供应链票证完善程度，大卖家建议走退税路线。

**创业合伙风险防范**
合伙做跨境电商需提前约定：股权比例、决策权、退出机制、账号资产归属。建议签订书面合伙协议，避免口头约定。尤其注意店铺账号和品牌商标的权属登记，拆伙时是最大争议点。
