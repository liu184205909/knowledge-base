## 一、SEO核心技术

### 1.1 子域名 vs 子目录 vs 新域名

**核心原则**：不同主题关键词互不关联时，用子域名互相隔离更好。

- 子域名在搜索引擎看来是全新网站，每个子域名都有一个首页可拿下核心关键词
- Fandom.com 和 Comic.Studio 的做法：不同子站功能代码一致，仅主题/内容不同，后台配置即可分钟级上线新子站
- **站群模式**：多个域名解析到同一套程序，后台配置即可快速发布新网站

**竞争升级路径**：
```
别人用一个页面覆盖多词 → 我们为每个关键词建一个页面
别人用内页 → 我们用子目录
别人用子目录 → 我们用子域名
别人用子域名 → 我们用主域名
```

### 1.2 Canonical + 307 策略（consensus.app 案例）

consensus.app 做了一次极其精妙的 URL 迁移：
1. 原先首页是落地页，实际功能在 `/search/` 内页
2. 后来用 search 内页内容替换首页
3. 为防止重复页面，把首页 canonical 指向 search 内页
4. search 内页 307 跳转到首页
5. 结果：用户行为数据、外链权重、排名全部保留在 search 内页 URL 上

**启发**：
- 最好上线时就直接把核心功能放在首页
- 用首页高权重打竞品内页，才是正确策略
- 307 是临时重定向，搜索引擎仍抓取原始 URL 的内容

### 1.3 FAQPage 结构化数据的历史与现状

**历史**：
- 2019年：Google 支持 FAQPage，搜索结果展示富文本问答，点击率大增
- SEO圈"万物皆可 FAQ"，所有页面尾部都加 FAQ 模块
- 2023年8月：Google 收紧，只给政府/健康类网站展示
- 2026年5月7日：彻底下线，所有网站不再展示 FAQ 富文本

**结论**：
1. 老页面 FAQPage 可留着，无负面影响
2. 新页面建议继续写——别的搜索引擎和 AI 爬虫可能还在用
3. **页面的 FAQ 模块一定要保留**——主要给用户看，打消疑虑、提高转化率

### 1.4 Keywords 标签的消亡史

- Google 从 1998 年成立就没用过 Keywords 标签
- 站长滥用 Keywords 堆砌→搜索结果质量下降→搜索引擎放弃读取
- **规律**：任何站长可直接控制的信号，最终都会被滥用→搜索引擎只奖励难以伪造的信号

### 1.5 新站批量化上页面必死

- 新站大量批量生成低质量页面会被 Google 惩罚
- 正确做法：一个关键词一个页面，做好每一个页面，控制质量
- 可以自动化这个过程，但页面生成速度要控制（一天5万页面不行）

### 1.6 GSC 成就系统

Google Search Console 的"成就"页面可作为 SEO 健康度检查：
- 第一个成就只需 28 天内 80 个点击
- 门槛设置非常务实，对新手友好
- 如果增长停滞（半年达不到下一个成就），说明操作方向有问题

### 1.7 外链策略

- 高质量外链→提升排名；低质量外链→充门面吓退竞争者
- **不要买垃圾外链**（淘宝/闲鱼 100块500条那种）
- Spam 标记的外链绝对不能要，给钱都不能要
- 蓝海词拿到排名后，要持续加外链让竞争者望而却步
- Ahrefs 免费版也能看到 Spam 标记

### 1.8 vercel.app 金矿找需求法

通过查看 vercel.app / github.io / netlify.app 等免费子域名的流量，找到不懂 SEO 但有真实需求的项目：
- 用 Similarweb 的"关键词研究→着陆页"工具查看子域名
- 这些站只解决需求不做 SEO，我们用 SEO 能力降维打击
- 这不是一次性金矿，而是随 AI 编程普及不断增长的金矿

**同类平台子域名列表**：netlify.app, pages.dev, web.app, surge.sh, onrender.com, railway.app, fly.dev, deno.dev, glitch.me, replit.app, lovable.app 等

### 1.9 新站优待期与曝光覆盖率

- Google 会给部分用户在前两页展示你的新站
- 通过首批用户的点击/停留数据判断质量
- 满足需求→加大曝光；不满足→减少曝光
- 优待期过后没曝光 = 竞争力不足

### 1.11 GSC 四级信号详解（2025-12-07 哥飞小课堂）

Google Search Console 中的页面状态信号，从轻到重依次为：

**第一级：已发现已抓取已收录，但不出词**
- 页面质量达标就能出词（低难度词）
- 高难度词还需要网站权重和外链支持
- 每个关键词的搜索结果都是一个排行榜，你只需跟当前排名的竞品竞争

**第二级：已发现已抓取不收录**
- 大量收录但不出词的页面还在持续产出→触发此警告
- 解决：暂停当前内容策略，已收录的能改就改，改不了就404下架

**第三级：已发现不抓取不收录**
- 第二级信号无动于衷→升级为第三级
- Google 明确告诉你不喜欢你的页面
- 解决：暂停内容生成 + 所有不收录/不抓取的页面全部404

**第四级：已收录页面排名骤降、曝光骤跌**
- 前三级几乎不影响已有排名的页面
- 第四级是**整站惩罚**，基本废站
- 解法：吸取教训，注册新域名重头再来

**其他触发第四级的原因**：
- 首页被意外改坏→单词数量暴增、主题无关内容→关键词密度分散→排名骤降
- 搞了大量垃圾外链试图操纵排名

### 1.12 面包屑导航的 SEO 细节（2026-02-12）

- Google 搜索结果中显示的层级路径（位置1），是基于页面面包屑导航推理出来的，不代表真实网址
- 首页的面包屑导航应该写：`网站名称 > 首页`
- 如果首页错写成内页格式（如 `Home > games > 某游戏名称`），Google 会显示错误的结构
- **教训**：面包屑导航要根据页面层级正确书写

### 1.13 aidocmaker.com SEO 案例分析（2026-01-24）

aidocmaker.com 短短两年月访问量30万+，在多个关键词上排名第一，超过 Canva、Gamma 等大站。

**哥飞总结10条结论**：

1. **首页打内页**依然是最有效的打法
2. 针对特定关键词注册域名 + 用首页打内页，外链不需要太多（400+域名外链即可稳排）
3. **一个页面一个关键词组**原则要坚守——同类关键词可组成词组合理分布
4. 一个页面可以拿到多个语义一致的同类关键词排名
5. TD（标题+描述）要组合多个关键词，自然语句而非堆砌
6. 页面内容一两千单词合理，少于600单词严重不合格
7. 内容越长越要控制关键词密度（AITDK插件检查）
8. 通过 Headings 控制文章脉络结构→控制关键词密度
9. **满足用户需求才是重中之重**（人均3页/跳出率37%/停留2分钟）
10. 流量增长与外链增长正相关，老词新站需要耐得住寂寞

### 1.14 EMD（Exact Match Domain）域名策略（2026-01-17）

**核心方法**：
- 找到有一定搜索量的需求关键词→直接用关键词注册域名→搞外链提权重
- 慢慢把需求关键词变成网站品牌词（把搜需求词的人变成搜你品牌词的人）
- 例：dinosaur-game.com 拿走了 "dinosaur game" 绝大多数点击

**seo.box 案例**：
- 搜索 "SEO Box" 排名第一，但月搜索量只有一两百
- 很多 SEO 服务商给甲方做这类"排名靠前但无搜索量"的词
- **警示**：不要被自己的品牌词排名迷惑，要看真实搜索量

**选择策略**：
- 新手：找几乎没人做的词，用 KGR 公式判断，避开竞争先拿正反馈
- 有经验者：可直接挑战搜索量大、难度高的词（已知道怎么做对）

## 二、需求洞察与选品

### 2.1 在线测试站品类

- Speedtest.net 月访问量1亿，Ookla 卖了12亿美金
- monkeytype.com（打字测试）月访问量1369万
- IQ测试、MBTI测试、鼠标测试、记忆力测试都是赚钱品类
- 可以按地区+测试类型组合出大量长尾词

### 2.2 追热点新词的方法

- 火过的东西可能再次火起来（如12月火过的词4月又火）
- 上站拿到流量后要持续维护，不然排名会掉
- 晚一两个月也还有机会，别焦虑
- 速度本身就是产品能力的一部分

### 2.3 Markdown 在线预览需求增长

- AI 生成内容基本都是 Markdown 格式，带动了预览工具需求
- markdownlivepreview.com 过去12个月访问量持续上升
- **观察方法**：当一个技术趋势出现后，相关周边工具的需求也会跟随增长

### 2.4 游戏站/粉丝站选品

- 新上映的电影/电视剧/游戏→做粉丝站（攻略、资讯、wiki、讨论区）
- 皇后放置游戏 playqueensgame.com 的数据：
  - 2025年2月注册，月访问量122万
  - 平均停留10分钟，跳出率32%
  - 持续13个月增长（中间6个月在20多万徘徊）

### 2.5 同一个关键词不同人群做不同产品

以 "AI Video Generator" 为例：
1. 娱乐视频生成
2. 广告视频生成
3. 自媒体短视频制作
4. 短剧生成

越细分的场景可能越赚钱（类比：卖小麦 < 卖面粉 < 卖蛋糕面包）

### 2.6 Chrome 内置小游戏搬运（2026-03-12）

- Chrome 浏览器内置的恐龙小游戏搬运到网页，居然还有85万/月访问量
- 恐龙游戏用 EMD 域名 dinosaur-game.io 拿走绝大多数点击
- Neal.fun 的 "Spend Bill Gates' Money" 爆火后，2021年有人做了 "Spend Elon Musk's Money"，目前还有63K访问
- **方法论**：同一类型游戏可以不断换皮做新游戏（换名人、换场景）
- Github Readme 文件生成器 profile-readme-generator.com 月访问6万，主要靠搜索引擎

### 2.7 drawaperfectcircletool.com 小工具站

- 去年9月注册域名，目前月访问量四五万
- 典型的"小需求也有大流量"案例

## 三、商业模式与创业案例

### 3.1 本地服务信息目录站

**模式**：为各行业个体从业者建信息目录站，SEO/SEM 获客，向 B 端收费。

**DoulaMatch.net 案例**：
- 连接待产家庭和陪产员，2008年创立
- 累积37,000名陪产员，月访问66K
- 冷启动：先免费收录服务提供者→再向需求方推广→从展示升级为预约平台
- 创始人本身就是执业陪产员，从自身痛点出发

**AVVO 律师评级站案例**：
- 2007年上线时覆盖全美律师
- 冷启动：抓取州律师协会+律所网站公开信息→建评级算法→"未认领不显示评分"逼律师认领
- 上线9天被起诉→联邦法官判"评级是主观意见，受言论自由保护"→奠定目录站行业法律基础
- 由 Expedia 创始人圈子发起

**收费模式**：基础订阅费 + 佣金提成 + 曝光竞价 + 建站/SEO服务

### 3.2 TextExpander（文本展开工具）

- 年营收620万美金，24万月访问量
- 核心功能：文本快捷替换（团队共享知识库）
- 收费策略：无免费版，只提供1个月免费试用（无API成本所以敢这样）
- 2003年起步，2022年PE融资4140万美金后被收购
- Chrome插件周活10万，竞品Text Blaze周活70万但收入更低（免费模式拖累）

**反面案例：Magical**
- 2020年上线走完全免费策略，拿到4100万美金融资
- 88人团队，后来发现免费用户不愿付费，转型Agentic AI平台
- **教训：能在第一天收费就一定要收费。免费筛选不出愿意付费的人群**

### 3.3 Sudowrite（AI写作工具冷启动五阶段）

1. **身份资本**：创始人本身就是科幻写作小组成员
2. **MVP验证**：先为自己和朋友做工具
3. **KOL背书**：绑定畅销小说作家Hugh Howey等资深作者
4. **媒体+融资PR**：豪华天使名单（Twitter/Medium创始人、WordPress创始人等）
5. **飞轮转动**：作家越深入使用→资产沉淀越多→越离不开

**素人怎么做**：不一定要走完全部阶段，适时引入外部资源/合伙人/投资让产品走更远

### 3.4 Chatbooks.com（照片打印订阅）

- 帮宝妈把Instagram照片自动打印成册寄出，每60张印一次
- 冷启动：给网红妈妈送样品→Instagram传播→拍"Real Mom"病毒视频（Facebook 8900万播放）
- **教训**：年收390万美元但有140人团队（人均创收2.8万），小生意不应拿大融资

### 3.5 freegen.app（免费AI+广告变现）

- 保加利亚公司，2025年9月域名，月访问量26万
- 基础模型免费使用→吸引用户→广告变现
- 外链不到200个，DR很低，但3月算法更新后起飞
- API成本可视为营销成本的一部分

## 四、网站运营与变现

### 4.1 定价与免费策略

**哥飞自己网站的经验**：
- 免费积分（注册送+每日签到）+ 付费套餐
- 付费金额 > API 成本，相当于"不花钱得到一个不断增长的网站"
- 技术可以解决的问题不应成为运营策略的拦路虎（防薅羊毛：不给印度IP积分、CF人机验证等）

### 4.2 成长阶段论

```
找不到需求不知道做什么
    → 需求太多做不过来
        → 找准需求一出手一个准
            → 这些需求不值得我出手了
```

### 4.3 Similarweb 数据仅供参考

群友游戏站实际UV 180K / PV 500K，但Similarweb显示716K访问量，说明Similarweb数据会高估。

### 4.4 广告账户冷启动五步法

1. 创建转化事件（注册、生成、加购、订阅）
2. 精准选投放地区+设置最高出价（不要投太多国家）
3. 从点击出价→转化次数出价→转化价值出价（逐步升级）
4. 日预算300-500起步，每次加预算不超20%
5. 冷启动成功率不高，要有亏几天的心理准备

### 4.5 2026年4月新词新站比赛数据

58位群友提交65个网站，前10名合计 UV 713万 / PV 2003万：
- 第1名赖嘉伟Gary：千万PV（SBTI相关）
- 第2名孙海军：当天100万UV
- 第5名诗意听涛：游戏攻略站，日入200刀

### 4.6 用户留存核心

- 让用户留下可复用的资产（Canva的模板、Sudowrite的小说资产）
- B端用户才有资产复用需求，C端用完就走正常
- 工具站不必转社区

### 4.7 Google Ads 投放详细策略（2026-01-26）

**哥飞6周 ROI 数据**：11% → 45% → 59% → 99% → 81% → 123%

**核心思路**：
- 初期不看 ROI，只求出单，让广告系统学习付费人群画像
- 之后不断优化 ROI，最终达到不亏且有赚
- 100个人有100种操作手法，别人的不一定适合你，要自己花钱交学费摸索

**点击出价策略**：
- 初期按点击出价，**一定要设最高点击单价**（否则半小时花光预算）
- 积累一定付费次数后→改为按付费次数出价→最终按价值出价
- 点击单价调低些→更多点击→更有可能碰到愿意付费的人

**投放地区策略**：
- 激进派（哥飞）：新账户冷启动投美国+英语，价格高没关系，要的是出单
- 省钱派：选中多国家地区+严控点击单价，但冷启动学习期更长
- **预算不高建议先从低广告单价开始，花时间换金钱**

**冷启动注意事项**：
- 初期不要同时冷启动多个广告计划
- 先建一个计划+多一些关键词，跑一段时间看哪些国家和词能出单
- 之后把不好的词屏蔽，能出单的国家单独建计划、单独出价

### 4.8 Google Ads 手机端 vs PC端分离（2026-03-13）

- 一条广告计划不屏蔽手机端→大概率亏本
- 只投PC端→经过调优可以回本
- 后来建了一条**只投手机端**的计划→调优后也能回本
- **结论**：手机端和PC端的人群不同，混在一条计划里不好跑，**应该分开建计划**
- 大家也可以单独建手机端计划试试

### 4.9 域名迁移案例：Quizizz → WayGround（2026-01-23）

**发现方法**：
1. 看到网站"旱地拔葱"直接起飞→排除热点（后续流量不坚挺）
2. 查外链→发现大量 301 重定向→找到原域名 quizizz.com
3. Similarweb 查两个域名12个月曲线→经典X形（老域名下降、新域名上升）

**迁移策略**：
- 新站多个明显位置写清楚品牌变更信息→承接老用户流量
- 写了一篇诚意满满的改名公告（品牌故事、为什么换名）
- **启示**：如果要迁移域名，要做好301重定向+用户告知+品牌故事

### 4.10 内页优化效果立竿见影（2026-01-25）

群友案例：月初优化内页，不到半个月全站曝光点击翻倍。

**内页分两类**：
1. **功能页**：围绕现有能力向相近需求扩展（image to text → image to pdf → pdf to word）
2. **关键词页**：覆盖近义词（photo/picture/image generator），功能可能一样

**上内页的价值**：
- 告诉搜索引擎：网站在稳定产出优质内容（不更新→曝光必然逐步下降）
- 告诉用户：网站在持续更新新功能→提高付费意愿和复用习惯

**流量不够=不出单**：确认需求找对后，唯一要做的就是提升流量——发帖、投广告、发外链、加内页，都在提升流量。

## 八、实战经验与教训

### 8.1 网站改版的教训（哥飞亲历）

- 网站曝光和点击在增长时，手痒改版→曝光点击双双下降
- 改版导致停留时长降到原来一半，跳出率上升，人均访问页面降到一半
- 原因：新版把多页面操作合并为单页面，体验提升了但数据下降了
- **教训**：改版一定要严格测试，不要仓促上线。一旦Google重新评估，排名下降需要数月爬坡恢复
- 改版前后网页内容没变（只换了后端框架/前端UI）是唯一保住部分排名的原因

### 8.2 排名前进几名 = 点击成倍增长

- 一个站过去28天曝光没增长，但排名前进了几位，点击增长了三倍
- 当排名在第一二页但还没进前三时，通过提升权重使排名提升，点击数能成倍增长

### 8.3 竞价广告的时间窗口策略

- 一天最后一小时（23点），可能竞争对手的当日预算花完了，你的还没花完→Google把流量给你
- 哥飞实测：23点60分钟内1030刀

### 8.23 谷歌趋势6种用法（2025-04-03 哥飞小课堂）
