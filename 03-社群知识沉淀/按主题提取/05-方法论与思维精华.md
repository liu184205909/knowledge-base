## 一、核心认知

### 1.1 AI是最大杠杆

- 无论做亚马逊、B2B、TikTok还是Shopee，AI可赋能业务方方面面
- AI只是提效和杠杆，方法和策略才是核心
- AI是放大器，放大的是你自身的能力
- 每个公司必须降本增效，拥抱AI

### 1.2 迁移能力 > 追逐工具

- 不要被"术"牵制，要学解题思路和方法
- 追最新AI工具的人做不大——技术思维不是生意思维
- 元技能vs技巧：AI变化很快，不变的是如何挖掘数据、分析竞品、管理团队——掌握"元技能"而非追逐技巧
- 工具只会越来越傻瓜，理解业务才是核心

### 1.3 学习方法

- 专注一个点：12个专题中能把一个点学深学精就够了
- 上课前先理清自己公司的SOP，明确用AI替代哪一部分，带着目标和问题学习
- 学习能力=快速掌握新事物的能力，这比具体工具操作重要得多
- 拒绝"照猫画虎"式培训：传授底层方法论和全新学习思维

## 二、行业趋势判断

### 2.1 去运营化

- TK的GMV MAX只需设ROI，其他交给AI代投
- Facebook算法变化：做好素材，其他交给AI投放
- 80%精力放产品开发：核心是发现需求、迭代产品、理解用户痛点
- 运营技巧没那么重要了，重要的是对市场和人性的理解

### 2.2 AI时代竞争壁垒

- 真正的竞争壁垒：不再是"会做什么"，而是"别人看懂了也做不到的"
- 对商业、市场、人性的深层理解，以及独特的稀缺性资源
- 公司六个稀缺性资源：个人IP、流量、专利技术、私域、高效SOP、高效AI运用
- 简化公式：**复利赛道 + 高效团队 + 稀缺资源**

### 2.3 数字化基础决定AI上限

- 先做好基础数字化（SOP、知识库、多维表格），再叠加AI自动化
- 好的公司很快就能上AI是因为SOP做得强
- 工作流=数字员工+公司数字资产
- 让公司从"依赖个人能力"进化为"系统化产出"

## 三、落地心法

### 3.1 老板必须亲自下场

- 员工研究AI缺乏内驱力（觉得在替代自己工作）
- 老板研究后更能看清业务走向和人员配置方向
- 没必要让人人都会，老板搭建工作流分发给团队即可

### 3.2 AI工具学习打破死循环

- **死循环**：n8n教程太难→找模板→模板也要改→不会改→回头要教程
- **解决**："上来就干"，先跑通一个能运行的工作流再说
- 主要还是要有想法，会提需求，让AI找解决方案

### 3.3 数字基建

- 搭建公司级数字基建——知识体系、SOP、业务流程系统化归档到飞书/钉钉
- 内容不仅要给团队看，还要给AI看
- CLI协作中的"踩坑记忆"机制：把问题和解决方案存入知识库，为AI建立Memory

## 四、精准需求方法论（小渔坚持）

> 来源：微信公众号文章「精准需求的转化非常高」| 作者：小渔坚持
> 沉淀时间：2026-05-18

### 4.1 核心观点

- **精准需求 > 宽泛需求**：找到一个"精准+垂直"的需求，用产品/内容精准匹配，转化率远高于泛流量
- 作者做 Affiliate Marketing（产品推荐），平均转化率 10%，最高 50%
- 核心公式：**关键词研究 → 找异常值（高流量/排名但低权威站点）→ 建精准产品**

### 4.2 执行步骤

1. **关键词研究**：用工具（Ahrefs/Semrush）搜索目标领域关键词
2. **找异常值**：发现高搜索量但 SERP 上排名靠前的网站权威度低的词 — 说明竞争不充分
3. **判断需求精准度**：这个搜索词背后是否对应一个非常具体的痛点？
4. **建产品/页面**：针对这个精准需求做一个独立的产品页面、工具页面或内容页面
5. **产品形式**：不一定是实物，可以是网站、电子表格、在线工具

### 4.3 关键数据

| 指标 | 数值 |
|------|------|
| 平均转化率 | ~10% |
| 最高转化率 | ~50% |
| 策略核心 | 一个产品 = 一个精准需求 |

### 4.4 与水晶项目的关联

这个方法论验证了我们内容策略中"场景词+长尾词"方向的正确性。水晶项目的场景应用系列（"crystals for anxiety"、"crystals for sleep"等）本质上就是"精准需求"策略的落地：

- "crystals for anxiety"（90,500月搜索量）→ 焦虑人群的精准需求
- "crystals for sleep"（60,500月搜索量）→ 失眠人群的精准需求
- "crystals for wealth"（27,100月搜索量）→ 求财人群的精准需求

**优化方向**：当前内容策略偏"广覆盖"（709篇），可以增加一个"深转化"支线 — 挑出 10-20 个最高转化潜力的精准需求词，每个词做一个"超级着陆页"（不仅是博客文章，而是工具+内容+产品推荐的组合页）。
