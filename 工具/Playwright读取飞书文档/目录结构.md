## 🎯 使用流程

### 方式 A: 使用 Claude Code（最简单）
```
1. 给 Claude Code 提供 Cookies
2. Claude Code 使用 Playwright MCP 读取
3. 自动总结文档内容
```

### 方式 B: 使用 Node.js 脚本
```bash
# 1. 配置 Cookies
vim feishu-cookies.json

# 2. 运行脚本
node read-feishu-with-cookies.js

# 3. 查看结果
cat feishu-doc-content.txt
```
