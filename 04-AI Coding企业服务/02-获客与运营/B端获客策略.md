# B端获客策略

> 最后更新：2026-05-30

## 一、获客渠道优先级

### 渠道矩阵

| 优先级 | 渠道 | 获客方式 | 转化周期 | 预期成本/线索 |
|-------|------|---------|---------|-------------|
| ★★★ | **内容营销（公众号/知乎）** | 深度文章→加微信→咨询 | 1-4周 | 低（时间成本） |
| ★★★ | **短视频（小红书/抖音）** | 效率对比视频→私信咨询 | 1-2周 | 低 |
| ★★☆ | **企业内训转介绍** | 满意客户推荐→新客户 | 即时 | 极低 |
| ★★☆ | **行业活动/大会** | 演讲→交换名片→跟进 | 2-8周 | 中 |
| ★★☆ | **合作伙伴** | 培训机构/咨询公司导流 | 2-4周 | 中（分成） |
| ★☆☆ | **SEO/官网** | 搜索流量→表单提交 | 4-12周 | 低（长期） |
| ★☆☆ | **付费广告** | 百度/抖音投放→落地页 | 1-4周 | 高 |

### 第一阶段（0-3个月）聚焦渠道
1. **小红书**（短期见效最快）
2. **微信公众号**（建立专业形象）
3. **知乎**（精准触达技术管理者）

## 二、内容营销策略

### 核心内容主题

| 主题 | 目标受众 | 内容形式 | 发布频率 |
|------|---------|---------|---------|
| AI Coding 效率对比（Before/After） | 研发人员 | 短视频+图文 | 每周2-3条 |
| 企业AI Coding落地案例 | 技术管理者 | 深度文章 | 每月2篇 |
| Claude Code/Cursor 实操教程 | 研发人员 | 图文/视频 | 每周1-2条 |
| AI转型行业趋势 | 高管 | 数据图表+洞察 | 每月1篇 |
| 培训现场花絮/学员反馈 | 潜在客户 | 短视频 | 不定期 |

### 内容生产 SOP

1. **选题**：从社群问题、客户咨询、行业热点中提取
2. **创作**：用AI辅助生成初稿，人工打磨
3. **多平台分发**：一篇内容适配多个平台
   - 深度文章 → 公众号 + 知乎 + 掘金
   - 实操教程 → 公众号 + 小红书 + B站
   - 短视频 → 小红书 + 抖音 + 视频号
4. **数据追踪**：每周分析各平台数据，调整内容方向

### 引流路径设计

```
内容曝光（小红书/抖音/知乎）
    ↓
关注/点赞/收藏
    ↓
引导加微信（"后台回复xxx获取实操清单"）
    ↓
微信私域运营（朋友圈+群）
    ↓
免费体验课/直播
    ↓
付费内训/陪跑
```

## 三、线索培育（微信私域）

### 微信人设打造
- **头像**：专业商务照
- **签名**：AI Coding 落地顾问 | 帮企业把AI编程从个人技能升级为组织能力
- **朋友圈**：培训现场 + 学员反馈 + 行业洞察 + 工具更新（比例 3:2:3:2）

### 自动回复设置
```
新好友自动回复：
"Hi，我是[名字]，AI Coding 企业培训顾问。
看到你对AI编程感兴趣！
免费领取《Claude Code 企业实操清单》：[链接]
有企业培训需求可以随时问我~"
```

### 社群运营
- **免费群**：AI Coding 学习交流群（定期分享干货、答疑）
- **付费群**：企业AI Coding 落地会员群（深度交流、案例分享、优先体验）

### 线索分级

| 级别 | 特征 | 跟进策略 |
|------|------|---------|
| A级（热线索） | 明确表达内训需求、有预算、有决策权 | 48小时内电话/视频沟通 |
| B级（温线索） | 关注内容、问过价格、无明确时间 | 每周1次价值内容推送 |
| C级（冷线索） | 加了微信但不活跃 | 朋友圈触达 + 月度活动邀请 |

## 五、定价策略

### 定价参考

| 产品 | 建议定价 | 备注 |
|------|---------|------|
| 免费直播/体验课 | ¥0 | 引流 |
| 半天工作坊 | ¥500-800/人 | 低门槛体验 |
| 1天标准培训 | ¥2000-3000/人 | 公开课 |
| 2天深度培训 | ¥4000-6000/人 | 公开课 |
| 企业内训（1天） | ¥3-5万 | 10-30人 |
| 企业内训（2天） | ¥5-8万 | 10-30人 |
| 月度陪跑 | ¥5-10万 | 含培训+模板+跟踪 |
| 季度咨询 | ¥15-30万 | 全面落地 |

### 定价心理学
- **锚定效应**：先报陪跑价格（高），再说单次培训价格（显得合理）
- **套餐捆绑**：内训+3个月线上答疑 = 加量不加价
- **早鸟折扣**：提前2周报名享8折
