## 🎯 什么是GEO？

**GEO = Generative Engine Optimization（生成引擎优化）**

在AI搜索时代，用户通过ChatGPT、Perplexity、Claude等AI平台获取答案。GEO的目标是让AI在回答用户问题时**推荐你的品牌**。

### GEO vs SEO 核心区别

| 维度 | SEO（搜索引擎优化） | GEO（生成引擎优化） |
|------|-------------------|-------------------|
| **争夺目标** | Google搜索排名 | AI对话中的推荐 |
| **核心机制** | 关键词匹配 + 外链权重 | 语义理解 + 信源权威性 |
| **流量逻辑** | 搜索 → 点击 → 转化 | 提问 → AI推荐 → 转化 |
| **成功指标** | 网站排名、流量 | AI可见度、推荐度 |
| **时间周期** | 3-6个月见效 | 6-12个月见效 |

**关键洞察**：
```
SEO: 用户搜索 → 看到你的网站 → 点击 → 进入网站
GEO: 用户提问 → AI推荐你的品牌 → 直接了解或搜索品牌词

SEO追求"点击"，GEO追求"提及"和"推荐"
```

### 零点击时代

- **60%的Google搜索不产生点击**
- AI直接回答问题，用户无需访问网站
- 品牌认知 > 流量获取

## 📋 品牌自检流程（6步法）

> 在开始GEO优化前，先完成品牌自检，识别信息真空和声誉问题

### 第1步：品牌事实核查（Fact Check）

**目标**：确保品牌基础信息准确、一致

**检查清单**：
- [ ] 品牌名称在各平台完全一致（不要用"TaskFlow.io"、"Task Flow"等变体）
- [ ] 品牌定位一句话描述（不超过20字）
- [ ] 成立日期统一（所有平台一致）
- [ ] 联系方式一致（邮箱、电话、地址）
- [ ] 核心数据统一（用户数、融资数据等）

**常见问题**：
- ❌ LinkedIn写"TaskFlow.io"，官网写"TaskFlow"
- ❌ Wikipedia写"2020年"，官网写"2021年"
- ❌ 不同平台的品牌定位描述完全不同

**解决方案**：
- 创建"品牌事实清单"文档
- 所有平台使用统一信息
- 定期检查各平台信息一致性

---

### 第2步：信息真空检测

**目标**：识别AI找不到官方信息的空白点

**检测方法**：
```bash
# 测试Prompt
"你知道[品牌名]吗？"
"请介绍一下[品牌名]"
"[品牌名]是做什么的？"

# 检查AI回答
- ❌ AI说"没有相关信息" → 信息真空
- ❌ AI回答模糊、不具体 → 信息不足
- ❌ AI回答与事实不符 → 错误信息
```

**常见信息真空**：
1. 产品功能详细说明缺失
2. 定价和套餐信息不明确
3. 公司基本信息不完整
4. 客户案例和数据缺失
5. FAQ覆盖不足

**解决方案**：
- 创建FACTS页面（事实来源）
- 建立详尽FAQ（20-30个问题）
- 提供所有可公开的数据
- 使用具体数据，避免空话

**目标**：了解竞品在AI中的表现，识别差距

**对比方法**：
```bash

- AI推荐它们的理由是什么？
- 你的品牌是否被提及？

```

**对比维度**：

| 维度 | 你的品牌 | 竞品A | 竞品B | 差距分析 |
|-----|---------|-------|-------|---------|
| AI认知度 | ? | ? | ? | 是否被提及 |
| 提及位置 | ? | 第1 | 第2 | 排名差距 |
| 引用来源 | ? | 官网+Reddit | 仅官网 | 来源多样性 |
| 信息密度 | ? | 高（具体数据） | 低（空话） | 信息详细度 |
| 品牌一致性 | ? | 完全一致 | 部分不一致 | 多平台统一度 |

**差距识别**：
- 竞品有你没有的信息源？（如Reddit讨论、媒体报道）

## ⚙️ GEO三大核心环节

```
让AI引擎看懂并引用你的网站 = 抓取 → 理解 → 引用
```

### 环节1：抓取优化

#### 1.1 允许AI爬虫访问

**检查robots.txt**：
```txt
# 正确配置：允许AI爬虫
User-agent: GPTBot
Allow: /

User-agent: PerplexityBot
Allow: /

User-agent: Google-Extended
Allow: /
```

**主要AI爬虫**：
- GPTBot（OpenAI/ChatGPT）
- PerplexityBot（Perplexity AI）
- Google-Extended（Google Gemini）
- Claude-Web（Anthropic Claude）

#### 1.2 确保HTML内容可见

**问题**：AI爬虫不执行JavaScript，动态渲染的内容看不到。

**检查方法**：

## 📊 GEO KPI体系

> 4个核心指标，全面衡量GEO效果

### KPI 1：AI Visibility（AI可见度）⭐⭐⭐

**定义**：品牌在AI平台中的可见程度

**计算方法**：
```bash
# AI Visibility计算

AI Visibility = (AI平台提及数 / 目标提及数) × 100%

示例：
- ChatGPT提及：3次
- Perplexity提及：2次
- Claude提及：1次
- 总提及：6次
- 目标提及：10次
- AI Visibility = (6/10) × 100% = 60%
```

**监测方法**：
1. 手动测试（每周）
   ```bash
   测试Prompt：
   - "你知道[品牌名]吗？"
   - "推荐5个最好的[品类]"
   - "[品牌A] vs [品牌B]，哪个更好？"

   记录：
   - 是否被提及
   - 提及位置（第1、第2、第3？）
   - 提及内容是否准确
   ```

2. 工具监测（每日）
   - GEO Platform
   - AISEOlytics
   - 自建监测脚本

**目标值**：
- 初期（1-3月）：20-40%
- 中期（3-6月）：40-60%
- 长期（6-12月）：60-80%

---

### KPI 2：Share of Voice（SOV，声量份额）⭐⭐⭐

**定义**：品牌在AI推荐中的占比

**计算方法**：
```bash
# SOV计算

SOV = (品牌提及次数 / 所有竞品提及总数) × 100%

示例：
- 你的品牌：6次提及
- 竞品A：8次提及
- 竞品B：4次提及
- 竞品C：2次提及
- 总提及：20次
- SOV = (6/20) × 100% = 30%
```

**监测维度**：
- 按平台：ChatGPT SOV、Perplexity SOV
- 按关键词：核心关键词SOV、长尾关键词SOV
- 按时间：月度SOV、季度SOV

**目标值**：
- 初期：5-15%
- 中期：15-25%
- 长期：25-40%

---

### KPI 3：Sentiment Score（情感得分）⭐⭐

**定义**：AI回答中品牌的情感倾向

**评分标准**：
```bash
# Sentiment Score计算

正面提及：
- 强烈推荐（+3分）
- 推荐（+2分）
- 中立偏正（+1分）

中性提及：
- 中立（0分）
- 事实陈述（0分）

负面提及：
- 中立偏负（-1分）
- 不推荐（-2分）
- 强烈不推荐（-3分）

Sentiment Score = (总分 / 提及次数)

示例：
- 强烈推荐：2次（6分）
- 推荐：3次（6分）
- 中立偏正：1次（1分）
- 总分：13分
- 总次数：6次
- Sentiment Score = 13/6 = 2.17（正面）
```

**分析方法**：
1. 人工分析（每周）
   - 记录每次提及的情感
   - 计算Sentiment Score
   - 分析情感趋势

2. 工具分析（每日）
   - Brand24（品牌提及监控）
   - Mention.com（社交媒体）
   - 自定义情感分析

**目标值**：
- 优秀：2.0+（强烈正面）
- 良好：1.0-2.0（正面）
- 一般：0-1.0（中性偏正）
- 较差：<0（有负面）

---

### KPI 4：Truth Alignment Score（真实对齐得分）⭐⭐⭐

**定义**：AI认知与品牌事实的吻合度

**评分标准**：
```bash
# Truth Alignment Score计算

准确性评估：
- 完全准确（100分）
- 基本准确（80-99分）
- 部分错误（50-79分）
- 严重错误（0-49分）

评估维度：
1. 品牌基础信息（25分）
   - 名称、定位、成立日期
   - 创始人、总部、官网

2. 产品信息（25分）
   - 功能描述、定价
   - 产品特性、使用场景

3. 数据准确性（25分）
   - 用户数、覆盖范围
   - 性能指标、客户满意度

4. 权威性信息（25分）
   - 媒体报道、认证
   - 合作伙伴、奖项

Truth Alignment Score = 各维度得分平均

示例：
- 品牌基础信息：25/25（完全准确）
- 产品信息：20/25（基本准确，定价有小误差）
- 数据准确性：22/25（基本准确，用户数略有偏差）
- 权威性信息：23/25（基本准确）
- Truth Alignment Score = (25+20+22+23)/4 = 90/100（优秀）
```

**监测方法**：
1. 定期测试（每月）
   - 使用标准化测试Prompt
   - 对比AI回答与品牌事实
   - 计算吻合度

2. 偏差追踪
   - 记录所有错误信息
   - 分类偏差类型
   - 追踪纠正进度

**目标值**：
- 优秀：90-100分
- 良好：80-89分
- 一般：70-79分
- 较差：<70分

## 📋 常见问题

### Q1：为什么我的内容没有被AI引用？

**可能原因**：
1. 信息真空：官网信息不足
2. 内容太空泛：使用"行业领先"等空话
3. 品牌一致性不足：各平台描述不一致
4. 时间不够：需要6-12个月积累

### Q2：我需要同时做SEO和GEO吗？

**答案**：是的，两者互补。
- SEO是GEO的基础
- 建议资源分配：70% SEO + 30% GEO（初期）
- 随着零点击率上升，逐步增加GEO比例

### Q3：不同AI平台的结果为什么不一样？

**原因**：
- ChatGPT：更依赖官网和权威来源
- Perplexity：更依赖Reddit和论坛
- Gemini：可能更信任Medium等平台

**策略**：分别测试各平台，针对性优化。
