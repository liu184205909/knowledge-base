## 🎯 GEO是什么？

**GEO = Generative Engine Optimization（生成引擎优化）**

在AI搜索时代（ChatGPT、Perplexity、Claude），GEO的目标是让AI在回答用户问题时**推荐你的品牌**。

### GEO vs SEO核心区别

| 维度 | SEO | GEO |
|------|-----|-----|
| **争夺目标** | Google搜索排名 | AI对话中的推荐 |
| **核心机制** | 关键词匹配 + 外链权重 | 语义理解 + 信源权威性 |
| **流量逻辑** | 搜索 → 点击 → 转化 | 提问 → AI推荐 → 转化 |
| **成功指标** | 排名、流量、CTR | AI提及、品牌搜索、信任度 |
| **时间周期** | 3-6个月 | 6-12个月 |

**关键洞察**：
```
SEO: 用户搜索 → 看到你的网站 → 点击 → 进入网站
GEO: 用户提问 → AI推荐你的品牌 → 直接了解或搜索品牌词

SEO追求"点击"，GEO追求"提及"和"推荐"
```

**零点击时代**：60%的Google搜索不产生点击，AI直接回答问题

## 📋 品牌自检流程（6步法）

> 在开始GEO优化前，先完成品牌自检

### 第1步：品牌事实核查

**目标**：确保品牌基础信息准确、一致

**检查清单**：
- [ ] 品牌名称在各平台完全一致（不要用变体）
- [ ] 品牌定位一句话描述（不超过20字）
- [ ] 成立日期统一（所有平台一致）
- [ ] 联系方式一致（邮箱、电话）
- [ ] 核心数据统一（用户数、融资等）

**解决方案**：
- 创建"品牌事实清单"文档
- 所有平台使用统一信息
- 定期检查各平台信息一致性

---

### 第2步：信息真空检测

**目标**：识别AI找不到官方信息的空白点

**检测方法**：
```bash
# 测试Prompt
"你知道[品牌名]吗？"
"请介绍一下[品牌名]"
"[品牌名]是做什么的？"

# 检查AI回答
- ❌ AI说"没有相关信息" → 信息真空
- ❌ AI回答模糊、不具体 → 信息不足
- ❌ AI回答与事实不符 → 错误信息
```

**常见信息真空**：
1. 产品功能详细说明缺失
2. 定价和套餐信息不明确
3. 公司基本信息不完整
4. 客户案例和数据缺失
5. FAQ覆盖不足

**解决方案**：
- 创建FACTS页面（事实来源）
- 建立详尽FAQ（20-30个问题）
- 提供所有可公开的数据
- 使用具体数据，避免空话

---

### 第3步：声誉审计

**目标**：评估品牌在各平台的声誉现状

**审计维度**：

**A. Wiki平台**
- [ ] Wikipedia条目是否存在
- [ ] Wikidata条目是否完整
- [ ] 信息是否准确

**B. 评论网站**
- [ ] Trustpilot评分和评价数
- [ ] G2/Capterra评分和评价数
- [ ] 负面评价处理情况
- [ ] 评分是否在4.0以上

**C. 社区论坛**
- [ ] Reddit品牌提及次数
- [ ] Quora品牌提及次数
- [ ] 正面/负面提及比例

**D. 媒体报道**
- [ ] 是否有权威媒体报道
- [ ] 报道内容的正面性

**评估标准**：
- ⭐ 优秀：Wiki完整 + 4.5+评分 + 100+Reddit正面提及 + 媒体报道
- ⭐⭐ 良好：Wiki存在 + 4.0+评分 + 50+Reddit提及
- ⭐⭐⭐ 一般：无Wiki + 3.5-4.0评分 + 10-50Reddit提及
- ⭐⭐⭐⭐ 较差：无Wiki + <3.5评分 + <10Reddit提及

## 📊 GEO KPI体系

### KPI 1：AI Visibility（AI可见度）⭐⭐⭐

**定义**：品牌在主流AI平台中被提及的频率

**计算方法**：
```bash
AI Visibility = (有品牌提及的查询数 / 总测试查询数) × 100%

测试查询示例（20-30个）：
- "推荐最好的[品类]"
- "[品牌A] vs [品牌B]，哪个好？"
- "我需要[具体需求]，推荐哪个？"
```

**目标值**：
- 初期（1-3月）：20-30%
- 中期（3-6月）：30-50%
- 长期（6-12月）：50-70%

---

### KPI 2：SOV（Share of Voice）⭐⭐⭐

**定义**：品牌在AI推荐中的竞争占比

**计算方法**：
```bash
SOV = (你的品牌提及次数 / 所有品牌提及总次数) × 100%

示例：
AI推荐5个品牌，你的品牌被提及3次
SOV = 3/5 = 60%
```

**目标值**：
- 初期（1-3月）：5-15%
- 中期（3-6月）：15-25%
- 长期（6-12月）：30-50%

---

### KPI 3：Sentiment Score（情感评分）⭐⭐

**定义**：AI提及你的品牌时的正面程度

**计算方法**：
```bash
人工评估AI回答（1-3分）：
- 3分：强烈推荐（"强烈推荐"、"最佳选择"）
- 2分：一般推荐（"可以考虑"、"不错"）
- 1分：中性提及（仅提及品牌名）

Sentiment Score = 平均分

目标值：
- 优秀：2.0+
- 良好：1.5-2.0
- 一般：1.0-1.5
```

---

### KPI 4：Truth Alignment（真实对齐度）⭐⭐

**定义**：AI认知与品牌事实的一致性

**计算方法**：
```bash
测试10个关键事实（品牌名、定位、功能、定价等）

Truth Alignment = (正确事实数 / 总事实数) × 100%

目标值：85%+
```
