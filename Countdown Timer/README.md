# Countdown Timer 倒计时器

## 项目简介
这是一个简单的倒计时器应用，可以设置特定的时间进行倒计时。

## 功能特点
- 可设置小时、分钟和秒数
- 支持开始、暂停和重置功能
- 时间到达时发出提醒
- 简洁直观的用户界面

## 详细配置说明
### 基础设置
1. 时间设置
   - 小时：0-99
   - 分钟：0-59
   - 秒数：0-59

2. 提醒设置
   - 声音提醒：可选择提醒音效
   - 弹窗提醒：可开启/关闭
   - 震动提醒：移动设备可用

### 高级设置
1. 主题设置
   - 浅色主题
   - 深色主题
   - 自动跟随系统

2. 快捷时间
   - 可设置最多 5 个常用时间
   - 支持快速切换

## 使用说明
1. 在输入框中设置所需的时间
2. 点击"开始"按钮启动倒计时
3. 可随时点击"暂停"按钮暂停计时
4. 点击"重置"按钮可重新设置时间

## 常见问题解答
### Q1: 为什么倒计时不准确？
A: 浏览器标签页在后台时可能会导致计时不准确，建议保持标签页在前台运行。

### Q2: 如何关闭提醒音效？
A: 在设置页面中可以调节音量或完全关闭提醒音效。

### Q3: 是否支持后台运行？
A: 目前版本需要保持浏览器窗口开启，后续版本将支持后台运行功能。

### Q4: 能否同时运行多个倒计时？
A: 当前版本仅支持单个倒计时，多个倒计时功能正在开发中。
