# 超级个体实践指南

> 基于刘小排一年创业实战经验总结 | 验证时间：2025年

## 核心理念

**超级个体可以培养，不是只能筛选**

结合AI能力，一个人可以完成从产品设计、编程开发、运营推广到商业盈利的全流程工作。

---

## 实践验证数据

### 个人案例
- **零基础成长周期**：6-12个月
- **收入水平**：单人月收入3万美金+
- **单日峰值**：6000美金+（真实案例）
- **行业背景**：包括夕阳传统行业、零互联网基础

### 团队案例
- **全员写代码**：产品经理、运营、设计师都学会了编程
- **AI工具加持**：每个人同时是PM+运营+设计师+程序员
- **不加班、不PUA**：健康的工作文化也能产出优秀产品

---

## 关键成功因素

### 1. AI编程工具链
- **主力工具**：Claude Code
- **辅助工具**：Cursor、MCP
- **开发模式**：Vibe Coding（口喷编程）

### 2. 成长型思维
- 固定型思维：超级个体只能筛选，不能培养
- **成长型思维**：王侯将相宁有种乎，通过努力可以达成

> "人们相信什么，就会看到什么。人们看到什么，就会发生什么。"

### 3. 技能叠加策略
- AI把所有技能拉到**80分**
- 每个人有一项**95分**的长板能力
- 组队不是补短板，而是叠加长板

### 4. 对不确定性的承受能力
比乐观更重要的创业素质：
- 业务模型不依赖工时
- 不以工时长短论英雄
- 接受收入波动是常态

## 团队管理哲学

### 核心原则
1. **己所不欲，勿施于人**
   - 不加班文化
   - 不PUA员工
   - 全员发iPhone 17 Pro Max

2. **把用户当你妹**
   - 你会骗你妹吗？
   - 你希望你妹花这个价格买这款产品吗？
   - 你会对你妹失去耐心吗？

3. **认知谦逊**
   - 承认自己需要知道的远比已经知道的更多
   - 向任何人学习，包括nobody
   - 空杯心态，像小白一样请教

4. **情绪管理**
   - 我的情绪 = 同事的办公环境
   - 情绪不好就调整好了再去公司
   - 君子修己以安人

## 学习路径建议

### 第1-3个月：工具掌握
- 安装Claude Code
- 学习基础编程
- 完成第一个小产品

### 第4-6个月：全流程实践
- 产品设计
- 编程开发
- 运营推广
- 实现第一笔收入

### 第7-12个月：规模化
- 优化产品矩阵
- 建立自动化流程
- 探索增长策略
- 目标：单人月收入3万美金+
