## 💡 Prompt设计核心要素

### 1. 明确目标
```
❌ 差："帮我写篇文章"
✅ 好："写一篇1000字的SEO文章，关键词是'AI工具评测'，
       目标受众是开发者，语气要专业但易懂"
```

### 2. 提供上下文
```
❌ 差："优化这段代码"
✅ 好："这是一个Python Flask API的性能优化任务，
       当前响应时间800ms，目标是降到200ms以下。
       主要瓶颈在数据库查询，需要添加缓存和索引。"
```

### 3. 明确输出格式
```
❌ 差："给我分析结果"
✅ 好："以Markdown表格形式输出，包含：
       | 问题 | 原因 | 解决方案 | 优先级 |"
```

### 4. 迭代优化
```
第1版：粗糙Prompt
第2版：根据AI输出调整要求
第3版：细化具体细节
...
最终版：高质量Prompt
```

## ⚠️ 常见误区

### 误区1：Prompt越长越好
❌ **错误**：写几千字的超长Prompt
✅ **正确**：简洁但包含所有必要信息

### 误区2：一次性写出完美Prompt
❌ **错误**：期望第一次就写出完美Prompt
✅ **正确**：迭代优化，逐步改进

### 误区3：忽视AI反馈
❌ **错误**：不接受AI的建议和问题
✅ **正确**：根据AI输出调整Prompt

### 误区4：照搬模板
❌ **错误**：直接使用他人Prompt，不修改
✅ **正确**：理解模板原理，根据场景调整
