# 本地化产品重写 Prompt 模板

## 🎯 核心原则

**不是机器翻译，而是"像做本地化产品一样"的重写。**

这意味着：
- ❌ 不是逐字逐句翻译
- ✅ 是深度理解后，用目标语言重新创作
- ✅ 考虑目标市场的文化、习惯、用户心理
- ✅ 让内容感觉像为该地区原生创作的

---

## 📝 基础 Prompt 模板

### 版本1：简洁版

```
你是一位精通[目标语言]的本地化产品专家，同时也是[产品类型]领域的资深从业者。

请将以下内容进行本地化产品式重写（不是机器翻译）：

[原文内容]

要求：
1. 深度理解原文的核心信息和产品功能
2. 用[目标语言]重新创作，而非逐字翻译
3. 考虑[目标地区]的文化背景、用户习惯和表达方式
4. 让内容感觉像为[目标地区]用户原生创作的产品文案
5. 保持原文的核心价值和产品功能描述准确
```

### 版本2：详细版（推荐）

```
角色定位：
你是一位精通[目标语言]的本地化产品经理，同时深度了解[产品类型/领域]。
你不仅要精通语言，更要理解产品功能、用户心理和本地文化。

任务：
将以下内容进行"本地化产品式重写"（不是机器翻译，而是像做本地化产品一样重新创作）：

---
[原文内容]
---

本地化要求：
1. **深度理解**：先理解原文的核心信息、产品功能、目标用户和价值主张
2. **文化适配**：考虑[目标地区]的：
   - 文化背景和价值观
   - 用户习惯和偏好
   - 常见的表达方式和语言习惯
   - 本地化的产品文案风格
3. **原生创作**：用[目标语言]重新创作，让内容感觉像为该地区用户原生创作的产品文案
4. **功能准确**：保持产品功能描述的技术准确性
5. **情感共鸣**：使用能引起[目标地区]用户共鸣的表达方式

输出格式：
- 重写后的内容
- [可选] 简要说明你做了哪些本地化调整
```

### 版本3：多语言批量版

```
你是一位精通32种语言的本地化产品专家，同时也是[产品类型]领域的资深从业者。

任务：
对以下内容进行本地化产品式重写，输出32种语言的版本。

原文：
[原文内容]

要求：
1. 对每种语言，都要像为该地区用户原生创作一样重写
2. 考虑每种语言对应地区的文化、习惯、表达方式
3. 不是机器翻译，而是深度理解后的重新创作
4. 保持产品功能描述准确

输出格式：
[语言1]：[重写内容]
[语言2]：[重写内容]
...
```

## 🔧 进阶 Prompt 技巧

### 1. **添加产品上下文**

```
[在基础模板前添加]

产品背景：
- 产品名称：[产品名]
- 产品类型：[类型]
- 核心功能：[功能1、功能2、功能3]
- 目标用户：[用户画像]
- 产品定位：[定位描述]

[然后使用基础模板]
```

### 2. **指定参考风格**

```
[在要求中添加]

参考风格：
- 参考[知名本地化产品/品牌]的文案风格
- 使用[目标地区]用户熟悉的表达方式
- 避免直译腔，使用地道的本地化表达
```

### 3. **强调关键要素**

```
[在要求中强调]

特别注意：
- SEO关键词：[关键词1、关键词2]（如果适用）
- 情感调性：[专业/亲切/幽默等]
- 长度要求：[保持原文长度/适当调整]
- 特殊要求：[如避免某些词汇、强调某些卖点]
```

## 💡 实际应用示例

### 示例1：产品介绍文案

```
你是一位精通日语的本地化产品专家，同时也是SaaS工具领域的资深从业者。

请将以下产品介绍进行本地化产品式重写：

原文：
"我们的AI写作助手可以帮助你快速生成高质量内容，支持多种写作场景，让创作更高效。"

要求：
1. 深度理解产品的核心价值（AI写作、高效、多场景）
2. 用日语重新创作，考虑日本用户对效率工具的偏好
3. 使用日本SaaS产品常见的表达方式
4. 让内容感觉像为日本市场原生创作的产品介绍
```

### 示例2：营销文案

```
你是一位精通西班牙语的本地化营销专家，深度了解电商产品。

请将以下营销文案进行本地化产品式重写：

原文：
"限时优惠！现在购买享受8折优惠，仅限前100名用户。"

要求：
1. 理解促销的核心信息（限时、折扣、限量）
2. 考虑西班牙语地区的促销表达习惯
3. 使用能引起当地用户购买冲动的表达方式
4. 保持紧迫感，但符合当地文化习惯
```

## ⚠️ 常见错误 vs 正确做法

### ❌ 错误写法

```
"请将以下内容翻译成[目标语言]"
"请用[目标语言]翻译这段文字"
"像本地化的人一样重写"（太模糊）
```

**问题：**
- 强调"翻译"，AI会倾向于逐字翻译
- 没有明确本地化产品的要求
- 缺少文化、用户心理等关键要素

### ✅ 正确写法

```
"你是一位精通[目标语言]的本地化产品专家，请进行本地化产品式重写（不是机器翻译）"
"像做本地化产品一样重新创作，考虑目标地区的文化、习惯和表达方式"
"让内容感觉像为该地区用户原生创作的产品文案"
```

**优势：**
- 明确角色定位（本地化产品专家）
- 强调"重写"而非"翻译"
- 明确要求（文化、习惯、原生感）

---

## 🎯 关键要素总结

一个优秀的本地化产品重写 Prompt 应该包含：

1. **角色定位**：本地化产品专家 + 领域专家
2. **明确任务**：本地化产品式重写（不是翻译）
3. **核心要求**：
   - 深度理解原文
   - 文化适配
   - 原生创作感
   - 功能准确性
4. **目标地区信息**：文化、习惯、表达方式
5. **产品上下文**：（可选）产品背景、功能、用户画像
