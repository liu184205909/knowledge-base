## 第1部分: SEO内容营销核心流程

### 1.1 关键词研究（30分钟）

#### 🎯 目标
找到20个高价值、低竞争的关键词

#### 执行步骤

**步骤1: 从用户洞察提取种子关键词**

基于[01-前置研究.md](./01-前置研究.md)的用户洞察，列出种子关键词：
```
示例: AI写作工具、SEO软件、内容营销工具
```

**步骤2: 使用AI研究关键词**

```bash
"基于用户洞察研究关键词：

种子关键词: [种子关键词]

【目标】
- KD < 30（竞争度低）
- 搜索量 > 100/月
- 长尾关键词优先
- 符合用户真实需求（场景化/情感化）
- 避开竞品已占据的高竞争关键词

输出: Top 20 关键词列表，包含：
- 关键词
- KD值
- 搜索量
- 用户意图（信息型/交易型/导航型）
- 推荐内容类型（教程/列表/对比/评测）"
```

**📋 变量表**：

| 变量名 | 说明 | 示例值 | 必填 |
|--------|------|--------|-----|
| SEED_KEYWORD | 种子关键词 | "AI工具" | ✅ |
| KD_THRESHOLD | 竞争度阈值 | 30 | ❌ |
| MIN_VOLUME | 最低搜索量 | 100 | ❌ |
| OUTPUT_COUNT | 输出数量 | 20 | ❌ |

**步骤3: 分析SERP，确定内容形态**

对每个关键词，分析Google前10名：
```
- 内容类型分布（文章/产品页/视频）
- 内容格式（列表/对比/教程）
- 内容长度（短/中/长）
- 媒体类型（文字/图片/视频）

总结：Google最想展示的内容形态是什么？
```

**输出**: 保存到Google Sheets或Notion

---

### 1.2 内容生成（10分钟/篇）

#### 🎯 目标
基于关键词生成2000-2500字的SEO优化文章

#### 执行步骤

**步骤1: 确定内容类型和角度**

基于SERP分析和用户洞察：
```
内容类型: 教程/列表/对比/评测
差异化角度: [从01-前置研究的用户洞察提取]
```

**步骤2: 生成文章**

```bash
"生成SEO优化文章：

【关键词】
[关键词]

【内容规划】
- 类型: [教程/列表/对比/评测]
- 字数: 2000-3000字
- 角度: [差异化角度，基于用户洞察]

【SEO优化】
- 标题含关键词（前置）
- Meta description: 150字符
- H1-H3含相关关键词
- 关键词密度: 1-2%
- 内部链接: 3-5个
- 外部链接: 3-5个
- 图片: 3-5张（带alt）

【差异化生成】
1. 分析竞品避免雷同
2. 基于用户洞察的差异化角度
3. 满足用户真实需求（而非表面诉求）
4. 匹配用户使用场景
5. 符合用户决策标准

【质量检查】
- SEO评分 > 90
- 可读性 > 60

输出: 完整文章（Markdown格式）"
```

**📋 变量表**：

| 变量名 | 说明 | 示例值 | 必填 |
|--------|------|--------|-----|
| TARGET_KEYWORD | 目标关键词 | "AI写作工具" | ✅ |
| CONTENT_LENGTH | 文章字数 | 2000 | ✅ |
| CONTENT_TYPE | 内容类型 | "教程/对比" | ✅ |
| DIFFERENTIATION_ANGLE | 差异化角度 | "免费工具对比" | ✅ |
| TONE | 语气风格 | "专业" | ❌ |
| SEO_SCORE_THRESHOLD | SEO评分阈值 | 90 | ✅ |

**步骤3: 质量检查**

```bash
"检查文章质量：

【文章内容】
[文章内容]

【检查项】
✅ SEO评分 > 90
✅ 关键词密度 1-2%
✅ H1/H2/H3结构完整
✅ Meta描述长度150字符
✅ 内部链接3-5个
✅ 外部链接3-5个
✅ 图片Alt标签完整
✅ 文章字数2000+
✅ 内容准确性（无错误信息）
✅ 内容独特性（非抄袭）

输出: 质量检查报告 + 优化建议"
```

**输出**: 保存到Notion或Google Docs

### 1.3 发布优化（10分钟）

#### 🎯 目标
将文章发布到WordPress并优化

#### 执行步骤

**步骤1: 准备发布材料**

- 文章内容（Markdown）
- 特色图片（1张）
- 配图（3-5张）
- Meta信息（标题、描述）
- 分类和标签

**步骤2: 发布到WordPress**

方法A - 使用Rube（自动化）：
```bash
"使用Rube发布文章到WordPress：

【文章信息】
- 标题: [文章标题]
- 内容: [Markdown格式]
- 分类: [选择分类]
- 标签: [自动生成5-8个标签]
- 特色图片: [上传图片]

【SEO设置】
- Meta标题: [文章标题]
- Meta描述: [150字符描述]
- 关键词: [目标关键词]

【发布选项】
- 状态: 草稿/发布
- 作者: [作者名]
- 允许评论: 是/否

立即发布"
```

方法B - 手动发布：
1. 登录WordPress后台
2. 文章 → 添加新文章
3. 粘贴内容（使用Markdown插件或复制粘贴）
4. 设置标题、分类、标签
5. 上传图片并添加Alt标签
6. 在Yoast SEO或Rank Math中设置Meta信息
7. 预览并发布

**步骤3: 发布后检查**

- [ ] 文章显示正常
- [ ] 图片加载正常
- [ ] 内部链接有效
- [ ] Meta信息正确
- [ ] 移动端显示正常

## 第2部分: 社交媒体自动化

### 2.1 平台授权与策略（30分钟）

#### 🎯 目标
选择合适的社媒平台并制定内容策略

#### 执行步骤

**步骤1: 选择社交平台**

| 平台 | 适合内容 | 发布频率 | 工作量 |
|------|---------|---------|--------|
| **Twitter/X** | 快速资讯、技巧 | 每天3-5条 | 中 |
| **LinkedIn** | 专业内容、B2B | 每周2-3篇 | 中 |
| **Instagram** | 视觉内容 | 每天1-2条 | 高 |
| **Facebook** | 综合内容 | 每天1-2条 | 中 |
| **TikTok** | 短视频 | 每周1-2个 | 高 |

**推荐组合**：
- B2B: LinkedIn + Twitter
- B2C: Instagram + Facebook + TikTok
- 内容营销: Twitter + LinkedIn

**步骤2: 授权平台**

使用Rube或Buffer授权：
1. 访问 https://rube.app
2. 连接社交平台账号
3. 测试发布一条内容

**步骤3: 制定30天内容策略**

```bash
"制定30天社交媒体内容策略：

【目标受众】
- 行业: [你的行业]
- 角色: [决策者/执行者]
- 痛点: [主要痛点]

【内容分布】
Twitter (每天3-5条):
- 资讯30% + 技巧30% + 互动20% + 推广10% + 转发10%

LinkedIn (每周2-3篇):
- 见解40% + 分析30% + 动态20% + 招聘10%

【内容日历】
创建30天发布计划，包含：
- 发布日期
- 内容类型
- 主题
- 发布平台

输出: 30天内容日历（保存到Notion）"
```

---

### 2.2 内容生成与自动发布（15分钟/周）

#### 🎯 目标
批量生成社媒内容并设置自动发布

#### 执行步骤

**步骤1: 基于文章生成社媒内容**

```bash
"基于以下文章生成社媒内容：

【源文章】
[文章URL或内容]

【生成要求】
Twitter (10条推文):
- 每条 < 280字符
- 包含话题标签（2-3个）
- 包含emoji
- 引导点击链接

LinkedIn (3篇帖子):
- 专业语气
- 包含见解和分析
- 300-500字
- 包含相关图片

【内容角度】
- 提炼文章核心观点
- 转化为 actionable tips
- 添加疑问引发互动
- 包含数据支持

立即生成"
```

**步骤2: 设置自动发布**

使用Rube或Buffer：
1. 创建发布计划
2. 设置发布时间：
   - Twitter: 每天 9am, 12pm, 3pm, 6pm, 9pm
   - LinkedIn: 每周二、周四 10am
3. 批量导入内容
4. 启用自动发布

**步骤3: 监控和互动**

- 每天查看通知
- 回复评论和私信
- 转发相关内容
- 分析互动数据

### 专题1: AI搜索优化 (GEO基础) ⭐

#### 核心洞察
**流量正在消亡，影响力正在崛起**

- 约59%的谷歌搜索是"零点击"
- ChatGPT只有不到2%会话会点击外部链接
- **停止优化流量，开始优化影响力**

#### GEO vs SEO

| **传统SEO** | **GEO (AI搜索优化)** |
|------------|---------------------|
| 优化排名 | 优化AI引用 |
| 获得点击 | 获得提及 |
| 流量为王 | 影响力为王 |
| 关键词匹配 | 语义清晰 |
| 10条蓝色链接 | 1个完整答案 |

#### 快速上手

**核心原则**：
1. **语义清晰** - 结构化内容，明确主题
2. **权威性** - 建立信任和声誉
3. **多模态** - 文本、图片、视频、音频

**平台差异化**：
- **ChatGPT**: 优化结构化数据和引用
- **Perplexity**: 优化实时信息和时效性
- **Gemini**: 优化多模态内容
- **Claude**: 优化深度分析和推理

**深入了解**: 📘 [GEO优化/GEO完全指南.md](../GEO优化/GEO完全指南.md)

### 专题2: 内容质量把控

#### 核心原则
**以人为本，而非搜索引擎**

Google并不惩罚AI内容，它惩罚的是"垃圾内容"

#### 质量标准

**❌ 搜索引擎优先（黑帽）**：
- 关键词堆砌
- 内容空洞
- 标题党
- 信息不完整

**✅ 以人为本（白帽）**：
- 分享真实经验
- 聚焦核心主题
- 分析背后原因
- 预见用户障碍

#### 质量检查清单

**内容准确性**：
- [ ] 事实正确
- [ ] 数据来源可靠
- [ ] 引用准确

**内容独特性**：
- [ ] 独特观点
- [ ] 原创内容
- [ ] 差异化角度

**内容实用性**：
- [ ] 解决具体问题
- [ ] 提供可执行建议
- [ ] 包含实例和案例

**技术质量**：
- [ ] SEO评分 > 90
- [ ] 可读性 > 60
- [ ] 无语法错误

#### 常见问题

**问题1: AI内容同质化**
- 解决: 加入真实案例和经验
- 解决: 提供独特观点

**问题2: 关键词堆砌**
- 解决: 自然语言优先
- 解决: 关键词密度1-2%

**问题3: 内容空洞**
- 解决: 增加具体数据和细节
- 解决: 添加真实案例

### 专题3: 数据科学驱动

#### 核心理念
**从"经验驱动"到"数据+AI驱动"**

#### 数据科学三支柱

1. **关键词数据** (Ahrefs/Semrush)
   - KD值
   - 搜索量
   - 竞争分析

2. **用户行为数据** (Google Analytics)
   - 跳出率
   - 停留时间
   - 转化率

3. **内容表现数据** (Search Console)
   - 排名
   - 点击率
   - 展现量

#### 数据闭环流程

```
数据收集 → 分析洞察 → 优化策略 → 效果验证 → 再次优化
```

#### 核心工具

- **Ahrefs**: 关键词研究、竞品分析
- **Google Search Console**: 排名和点击数据
- **Google Analytics**: 用户行为分析
- **Claude Code**: AI辅助分析

### 专题5: 追踪LLM流量

#### 核心挑战
**AI搜索引擎不产生传统"流量"**

#### 新KPI体系

**❌ 旧KPI（流量导向）**：
- 点击量(CTR)
- 访问量(Sessions)
- 跳出率
- 页面浏览量(PV)

**✅ 新KPI（影响力导向）**：
- 受众规模(Audience Size)
- 触达(Reach)
- 兴趣(Engagement)
- 销售额(Revenue)

#### 追踪方法

**1. Google Analytics自定义渠道**
```
创建"LLM Traffic"渠道
- 使用Regex识别AI搜索引擎
- 追踪展现量和转化率
```

**2. 品牌监控工具**
- Brand24: 品牌提及监控
- Mention: 社交媒体监控
- Google Alerts: 新闻和网页监控

**3. 社交媒体分析**
- 品牌搜索量
- 社交分享数
- 评论和互动

#### 实战建议

- ✅ 关注受众规模，而非点击量
- ✅ 优化影响力，而非排名
- ✅ 追踪品牌信号，而非流量

### 专题6: 2025最新趋势

#### 2025年重点趋势

**1. AI搜索全面崛起**
- ChatGPT搜索量增长300%
- Perplexity月活突破1000万
- Google AI Overviews覆盖50%搜索

**2. 零点击搜索突破60%**
- 用户不需要点击就能获得答案
- 品牌曝光变得更重要
- 直接搜索量增加

**3. 视频内容优先**
- 短视频搜索增长150%
- TikTok成为重要搜索引擎
- 视频Schema标记重要

**4. E-E-A-T更新**
- Experience（经验）权重增加
- 真实案例和体验更重要
- AI内容需要人工审核

#### 实战建议

**必做事项**：
- ✅ 投资GEO优化
- ✅ 建立品牌影响力
- ✅ 优化多模态内容
- ✅ 增加视频内容
- ✅ 积累真实案例

**避免事项**：
- ❌ 只关注传统SEO排名
- ❌ 忽视AI搜索
- ❌ 追求流量而非影响力
- ❌ 完全依赖AI生成内容
