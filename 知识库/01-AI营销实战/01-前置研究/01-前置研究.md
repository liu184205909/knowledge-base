# 01-前置研究：竞品分析与用户洞察

> **数字营销专用** | 第1章 | 所有营销活动的起点 | 必读 | v3.1

---

## 🎯 为什么需要前置研究？

**投入产出**：2小时调研 = 节省50%返工 + 提升3倍转化率

---

## 📋 完整流程（2小时）

```
阶段1: 竞品分析（1小时）
├─ 确定竞品名单（5分钟）
├─ 三类竞品分析（30分钟）
└─ 差距与机会分析（25分钟）

阶段2: 用户洞察（30分钟）
├─ 4维度洞察（20分钟）
└─ 洞察→机会转化（10分钟）

阶段3: 策略整合（30分钟）
├─ 制定整体策略（15分钟）
└─ 制定3-6个月路线图（15分钟）
```

---

## 🔍 阶段1: 竞品分析（1小时）

### 步骤1: 确定竞品名单（5分钟）

```bash
"帮我确定核心竞品名单

【我的业务】
[行业、产品、目标市场]

【竞品类型】
1. 直接竞品：相同产品/服务
2. 行业巨头：头部企业/大平台
3. 间接竞品：替代方案

【输出】
- 5-10个核心竞品
- 按优先级排序
- 标注竞品类型"
```

---

### 步骤2: 三类竞品分析（30分钟）

**使用工具**：Ahrefs/SEMrush/SimilarWeb

**分析维度**：

**1. 基础信息**
- 成立时间、融资、用户规模、定价

**2. PMF分析**
- 目标用户、产品定位、核心功能

**3. 营销数据** ⭐ 营销专用
- 内容：类型/频率/长度/质量
- SEO：流量关键词/月流量/外链数/DA
- 社媒：平台/粉丝数/互动率
- 外链：总数/来源/质量

【输出格式】
为每个竞品生成《竞品分析卡片》，包含：
- 基本信息
- PMF分析
- 营销策略（内容/SEO/社媒/外链）

---

### 步骤3: 差距与机会分析（25分钟）

```bash
"基于竞品分析，找出机会：

【我的业务】
[我的产品/服务]

【分析维度】
1. 内容机会：哪些关键词/话题竞品没覆盖？
2. SEO机会：哪些低竞争高价值关键词可以突破？
3. 渠道机会：哪些营销渠道竞品没利用？
4. 差异化：基于用户洞察，我们的独特价值是什么？

【输出】
- Top 10 内容机会
- Top 10 SEO机会
- 3-5个差异化角度"
```

---

## 👥 阶段2: 用户洞察（30分钟）⭐

### 4维度用户洞察（20分钟）

**分析平台**：Reddit/Quora/Twitter/Amazon评论/YouTube

**维度1: 查真实需求**
```bash
"分析用户真实需求：
【关键词】[关键词1, 关键词2]
【任务】
- 用户抱怨最多的痛点？
- 用户最常问的问题？
- 用户最想要的功能？
- 现有方案的不足？
【输出】Top 10痛点/需求/抱怨"
```

**维度2: 察行为动机**
- 驱动因素：为什么要做？
- 阻碍因素：为什么不做得更好？
- 情感触发：什么情况下强烈需要？
- 价值感知：愿意付出什么代价？

**维度3: 观场景痛点**
- 时间：什么时候最需要？急迫程度？
- 地点：什么环境？
- 设备：什么设备？有什么限制？
- 协作：个人还是团队？
- 压力：时间紧？预算少？

**维度4: 揣认知决策**
- 认知：如何知道问题存在？
- 搜索：如何搜索解决方案？（关键词/平台）
- 评估：用什么标准比较选项？
- 决策：最终影响购买的关键因素？
- 满意：使用后如何评好坏？

---

### 洞察→机会转化（10分钟）

```bash
"基于洞察，转化成机会：
【输入】4维度洞察结果
【任务】
1. 哪些洞察→内容选题？
2. 哪些洞察→SEO关键词？
3. 哪些洞察→独特卖点？
4. 如何触达这些用户？
【输出】
- Top 20内容选题
- Top 20关键词
- 3-5个差异化角度
- 营销策略建议"
```

---

## 🎯 阶段3: 策略整合（30分钟）

### 制定策略和路线图

```bash
"基于竞品分析和用户洞察，制定策略：

【输入】
- 竞品分析结果
- 用户洞察结果

【策略制定】
1. 目标定位：目标用户是谁？
2. 差异化价值：独特价值是什么？
3. 关键词策略：重点布局哪些关键词？
4. 内容策略：生产什么类型内容？
5. 渠道策略：重点使用哪些渠道？

【路线图】3-6个月计划
- Month 1: 基础内容（10篇文章）
- Month 2-3: SEO优化+社媒推广
- Month 4-6: 外链建设+深度内容

【输出】
- 整体营销策略（1000字）
- 月度计划+内容日历+KPI"
```

---

## ⚠️ 常见误区

**误区1: 跳过前置工作直接执行**
❌ 直接开始写文章/发社媒
✅ 先花2小时做竞对分析和用户洞察

**误区2: 只做一次前置工作**
❌ 做完就不更新了
✅ 每6个月重做，或进入新市场时重做

**误区3: 照搬竞品策略**
❌ 竞品做什么我们就做什么
✅ 基于用户洞察找差异化机会
