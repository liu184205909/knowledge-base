# MCP工具应用系统

> **扩展Claude Code能力的工具集成** | 需要配置，实现自动化增强

---

## 🎯 核心理念

**MCP (Model Context Protocol)** = 模型上下文协议

通过集成第三方工具，扩展Claude Code的能力边界，实现图片生成、视频制作、批量处理等功能。

```
无MCP工具:
Claude Code → 仅文本处理能力

有MCP工具:
Claude Code → 图片生成 → 视频制作 → Web自动化 → 社媒发布
```

---

## 📋 5个工具对比

| 工具 | 功能 | 实现方式 | 技术要求 | 适用场景 |
|-----|------|---------|---------|---------|
| **01-图片自动化** | AI配图生成 | MCP/Python/n8n | ⭐⭐ | 文章配图、产品图片 |
| **02-视频自动化** | 文章转视频 | Web/API/n8n | ⭐⭐ | 短视频制作、内容改编 |
| **03-批量生产流水线** | 批量内容生成 | Skills/Python/n8n | ⭐⭐⭐ | 大规模内容生产 |
| **04-Web自动化** | 浏览器自动化 | Dev Browser MCP | ⭐⭐ | 网站操作、数据抓取 |
| **05-社媒发布自动化** | 多平台发布 | Rube工具 | ⭐⭐ | 定时发布、内容分发 |

## 🔗 与其他模块的关系

```
00-RLM工作流编排 (主控Skills)
    ↓ 调用
03-MCP工具应用 (工具集成)
    ├─ 01-图片自动化 (被02-内容生产流水线调用)
    ├─ 02-视频自动化 (独立使用)
    ├─ 03-批量生产流水线 (可独立使用)
    └─ 05-社媒发布自动化 (被03-社媒营销全流程调用)
```

**关系说明**:
- RLM工作流是**主控层** (无需代码)
- MCP工具是**执行层** (需要配置)
- RLM可以引用MCP工具的文档，但不直接调用

## ⚙️ 技术实现

### MCP实现示例

```python
# 配置DALL-E MCP
# ~/.claude/claude_desktop_config.json

{
  "mcpServers": {
    "dalle": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-dalle"]
    }
  }
}
```

### Python脚本示例

```python
# 批量生成图片
import openai
openai.api_key = "your-api-key"

for i in range(10):
    response = openai.Image.create(
        prompt=f"Product image {i}",
        model="dall-e-3"
    )
    # 保存图片...
```

### n8n工作流示例

```
触发器 (定时) → 调用DALL-E API → 保存图片 → 发布到社媒
```
