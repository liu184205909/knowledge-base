## 🎓 学习路径

### 新手入门（0-1个月）

```
Week 1: 搭建环境
├─ 安装 Claude Code
├─ 安装 80+ 插件（1分钟）
├─ 安装 5 个核心 MCP（5分钟）
└─ 阅读快速开始（10分钟）

Week 2: 基础流程
├─ 营销方向: 前置研究 → 内容生产
└─ 开发方向: 产品前置工作 → feature-dev 流程

Week 3-4: 实践
├─ 完成 5 篇文章 + 社媒内容（营销）
├─ 完成 1 个功能开发（开发）
└─ 分析数据和效果
```

### 进阶提升（1-3个月）

```
Month 1: 专题学习
├─ GEO 快速开始
├─ Prompt 模板库熟练使用
└─ 创建自己的 Skills

Month 2: 深化
├─ GEO 完全指南（深度）
├─ 批量生产自动化
└─ 多 AI 协作模式

Month 3: 实战
├─ 结合实战案例学习
├─ 建立 6 个月内容计划
└─ 实现批量生产
```

### 专家精通（3-6个月）

```
阶段1: 系统学习
├─ 完整阅读 Claude Skills 完全指南
├─ 研究所有实战案例
└─ 掌握所有核心 MCP

阶段2: 自动化
├─ 图片自动化
├─ 视频自动化
├─ 批量生产流水线
└─ 建立自己的 Skills 库

阶段3: 持续优化
├─ 优化 ROI 和转化率
├─ 建立个人工作流 SOP
└─ 形成竞争壁垒
```
