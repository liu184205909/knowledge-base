# 跨电脑同步 - 快速开始

> 3分钟掌握跨电脑同步技能

---

## 🚀 新电脑首次使用（3步）

### 第1步：打开命令行
- 按 `Win + R`
- 输入 `cmd`
- 按回车

### 第2步：克隆仓库
```bash
cd "D:\Program Files (x86)\Project Code"
git clone https://github.com/liu184205909/knowledge-base.git 知识库
```

### 第3步：验证
```bash
cd 知识库
dir
```

看到 `README.md` 等文件就成功了！✅

---

## 🔄 日常工作流（2个命令）

### 工作开始：拉取最新更新
```bash
cd "d:\Program Files (x86)\Project Code\知识库"
git pull origin main
```

### 工作结束：推送本地修改
```bash
cd "d:\Program Files (x86)\Project Code\知识库"
git add .
git commit -m "update: 更新了某个文档"
git push origin main
```

## ⚠️ 常见错误

### 错误1：fatal: not a git repository

**原因**：不在知识库文件夹

**解决**：
```bash
cd "d:\Program Files (x86)\Project Code\知识库"
```

### 错误2：Please tell me who you are

**原因**：首次使用 Git 需要配置

**解决**：
```bash
git config --global user.email "your@email.com"
git config --global user.name "Your Name"
```

### 错误3：Your local changes would be overwritten

**原因**：本地有未提交的修改

**解决**：
```bash
# 方法1：提交本地修改
git add .
git commit -m "临时提交"
git pull origin main

# 方法2：放弃本地修改
git reset --hard HEAD
git pull origin main
```

## 🎓 进阶技巧

### 查看修改历史
```bash
git log --oneline -10
```

### 查看具体修改
```bash
git diff README.md
```

### 撤销某个文件的修改
```bash
git checkout -- README.md
```

### 暂存当前工作
```bash
git stash
git pull origin main
git stash pop
```
