## ⚙️ 配置要求

### MCP服务器（9个）
1. web-reader（网页读取）
2. web-search-prime（搜索）
3. zai-mcp-server（AI能力）
4. playwright（浏览器自动化）
5. firecrawl（网站爬取）
6. apify（数据抓取）
7. sequential-thinking（深度分析）
8. @magicuidesign/mcp（UI设计）
9. **rube**（500+应用集成）

### 核心工具
- **Rube MCP**: 连接Notion、Google Sheets、WordPress、社交媒体
- **Apify**: 爬虫和竞对分析
- **Firecrawl**: 深度网站爬取
