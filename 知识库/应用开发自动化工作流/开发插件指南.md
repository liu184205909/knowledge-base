## 📌 Claude Code Plugin System 简介

### 什么是 Plugin System？

**Claude Code Plugin System** 是官方推出的插件生态，让AI能够：
- 📦 **插件（Plugins）**：打包最佳实践和工具集
- 🎯 **技能（Skills）**：特定任务的专家级知识
- 🤖 **子代理（Sub Agents）**：独立的AI助手处理复杂任务
- 🔧 **Hooks**：在特定事件触发自定义操作
- 🔌 **MCP服务器**：集成外部工具和服务

### 快速开始

**1. 添加插件市场**：
```bash
# 方式1：通过命令行
claude plugin add https://github.com/anthropics/claude-plugins-official

# 方式2：通过VSCode界面
# Ctrl+Shift+P → "Claude Code: Add Plugin Market"
```

**2. 安装插件**：
```bash
claude plugin list                    # 查看可用插件
claude plugin enable <plugin-name>    # 启用插件
```

**3. 自动调用**：
安装后，AI会根据任务自动选择合适的插件，无需手动指定。

### 推荐插件

**开发必备**：
- **feature-dev** → 完整的功能开发工作流（需求→设计→开发→测试→发布）
- **code-review-ai** → AI代码审查，自动发现问题和优化点
- **python-development** / **javascript-typescript** → 语言开发支持

**效率提升**：
- **claude-mem** → 长期记忆，记住项目上下文和偏好
- **frontend-design** → 快速生成现代化UI组件
- **superpowers** → 增强AI能力，支持复杂多步骤任务

### 创建自己的插件

**1. 创建插件结构**：
```
my-plugin/
├── marketplace.json       # 插件元数据
├── prompts/               # 提示词目录
│   ├── task1.md
│   └── task2.md
└── README.md
```

**2. 配置 marketplace.json**：
```json
{
  "id": "my-plugin",
  "name": "My Plugin",
  "description": "我的自定义插件",
  "author": "Your Name",
  "version": "1.0.0"
}
```

**3. 发布到GitHub**：
- 推送到公开仓库
- 在插件市场中添加链接
- 其他用户即可通过 `claude plugin add` 安装
