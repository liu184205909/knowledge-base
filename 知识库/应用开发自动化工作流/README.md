## ⚙️ MCP服务器（9个）

1. **web-reader** - 网页读取（读取文档、API文档）
2. **web-search-prime** - 联网搜索（查找技术方案、Stack Overflow）
3. **zai-mcp-server** - AI能力（视觉理解、截图分析）
4. **playwright** - 浏览器自动化（E2E测试、爬虫）
5. **firecrawl** - 网站爬取（数据抓取）
6. **apify** - 数据抓取（爬虫和自动化）
7. **sequential-thinking** - 结构化思考（复杂问题分析）
8. **@magicuidesign/mcp** - UI设计（原型设计）
9. **rube** - 500+应用集成（第三方服务集成）

## ⚠️ 注意事项

### 前置工作优先级
- [OK] 新产品开发（必须先做前置工作）
- [OK] 大版本迭代（建议先做前置工作）
- [!] 小功能优化（可以跳过前置工作）

### 渐进式开发
- Week 1: 前置工作 + MVP设计
- Week 2-3: 核心功能开发
- Week 4: 测试 + 上线 + 收集反馈
