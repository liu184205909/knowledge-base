## 🔄 每日工作流

### 工作开始：拉取最新更新
```bash
cd "d:\Program Files (x86)\Project Code\知识库"
git pull origin main
```

### 工作结束：推送本地修改
```bash
cd "d:\Program Files (x86)\Project Code\知识库"
git add .
git commit -m "update: 描述修改内容"
git push origin main
```

## 🔍 常用命令

| 操作 | 命令 |
|------|------|
| 查看状态 | `git status` |
| 查看差异 | `git diff` |
| 查看日志 | `git log --oneline -5` |
| 撤销修改 | `git checkout -- <file>` |
| 暂存修改 | `git stash` |
| 恢复暂存 | `git stash pop` |

## 🆘 常见问题

### 拉取失败
```bash
git pull origin main --allow-unrelated-histories
```

### 推送失败
```bash
git fetch origin
git merge origin/main
```

### 放弃本地修改
```bash
git reset --hard HEAD
git pull origin main
```
