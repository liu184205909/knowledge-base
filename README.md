## 💡 核心方法论：RLM递归思想

### 什么是RLM？

**RLM (Recursive Language Model)** = 递归语言模型
= 让AI学会"分而治之"的设计思想

这不是一个新工具，而是一种**工作流设计方法**：
- 让AI像项目经理一样协调各个专业助手
- 通过Skills递归调用，完成复杂任务
- 一句话搞定，2-3分钟完成需要2小时的工作

### RLM工作流示例

**营销领域** (5个主控工作流)：
- **产品分析全流程**：选品→竞品→用户→内容 (调用4个Skills)
- **内容生产流水线**：选题→大纲→撰写→SEO→质检 (5个阶段)
- **社媒营销全流程**：创作→发布→分析→优化 (5个阶段)
- **批量选品分析**：评估→对比→排序→资源分配 (循环调用)
- **SEO优化全流程**：关键词→竞品→内容→技术→外链 (5个阶段)

**开发领域** (4种开发方式)：
- **Vibe-Coding思维**：流水线思维方法论
- **feature-dev流程**：单AI深度开发
- **Schaltwerk工具**：多AI协同开发
- **CSV+MCP系统**：自动化流水线
