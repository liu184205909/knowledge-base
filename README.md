## 💡 核心方法论：RLM递归思想

**RLM (Recursive Language Model)** = 递归语言模型 = 让AI学会"分而治之"的设计思想

**核心理念**:
- 让AI像项目经理一样协调各个专业助手
- 通过流程文档递归调用，完成复杂任务
- 一句话搞定，2-3分钟完成需要2小时的工作

**营销领域** (4个核心流程)：
- **01-市场分析**：产品评估+竞品分析+用户洞察
- **02-内容策略**：差异化策略+网站架构+SEO策略
- **03-博客内容生产**：SEO博客内容(1500-3000字)
- **04-社媒内容生产**：社媒平台内容(50-800字)

**开发领域** (4种开发方式)：
- **Vibe-Coding思维**：流水线思维方法论
- **feature-dev流程**：单AI深度开发
- **Schaltwerk工具**：多AI协同开发
- **CSV+MCP系统**：自动化流水线
