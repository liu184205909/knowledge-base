## 🎯 核心理念

> **Claude Skills = 个人经验的固化和标准化工作的 SOP**

本知识库的核心价值在于：
- 📦 **经验固化**：将实战经验、最佳实践固化成可复用的知识库
- 📋 **标准 SOP**：将工作流程标准化，确保每次执行都符合标准
- 🤖 **AI 自动执行**：Claude 自动识别场景，按标准执行
- 🔄 **持续迭代**：随着经验积累，不断优化 Skills

**简单来说**：把你脑子里的经验变成 AI 可以理解和执行的标准操作流程。
