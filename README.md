## 💡 核心方法论：RLM递归思想

**RLM (Recursive Language Model)** = 递归语言模型 = 让AI学会"分而治之"的设计思想

**核心理念**:
- 让AI像项目经理一样协调各个专业助手
- 通过Skills递归调用，完成复杂任务
- 一句话搞定，2-3分钟完成需要2小时的工作

**营销领域** (5个主控工作流)：
- **产品分析全流程**：选品→竞品→用户→内容
- **内容生产流水线**：选题→大纲→撰写→SEO→质检
- **社媒营销全流程**：创作→发布→分析→优化
- **批量选品分析**：评估→对比→排序→资源分配
- **SEO优化全流程**：关键词→竞品→内容→技术→外链

**开发领域** (4种开发方式)：
- **Vibe-Coding思维**：流水线思维方法论
- **feature-dev流程**：单AI深度开发
- **Schaltwerk工具**：多AI协同开发
- **CSV+MCP系统**：自动化流水线
