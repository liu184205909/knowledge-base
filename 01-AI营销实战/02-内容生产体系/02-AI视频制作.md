## 📊 AI视频制作全流程

### 标准工作流

```
第1步: 创意策划
├─ 确定主题和目标受众
├─ 选择视觉风格
└─ 规划视频时长

第2步: 剧本创作
├─ AI生成故事大纲 (ChatGPT/Kimi/DeepSeek)
├─ 完善剧本细节
└─ 输出分镜脚本

第3步: 角色与场景设计
├─ 生成角色设计图
├─ 建立主体库 (角色/道具/场景)
└─ 保持风格一致性

第4步: 分镜制作
├─ 根据剧本生成分镜图 (文生图)
├─ 优化构图和镜头语言
└─ 确认可用素材

第5步: 视频生成
├─ 图生视频 (推荐)
├─ 运镜控制
└─ 口型匹配 (如需要)

第6步: 后期制作
├─ AI配音
├─ AI配乐 (Suno/通义万相)
├─ 剪辑合成
└─ 添加字幕

第7步: 导出发布
├─ 质量检查
├─ 多平台适配
└─ 数据监控
```

### 三种实现方案对比

#### 方案A: 一站式平台 (适合新手) ⭐推荐

**工作流**:
```
输入故事 → MOKI自动生成脚本 → 选择视觉风格 → 生成分镜 → 生成视频 → 配乐 → 导出
```

**优势**:
- 无需学习多个工具
- AI自动完成所有环节
- 1-3天完成短剧

**劣势**:
- 定制化程度低
- 画质上限受限
- 人物一致性难保证

**适用**: 个人创作者、测试项目、快速迭代

#### 方案B: 专业工具组合 (适合进阶)

**工作流**:
```
剧本创作 (ChatGPT)
→ 角色设计 (Midjourney)
→ 分镜生成 (通义万相)
→ 视频生成 (Vidu Q2参考生视频)
→ 配音 (剪映)
→ 剪辑 (剪映/Premiere)
```

**优势**:
- 质量可控
- 灵活性高
- 可商业交付

**劣势**:
- 学习曲线陡
- 需要购买多个工具
- 制作周期长(7-30天)

**适用**: 商业项目、品牌片、精品内容

#### 方案C: 智能体工作流 (适合批量生产)

**工作流**:
```
设计定制化Bot
→ 自动拆解剧本
→ 自动生成提示词
→ 批量生成分镜
→ 批量生成视频
→ 自动剪辑导出
```

**优势**:
- 极高效率
- 一致性强
- 可规模量产

**劣势**:
- 需要技术能力
- 前期投入大
- 维护成本高

**适用**: 批量生产、工作室、专业团队

### 案例1: 《一掌熊猫》42分钟AI长片

**基本信息**:
- 时长: 42分钟
- 制作周期: 28天
- 团队规模: 10人
- 分镜数量: 2600+个

**核心工作流**:
```
1. 定制化智能体设计
   ├─ 脚本Bot: 自动拆解场次和镜头参数
   └─ Q2多参Bot: 适配Vidu Q2模型特性

2. 主体库建立
   ├─ 角色库: 主角熊猫侠
   ├─ 道具库: 武器/配饰
   ├─ 场景库: 废区/实验室
   └─ 特效库: 打斗光效

3. Vidu Q2参考生视频
   ├─ 从参考图到视频只需1步
   ├─ 压缩50%分镜制作时间
   └─ 20天工作量 → 7天完成

4. 动作戏处理
   ├─ 6分钟打斗戏耗时3天
   ├─ 动作精细拆解为3秒片段
   └─ 首尾帧功能避免拉伸
```

**关键突破**:
- ✅ 解决42分钟内角色长相一致
- ✅ 非线性工作流(剪辑和制作并行)
- ✅ 主体库批量生成连贯画面

- 2600个镜头 × $0.04 = $104 (约750元)

## 🎓 高阶技巧

### 1. 角色一致性保证

**方案A: Vidu Q2参考生视频**
```
建立主体库 (角色/道具/场景)
→ 生成定型参考图
→ Vidu Q2识别参考图
→ 批量生成不同景别/动态的连贯画面
```

**方案B: LoRA微调**
```
收集角色图片100+张
→ 训练角色LoRA模型
→ 应用于所有分镜生成
→ 确保风格统一
```

**方案C: 手动PS修图**
```
AI生成初稿
→ PS统一面部特征
→ 图生视频
```

### 2. 动作戏处理

**《一掌熊猫》经验**:
```
将连贯动作拆解为3秒片段
→ 使用短时长生成 (2秒)
→ 避免动作被"拉伸"
→ 剪辑拼接成完整打斗
```

**关键技巧**:
- 提示词必须设计动作逻辑
- 使用首尾帧功能
- 避免一次性生成复杂动作

### 3. 叙事逻辑控制

**挑战**: 短片求视觉连贯, 长片求逻辑通顺

**解决方案**:
```
1. 增加对白密度
   → 通过对话推动剧情
   → 声音情绪掩盖画面表演力不足

2. 剪辑和制作并行
   → 边做边改
   → 发现逻辑断裂及时补拍

3. 伏笔回应机制
   → 第30分钟剧情呼应第5分钟伏笔
   → 非线性工作流的优势
```

### 4. 成本控制策略

**分镜制作阶段**:
- 主体库前置准备: 压缩50%时间
- Vidu参考生视频: 省去手绘分镜环节

**视频生成阶段**:
- 图生视频 > 文生视频 (稳定性高)
- 批量生成 > 单个生成
- 使用免费额度 (可灵/即梦)

**配音阶段**:
- TTSMaker免费TTS
- 剪映自带配音
- RVC音色克隆

## ⚠️ 常见坑点与解决方案

### 1. 人物一致性差

**问题**: 不同镜头人物长相不一样

**解决方案**:
- ✅ 使用Vidu Q2参考生视频
- ✅ 建立主体库统一特征
- ✅ 训练角色LoRA模型
- ❌ 不要仅用文生视频

### 2. 动作生硬

**问题**: 人物动作不自然, 逻辑混乱

**解决方案**:
- ✅ 拆解为3秒短片段
- ✅ 使用首尾帧功能
- ✅ 精心设计动作提示词
- ❌ 不要一次性生成复杂动作

### 3. 剧情逻辑断裂

**问题**: 长片中前后矛盾

**解决方案**:
- ✅ 增加对白密度
- ✅ 剪辑和制作并行
- ✅ 及时补拍修正镜头
- ❌ 不要线性制作无法调整

### 4. 画质不达标

**问题**: 生成视频有崩坏/扭曲

**解决方案**:
- ✅ 优先用图生视频
- ✅ 手动PS优化分镜图
- ✅ 多生成多筛选
- ❌ 不要用第一次生成结果

### 5. 版权风险

**问题**: 未经授权改编小说

**解决方案**:
- ✅ 选择开放IP (番茄/阅文)
- ✅ 原创故事
- ❌ 不要直接改编热门小说
