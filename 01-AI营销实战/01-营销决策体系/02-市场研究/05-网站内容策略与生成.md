## 第一部分：网站内容策略（规划层面）

## 一、网站内容结构设计

### 1.1 标准网站内容架构

```
首页 (Homepage)
├─ Hero Section（首屏）
├─ 核心价值主张
├─ 核心产品/服务展示
├─ 信任信号（客户评价、数据）
└─ CTA（行动号召）

产品/服务页面 (Product/Service)
├─ 产品介绍
├─ 核心功能
├─ 使用场景
├─ 价格方案
└─ CTA

关于我们 (About Us)
├─ 品牌故事
├─ 团队介绍
├─ 使命愿景
└─ 价值观

其他页面
├─ 联系我们
├─ FAQ
├─ 博客（可选）
└─ 案例/成功故事（可选）
```

### 1.2 B2C 网站内容特点

**vs B2B 网站的区别**：

| 维度 | B2C 网站 | B2B 网站 |
|------|---------|---------|
| **内容风格** | 情感化、故事化 | 理性化、数据化 |
| **决策周期** | 短（冲动购买） | 长（多层级审批） |
| **重点内容** | 体验、情感、社会证明 | 功能、ROI、案例 |
| **CTA 类型** | 立即购买、加入购物车 | 预约演示、联系销售 |
| **信任信号** | 用户评价、社交媒体 | 行业认证、客户案例 |

## 二、首页内容策略

### 2.1 首页的5个核心要素

#### 要素 1: 清晰的价值主张 (Value Proposition)

**公式**：
```
价值主张 = 目标受众 + 核心利益 + 差异化优势
```

**示例**：
```
❌ 模糊的: "最好的宠物健康监测设备"

✅ 清晰的: "专为爱宠人士设计的智能健康监测项圈，
         让你随时了解宠物的健康状况，
         不同于普通项圈，我们提供兽医级准确度和AI健康预警"
```

**检查清单**：
- [ ] 是否在3秒内传达了核心价值？
- [ ] 是否针对具体受众？
- [ ] 是否说明了为什么选择我们？

#### 要素 2: 强有力的 Hero Section

**Hero Section 的结构**：
```
[主标题] - 抓住注意力，传达核心价值
   ↓
[副标题] - 补充说明，解决疑虑
   ↓
[CTA 按钮] - 明确的行动号召
   ↓
[产品图片/视频] - 展示产品或使用场景
   ↓
[信任信号] - 客户数量、评价、认证
```

**示例**：
```
主标题: "守护爱宠的每一刻"
副标题: "AI驱动的健康监测，7×24小时守护，异常情况实时预警"
CTA: "立即了解" / "观看演示"
信任信号: "已守护 50,000+ 只宠物"
```

#### 要素 3: 核心产品/服务展示

**展示原则**：
- ✅ **突出3个核心卖点**（不要超过5个）
- ✅ **每个卖点配图或短视频**
- ✅ **使用场景化描述**（而非功能列表）

**示例**：
```
❌ 功能列表:
- GPS追踪
- 健康监测
- 数据分析

✅ 场景化描述:
"实时定位：再也不用担心宠物走丢，GPS精度3米，
 支持电子围栏，越界立即推送通知"
```

#### 要素 4: 信任信号建立

**B2C 网站的信任信号类型**：

**社会证明 (Social Proof)**：
- 用户评价和评分
- 用户数量/客户数量
- 社交媒体粉丝数
- 用户生成内容（UGC）

**权威证明**：
- 行业认证
- 媒体报道
- 专家推荐
- 合作伙伴

**数据证明**：
- 具体数字（如："50,000+ 用户"）
- 效果数据（如："90% 用户在1周内看到改善"）
- 对比数据（如："比传统方法效率提升3倍"）

#### 要素 5: 多个 CTA (Call to Action)

**CTA 设计原则**：
- ✅ **明确的动词**（"立即购买" vs "了解更多"）
- ✅ **紧迫感**（"限时优惠" vs "随时购买"）
- ✅ **低门槛**（"免费试用" vs "立即购买"）
- ✅ **多位置**（Hero Section、产品介绍、页尾）

**CTA 文案示例**：
```
主页CTA:
- "立即购买，30天免费试用"
- "查看演示视频"
- "了解用户评价"

产品页CTA:
- "加入购物车"
- "选择套餐"
- "联系客服"

页尾CTA:
- "订阅优惠信息"
- "关注我们"
```

## 三、页面层级规划

### 3.1 一级页面（必须有的页面）

| 页面 | 目标 | 核心内容 | CTA |
|------|------|---------|-----|
| **首页** | 吸引注意，引导转化 | 价值主张、核心产品、信任信号 | 立即购买/了解更多 |
| **产品/服务** | 详细介绍，促成购买 | 功能、优势、价格、评价 | 加入购物车 |
| **关于我们** | 建立信任，情感连接 | 品牌故事、团队、使命 | 了解产品/联系我们 |

### 3.2 二级页面（根据需要添加）

| 页面 | 何时需要 | 核心内容 |
|------|---------|---------|
| **产品详情** | 多个产品 | 单个产品的详细介绍 |
| **博客** | 内容营销 | 行业文章、使用指南 |
| **案例/故事** | 需要证明效果 | 成功案例、用户故事 |
| **FAQ** | 复杂产品 | 常见问题解答 |
| **联系我们** | 需要客服支持 | 联系方式、表单 |

### 3.3 页面链接策略

**首页应该链接到的页面**：
```
Logo → 首页
导航栏 → 所有主要页面
Hero Section CTA → 产品页面
产品卡片 → 产品详情页
用户评价 → 案例故事页
页尾 → 所有主要页面 + 社交媒体
```

## 四、内容布局原则

### 4.1 F型和Z型浏览模式

**F型布局**（适合内容多的页面）：
```
用户浏览路径:
1. 水平阅读顶部
2. 垂直扫描左侧
3. 再次水平阅读（但不如第一次仔细）
4. 继续垂直扫描左侧

建议:
- 重要内容放在顶部和左侧
- CTA 按钮放在左侧
- 标题和图片沿F型放置
```

**Z型布局**（适合简单页面）：
```
用户浏览路径:
1. 左上 → 右上
2. 右上 → 左下（对角线）
3. 左下 → 右下

建议:
- Logo 在左上
- CTA 在右上
- 核心内容沿Z型放置
- 最终CTA在右下
```

### 4.2 移动端优先原则

**移动端内容布局**：
- ✅ **单栏布局**（不要使用多栏）
- ✅ **大按钮**（至少44×44px）
- ✅ **简洁导航**（汉堡菜单）
- ✅ **短段落**（2-3句话）
- ✅ **大标题**（H1至少30px）

### 4.3 视觉层次原则

**建立视觉层次的方法**：
1. **大小对比**：重要内容更大
2. **颜色对比**：CTA用醒目颜色
3. **空白对比**：重要内容周围留白
4. **位置对比**：重要内容放在顶部

## 五、转化路径设计

### 5.1 标准转化漏斗

```
首页访问者
    ↓ (点击 CTA)
产品浏览者
    ↓ (查看详情)
产品意向者
    ↓ (加入购物车)
购物车用户
    ↓ (开始结账)
结账用户
    ↓ (完成支付)
付费客户
```

### 5.2 降低流失的策略

**每个环节的优化**：

**首页 → 产品页**：
- ✅ CTA 文案清晰（"了解产品详情"）
- ✅ 产品卡片信息充足
- ✅ 快速加载（<3秒）

**产品页 → 购物车**：
- ✅ 明确的价格展示
- ✅ 清晰的购买按钮
- ✅ 多样化的支付方式

**购物车 → 结账**：
- ✅ 简化结账流程
- ✅ 游客结账选项
- ✅ 信任信号（安全认证）

### 5.3 微转化设计

**除了最终购买，还有其他转化目标**：
- ✅ **订阅邮件**（获取潜在客户）
- ✅ **下载白皮书**（获取联系方式）
- ✅ **观看演示视频**（增加了解）
- ✅ **联系客服**（回答疑问）

## 八、关于我们页面生成

### 8.1 关于我们页面的核心要素

**B2C 网站的"关于我们"应该包含**：

#### 要素 1: 品牌故事（情感连接）

```
起源故事
    ↓
问题/痛点
    ↓
灵感和使命
    ↓
解决方案
    ↓
影响和成就
```

```
 那时我才发现，宠物的健康问题往往发现得太晚。

 如果能早期发现，也许结果会不同。

 这个念头驱使我创立了PetGuard。
 我们的使命是：让每一位宠物主都能成为宠物的健康守护者。

 今天，PetGuard已守护50,000+只宠物，
 提前预警健康风险10,000+次，
 帮助无数家庭延长了与爱宠的相处时光。"
```

#### 要素 2: 团队介绍（可信度）

**B2C 团队介绍的重点**：
- ✅ **创始人的个人故事**

## 九、基于竞对网站生成内容

### 9.1 完整流程

**Step 1: 调用"竞对网站内容分析"Skill**

```
输入: 竞对网站 URL
输出: 竞对内容分析报告
```

**Step 2: 提取关键信息**

从竞对网站提取：
- 首页结构
- 价值主张
- 核心卖点
- CTA 策略
- 信任信号

**Step 3: 差异化分析**

识别：
- 竞对未提及的话题
- 竞对未覆盖的角度
- 竞对的弱点
- 我们的机会

**Step 4: 生成我们的内容**

基于竞对内容，但确保：
- ✅ 更强的价值主张
- ✅ 更好的内容结构
- ✅ 更有说服力的卖点
- ✅ 更有效的CTA
