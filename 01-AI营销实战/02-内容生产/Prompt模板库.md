## 1. 关键词研究Prompt

### 基础关键词研究

```
基于用户洞察研究关键词：

种子关键词: [种子关键词]

【目标】
- KD < 30（竞争度低）
- 搜索量 > 100/月
- 长尾关键词优先
- 符合用户真实需求

输出: Top 20 关键词列表，包含：
- 关键词
- KD值
- 搜索量
- 用户意图
- 推荐内容类型
```

## 2. 内容生成Prompt

### SEO文章生成

```
生成SEO优化文章：

【关键词】
[关键词]

【要求】
- 字数: 2000-2500字
- SEO优化: H1/H2/H3、Meta描述
- 关键词密度: 1-2%
- 包含: LSI关键词、内部链接
- 差异化角度: [基于用户洞察]

【质量检查】
- SEO评分 > 90
- 可读性 > 60
```

### 社媒内容生成

```
基于文章生成社媒内容：

【文章】
[文章URL或内容]

【生成要求】
Twitter (10条):
- < 280字符
- 包含话题标签
- 包含emoji

LinkedIn (3篇):
- 专业语气
- 300-500字
- 包含见解
```

## 3. 配图生成Prompt

### DALL-E图片生成

```
为文章生成配图：

【文章标题】
[标题]

【文章摘要】
[摘要200字]

【生成要求】
- 数量: 5张
- 风格: 专业、现代
- 格式: 16:9
- 关键词: [关键词]

【图片类型】
图片1: 封面图
图片2: 概念图
图片3: 对比图
图片4: 流程图
图片5: 总结图
```

### Alt标签生成

```
为图片生成Alt标签：

【图片】
[图片文件名]

【关键词】
[关键词]

【要求】
- 描述内容
- 包含关键词
- 10-15单词
- 清晰简洁
```

## 4. 视频脚本Prompt

### 文章转视频脚本

```
从文章提取视频脚本：

【文章】
[文章内容]

【脚本结构】(90秒)
- 0-5秒: 钩子
- 5-30秒: 观点1
- 30-55秒: 观点2
- 55-80秒: 解决方案
- 80-90秒: CTA

【要求】
- 口语化
- 包含数据
- 每部分20-30字
```

## 8. 质量检查Prompt

### SEO质量检查

```
检查文章SEO质量：

[文章内容]

【检查项】
✅ SEO评分 > 90
✅ 关键词密度 1-2%
```

4. **质量检查**: 生成后使用质量检查Prompt验证
