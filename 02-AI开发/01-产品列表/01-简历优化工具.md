# 产品候选：AI简历优化工具

> **一句话描述**：上传简历 + 目标岗位 → AI生成匹配度分析和优化建议
>
> **方法论来源**：[AI产品判断方法论-Cal-AI案例](../01-实战辅助/06-AI产品判断方法论-Cal-AI案例.md)

---

## 一、产品想法

**核心流程**：
```
用户上传简历（PDF/Word/粘贴文本）
  → 输入目标岗位（职位名 + JD）
  → AI分析匹配度 + 生成优化建议
  → 用户获得可直接使用的优化版简历
```

**本质**：把"改简历"从5步压缩到1步。

---

## 二、5条原则验证

### 原则1：成熟市场？ — 通过

| 证据 | 说明 |
|------|------|
| 已有付费产品 | TopResume、Resume.io、Kickresume、Teal、Resume Worded |
| 用户搜索量大 | "resume builder"/"resume review" 常年高搜索量 |
| 付费习惯已建立 | TopResume 单次$149-$219，Resume Worded Pro $49/月 |
| 痛点明确 | 用户知道自己简历不好，但不知道怎么改 |

### 原则2：AI减动作？ — 通过

| 旧流程（5步） | AI简化（1步） |
|--------------|-------------|
| 1. 搜索简历模板 | 上传简历 + 粘贴JD |
| 2. 阅读修改建议文章 | → AI直接输出匹配度+改写建议 |
| 3. 逐条对比修改 | → 一键生成优化版 |
| 4. 检查格式/排版 | |
| 5. 反复调整措辞 | |

**减少的动作**：少搜索（不用找模板）、少判断（AI直接告诉改哪里）、少打字（AI帮你重写）。

### 原则3：功能窄、卖点强？ — 通过

**MVP只做一件事**：简历 vs 岗位JD的匹配度分析 + 优化建议。

**不做的**（至少MVP阶段不做）：
- 不做求职全流程管理
- 不做面试准备
- 不做薪资谈判
- 不做LinkedIn优化

**wedge切口**："你的简历和这个岗位匹配吗？" — 一个足够尖的问题。

### 原则4：容易演示？ — 通过

**10秒演示画面**：
```
左侧：上传一份普通简历
右侧：粘贴一个JD
中间：点击"分析"
结果：匹配度73% + 3个关键修改建议 + 优化前后对比
```

TikTok/小红书短视频天然适合："HR看了你的简历会怎么想？"

### 原则5：onboarding付费路径？ — 通过

```
用户进入
  → 选择身份（应届生/转行/晋升/跳槽）
  → 粘贴目标岗位JD
  → 上传当前简历
  → 免费展示匹配度评分（让用户看到问题）
  → "查看具体修改建议" → 付费
```

**强动机品类**：找工作是高焦虑+高动机场景，用户愿意为"提高拿到面试的概率"付费。

## 四、竞品速览

| 产品 | 定价 | 核心功能 | 不足 |
|------|------|---------|------|
| TopResume | $149-$219/次 | 人工改简历 | 贵、慢（7-14天） |
| Resume Worded | $49/月 | AI评分+建议 | 不针对具体岗位JD |
| Teal | $9-40/月 | AI简历+求职追踪 | 功能太多，焦点分散 |
| Kickresume | $5-15/月 | AI生成简历 | 偏模板，分析弱 |

**差异化机会**：
- 现有产品大多是"通用简历评分"或"模板填充"
- **缺少"简历 vs 具体岗位JD"的精准匹配分析**
- 价格区间有空白：$5-15太便宜不值做，$149太贵用户犹豫，**$19-49单次**有空间

## 五、MVP方案

### 核心功能（P0）

1. **简历上传**：支持PDF/Word/纯文本
2. **JD匹配分析**：粘贴岗位JD → 匹配度评分 + 按维度拆解（技能/经验/关键词）
3. **优化建议**：针对不匹配项给出具体改写建议
4. **一键优化**：AI直接生成优化版简历

### 技术可行性

| 维度 | 方案 |
|------|------|
| 简历解析 | PDF → 文本提取（PyPDF2 / pdfplumber） |
| 匹配分析 | LLM对比分析（GPT-4 / Claude） |
| 优化生成 | LLM重写 + 关键词优化 |
| 前端 | Next.js / 简单Web App |
| 部署 | Vercel + Serverless |

### 商业模式

| 方案 | 定价 | 说明 |
|------|------|------|
| 免费层 | 0 | 1次免费分析（只看匹配度，不看建议） |
| 单次 | $9.9 | 完整分析+优化建议 |
| 包月 | $29 | 无限次分析+优化 |
| 求职套餐 | $49 | 简历+Cover Letter+面试准备 |

### 技术栈建议（AI开发）

```
前端：Next.js + Tailwind CSS
后端：Next.js API Routes / Python FastAPI
AI：OpenAI GPT-4 / Claude API
部署：Vercel / Cloudflare Pages
支付：Stripe / LemonSqueezy
```

## 七、风险与注意事项

| 风险 | 严重度 | 应对 |
|------|-------|------|
| AI生成内容不准确 | 高 | 明确标注"建议人工复核"，不做"保证通过"承诺 |
| 隐私合规（简历含个人信息） | 高 | 明确数据使用政策，提供删除功能，不做数据留存 |
| 竞品快速跟进 | 中 | 先占SEO位+社媒内容壁垒，速度是关键 |
| 低复购（找到工作就不再用） | 中 | 求职套餐提升客单价，覆盖一次就够了 |
| 简历格式复杂（表格/图片） | 中 | MVP先支持纯文本，后续迭代PDF解析 |
