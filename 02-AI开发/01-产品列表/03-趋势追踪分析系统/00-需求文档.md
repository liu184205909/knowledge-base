# 趋势追踪分析系统

> 自动发现值得建站/做产品的关键词机会

## 项目背景

基于 [Keywords Everywhere Trends](https://trends.keywordseverywhere.com/) 的趋势数据，自动分析并推荐有商业价值的关键词机会。

---

## 核心目标

| 目标 | 说明 |
|------|------|
| **发现机会** | 从海量趋势中筛选出值得入场的关键词 |
| **可行性分析** | 自动评估竞争度、商业价值、SEO难度 |
| **数据沉淀** | 历史数据存储，支持回溯和对比 |

---

## 数据来源

**Keywords Everywhere Trends**
- 网址：https://trends.keywordseverywhere.com/
- 数据量：8349+ 趋势
- 费用：目前免费（Beta）

---

## 筛选条件

| 维度 | 条件 | 说明 |
|------|------|------|
| 5年增长 | ✅ 正增长 | 长期趋势，非昙花一现 |
| 1年增长 | ✅ 正增长 | 中期持续，市场稳定 |
| 3个月增长 | ✅ 正增长 | 近期加速，值得现在入场 |

**筛选逻辑**：三个维度都增长才纳入分析范围

---

## 可行性分析维度

| 维度 | 分析方法 | 价值 |
|------|----------|------|
| 搜索竞争度 | 谷歌搜索结果数量 | <100万 = 机会较大 |
| 商业意图 | 是否有广告/购物结果 | 有 = 有钱赚 |
| 内容类型 | AI判断：产品/工具/信息站 | 决定做什么类型的站 |
| 域名可用 | .com 是否可注册 | 品牌化考虑 |
| 竞品分析 | 前10名网站权重 | 评估SEO难度 |

---

## 呈现方式

**网站**（主）
- 今日推荐（Top 10）
- 趋势列表（可筛选/排序）
- 单条详情（完整分析）
- 历史记录（按日期查看）

**TG 推送**（辅）
- 每天通知：有新数据更新

---

## 技术方案

| 组件 | 技术 |
|------|------|
| 数据抓取 | OpenClaw |
| 数据存储 | SQLite |
| 可行性分析 | AI（Claude/其他） |
| 网站 | Python + Streamlit（快速版）或 Next.js（正式版） |
| 推送 | Telegram Bot |

---

## 整体流程

```
每天定时（OpenClaw）
    ↓
抓取趋势数据
    ↓
筛选（5y+1y+3m 都增长）
    ↓
去重（排除已分析的）
    ↓
可行性分析（AI）
    ↓
存入数据库 + 更新网站
    ↓
TG 推送通知
```

---

## 数据结构

### 趋势表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name_en | TEXT | 趋势名称（英文） |
| name_zh | TEXT | 趋势名称（中文） |
| growth_5y | TEXT | 5年增长 |
| growth_1y | TEXT | 1年增长 |
| growth_3m | TEXT | 3个月增长 |
| search_competition | INTEGER | 搜索竞争度 |
| commercial_intent | INTEGER | 商业意图评分 |
| content_type | TEXT | 产品/工具/信息站 |
| domain_available | BOOLEAN | .com是否可注册 |
| seo_difficulty | INTEGER | SEO难度评分 |
| overall_score | INTEGER | 综合评分 |
| status | TEXT | 待评估/推荐/放弃 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |
