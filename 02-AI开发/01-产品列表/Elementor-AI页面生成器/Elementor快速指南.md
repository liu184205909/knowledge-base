# 如何用AI生成Elementor页面

> **适用于**: Elementor 3.0+ + WoodMart主题
> **更新日期**: 2026-02-03

---

## 🎯 你只需要做一件事

**描述你想要的页面** → AI生成JSON → 导入到Elementor → 完成

就这么简单！

---

## 📝 最简单的请求方式

### 标准格式：
```
创建一个[页面类型]页面，包含：
1. [区块1名称] - [简要描述]
2. [区块2名称] - [简要描述]
3. [区块3名称] - [简要描述]
主色调：[颜色]
```

### 实际示例：

#### 示例1：电商首页
```
创建一个电商首页，包含：
1. Hero - 全屏背景，标题"精选商品"，购物按钮
2. 产品展示 - 产品网格（4列），8个产品
3. 分类导航 - 产品分类图标
4. 促销Banner - 2个促销横幅
5. 特色服务 - 信息框（4列）
6. 客户评价 - 证言轮播
7. CTA - 联系表单
主色调：紫色 #6610f2
```

#### 示例2：落地页
```
创建一个产品落地页，包含：
1. Hero - 标题"革命性新产品"，两个按钮
2. 特性展示 - 3列图标卡片
3. 产品截图 - 大图展示
4. 客户评价 - 3条证言
5. CTA - "免费试用"按钮
主色调：蓝色 #007bff
```

#### 示例3：服务公司
```
创建一个服务公司首页，包含：
1. Hero - 标题"数字化转型专家"
2. 服务介绍 - 3个服务（交替布局）
3. 数据展示 - 4列大数字统计
4. 作品展示 - 6个项目卡片
5. CTA - 联系我们按钮
主色调：深蓝 #007bff
```

## 🚀 生成后如何使用

### 步骤1：AI生成JSON
根据你的描述，AI生成完整的JSON代码

### 步骤2：保存文件
复制JSON内容，保存为 `.json` 文件（例如：`homepage.json`）

### 步骤3：导入到Elementor
```
WordPress后台 → 页面 → 新建页面
→ 点击"使用Elementor编辑"
→ 点击"添加模板"图标（文件夹图标）
→ 上传JSON文件
→ 插入到页面
```

### 步骤4：编辑和发布
在Elementor编辑器中：
- 修改文字内容
- 替换图片
- 调整颜色和样式
- 添加/删除组件
- 点击"发布"

## ❓ 常见问题

### Q: 我不懂代码怎么办？
**A:** 不需要懂代码，只需要用自然语言描述。

### Q: 生成后能修改吗？
**A:** 完全可以！在Elementor中像普通页面一样编辑。

### Q: 支持哪些组件？
**A:** 支持所有Elementor官方组件 + WoodMart电商组件。

### Q: 响应式设计？
**A:** AI会自动生成适配移动端的布局。

### Q: 图片从哪里来？
**A:** AI使用占位图片，你在Elementor中替换成自己的。
