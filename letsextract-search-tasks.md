## 执行建议

1. **操作方式**：每组关键词 + 城市直接批量粘贴到工具里，一次跑完该组全部组合
2. **每天执行量**：建议每天跑 1-2 组（避免Google封IP）
3. **优先跑的顺序**：
   - 第1天：P0-A（压铸核心，150条）
   - 第2天：P0-B（汽车零部件，180条）
   - 第3天：P1-A + P1-B（卫浴五金，149条）
   - 第4天：P2 + P3（EV + 通用铸造）
   - 第5天：P0-C + P4（灰铁铸造 + 钣金）
   - 第6天：P5 + P6（航空 + 医疗）
4. **去重**：不同组之间会有重复公司，最终合并去重
5. **Google Maps 优先**
6. **墨西哥优先**：Monterrey/Saltillo 密度最高，接受度最好
