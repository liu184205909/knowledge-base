**5大Presence KPI**：

| KPI | 回答的问题 | 计算方法 |
|-----|-----------|---------|
| Prompt Coverage | 品牌出现的prompt比例 | 出现数 / 总追踪数 |
| Recommendation Rate | 被推荐vs仅被提及 | 推荐次数 / 出现次数 |
| Linked Citation Rate | 引用是否可点击 | 带链接次数 / 出现次数 |
| Comparative Win Rate | 比较中获胜 | 被偏好次数 / 比较中出现次数 |
| Representation Accuracy | 描述是否准确 | 准确次数 / 总出现次数 |

**最小可行AI搜索优化工作流**：
1. 选择30-50个商业相关prompt，按产品/服务/市场/角色分组
2. 在2-3个最相关AI搜索平台上运行
3. 记录品牌是否出现、被推荐、被链接、被引用、被准确描述
4. 捕获被引用或明显影响回答的来源
5. 识别竞品出现在哪里以及原因
6. 将缺口映射到三大修复类型：自有内容 / 第三方来源 / 数据实体
7. 优先级公式：**可见性缺口严重程度 × 业务重要性 × 影响可能性 / 实施工作量**

## 二、Ghost Citation：AI引用的隐藏真相

> 来源：Kevin Indig × Semrush，3,981域名 × 115提示 × 14国家 × 4引擎

### 什么是Ghost Citation

当AI使用你的内容回答问题时，它通常会给来源链接。但**62%的情况下，它不会说出你的品牌名**。链接在，品牌名不在。

| 指标 | 数值 |
|------|------|
| 被引用（有源链接）的域名 | 74.9% |
| 被提及品牌名的域名 | 38.3% |
| Ghost Citation比例（有链接但零品牌提及） | **61.7%** |
| 同时获得引用+提及 | 仅13.2% |

### 四引擎行为完全不同

| 引擎 | 引用率 | 品牌提及率 | 行为模式 | 策略方向 |
|------|--------|-----------|---------|---------|
| **Gemini** | 21.4% | **83.7%** | 像对话者，直接提及品牌 | **品牌知名度建设** |
| **ChatGPT** | **87.0%** | 20.7% | 像学术论文，给脚注不提品牌 | **SEO排名 + 结构化内容** |
| **Google AI Overviews** | 中等偏高 | 中等 | 偏向引用 | **传统SEO + Schema** |
| **Google AI Mode** | 类ChatGPT | 比ChatGPT多~17% | 更接近学术+对话混合 | **综合策略** |

**核心行动**：Gemini策略和ChatGPT策略完全不同。任何跨LLM聚合的"AI可见性指标"都有误导性。

### 内容类型与引用模式

| 内容类型 | 引用模式 | 品牌提及率 | 策略建议 |
|---------|---------|-----------|---------|
| 比较类（evaluation/comparison） | 产生品牌提及 | 高 | **品牌曝光首选** |
| 信息类（informational） | 匿名喂养AI | 极低 | 适合"被引用"而非"被提及" |
| 列表文章（Listicles） | 占AI引用52.9% | 中等 | **引用率最高格式**（但窗口正在关闭） |

**引用分布 — Ski Ramp模式**（Kevin Indig, 1.2M ChatGPT引用分析）：

| 页面位置 | 引用占比 |
|---------|---------|
| 前1/3 | **44.2%** |
| 前半部分 | **78.4%** |
| 后半部分 | 21.6% |

→ **答案必须前置**：44.2%的引用来自前30%内容

### 国家差异

- 印度/瑞典：提及率最高（50%）— 更对话化的查询模式
- 意大利/巴西/荷兰：提及率最低（18-22%），引用率极高（82-94%）

## 五、Perplexity优化 — AED内容模板

> 来源：ZipTie.dev深度框架、Erlin AI排名因素

### Perplexity关键数据

| 指标 | 数值 |
|------|------|
| 月查询量 | 7.8亿次（2025.5），同比增长239% |
| 与Google Top 10重叠率 | 60%（**40%来自Google之外**） |
| AI流量转化率 | 14.2%（Google有机仅2.8%，**5倍差距**） |
| AI访客停留时间 | 9:19（Google 5:33，高出67.7%） |

### AED内容模板（Answer-Evidence-Depth）

每个优化段落三段式结构：

```
Answer（前50词）：直接、独立完整的回答。读者只看这句话就能得到完整答案
  ↓
Evidence（接下来100-150词）：支撑数据、统计、来源引用
  ↓
Depth（剩余内容）：扩展上下文、示例、边界情况、相关概念
```

**引用率提升数据**：
- 被引用内容比未被引用内容多**32%的显式概念**
- Q&A格式将引用率提高约**3倍**
- 包含引用和统计的内容可见性提升约**40%**

### 60天新鲜度循环

| 内容类型 | 刷新频率 | 最低可行更新 |
|----------|---------|-------------|
| 趋势/新闻 | 每周或48小时内 | 新数据 + 时间戳 |
| 竞争性常青内容 | 每60天循环 | 更新统计 + "As of [date]" |
| 参考/信息性 | 每季度 | 时间戳 + 一个新数据点 |
| 低竞争常青 | 半年 | 可见日期更新 |

**2小时前更新的内容比一个月前更新的内容引用率高38%。**

### Perplexity引用排名因素

1. **语义相关性**（最关键）— 概念完整性和实体关系密度
2. **来源在回答中被直接引用**（非常重要）
3. **新鲜度**（高优先级）— 50%引用来自当年内容
4. **可读性和结构** — Q&A格式、数据块、对比表格、编号步骤

## 七、llms.txt — AI爬虫引导新规范

### llms.txt ≠ robots.txt

| | robots.txt | llms.txt |
|---|-----------|---------|
| **功能** | 控制爬虫访问（允许/禁止） | 引导AI模型获取最相关内容 |
| **位置** | /robots.txt | /llms.txt |
| **强制性** | 爬虫"应该"遵守 | 完全非强制 |
| **类比** | 门禁系统 | 藏宝图 |

### 采用情况

- **844,000+** 网站已采用
- 品牌包括：Anthropic、Vercel、Stripe、Cloudflare、Samsung、KIA、AstraZeneca

### 实操要点

- **目前没有直接证据**能提升Google AI Overview或ChatGPT引用排名
- 但作为**新兴最佳实践**被广泛推荐
- 电商可用：引导AI发现和展示产品
- 实施步骤：创建文件 → 定义AI代理目标 → 允许高价值内容 → 禁止低价值页面

## 八、Claude SEO Prompt执行库

> 资源来源：Sarvesh Shrivastava (@bloggersarvesh) 20+ prompt模板、Charles_SEO 10 prompt系统、社区实践

### 使用原则

1. **Claude Cowork（Computer Use）**是这些prompt的最佳运行环境，需Claude Pro/Team

### 8.1 AI可见性诊断Prompt

```
你是一位AI搜索优化专家。请分析以下品牌在AI搜索引擎中的表现。

品牌名：[品牌名]
网站：[URL]
目标品类：[品类]
目标市场：[国家/语言]

请执行以下诊断：

1. **Prompt测试矩阵**
   生成20个目标prompt，按以下维度分组：
   - 推荐类（"Best [品类] for [场景]"）
   - 对比类（"[品牌] vs [竞品]"）
   - 信任类（"Is [品牌] legit?"）
   - 解答类（"How to choose [品类]"）
   每组5个，覆盖用户旅程各阶段。

2. **存在感评分框架**
   对每个prompt，评估：
   - 是否出现（0/1）
   - 被推荐还是仅被提及（0-2分）
   - 是否有可点击链接（0/1）
   - 比较中是否胜出（0/1）
   - 描述是否准确（0/1）
   总分 = /6

3. **差距诊断**
   将每个缺口映射到修复类型：
   - A类：自有页面/内容问题
   - B类：第三方来源生态问题
   - C类：数据/实体/商业清晰度问题

4. **优先级排序**
   可见性缺口严重程度 × 业务重要性 × 影响可能性 / 实施工作量

输出格式：Markdown表格 + 行动清单（按优先级排序）
```

### 8.2 Ghost Citation检测Prompt

```
分析[品牌名]在AI搜索中的Ghost Citation问题。

步骤1：在以下4个平台分别搜索10个与[品牌]相关的prompt
- ChatGPT
- Gemini
- Google AI Overviews
- Perplexity

步骤2：对每个回答记录：
| 平台 | Prompt | 是否被引用 | 是否被提及品牌名 | Ghost Citation? | 回答中的措辞 |
|------|--------|-----------|----------------|----------------|------------|

步骤3：计算各平台的：
- 引用率（cited率）
- 品牌提及率（mentioned率）
- Ghost Citation率 = 引用但未提及的比例

步骤4：针对Ghost Citation率高的平台，制定差异化策略：
- ChatGPT（学术模式）→ 优化：结构化引用格式、数据支撑、权威来源
- Gemini（对话模式）→ 优化：品牌故事、比较内容、第三方背书

输出：分平台的优化策略清单
```
