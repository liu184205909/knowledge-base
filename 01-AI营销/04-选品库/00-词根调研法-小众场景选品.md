# 词根调研法：小众场景 × 功能组合选品

> **核心思路**：用英文词根在亚马逊/电商平台搜索，发现小众场景下被忽视的高利润产品。小众场景竞争少、用户价格敏感度低、功能组合创新空间大。（来源：[小渔聊独立站](https://mp.weixin.qq.com/s/jMkpc8iDj--vyUI9PA5PnQ)）

---

## 1. 方法论

```
选一个词根 → 在 Amazon 搜索 → 逐个翻看产品 → 记录高毛利/低销量/功能组合机会 → 筛选验证
```

**为什么词根有效**：
- 词根 = 场景/功能/问题的英文表达，直接命中用户搜索意图
- 同一词根下往往覆盖多个品类，跨品类发现机会
- 小众词根的搜索结果少，竞争低，容易被忽视

**选词根的原则**：
1. 描述一个**具体问题或场景**（emergency、survival）
2. 描述一个**重复发生的动作**（calculate、filter、dispense）
3. 描述一个**需要周期性更换的消耗**（replacement、refill）
4. 描述一个**特定的用户群体**（orthopedic、ergonomic）

---

## 2. 词根清单

### 2.1 场景/问题类

> 描述一个"出问题时需要什么"的场景

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Emergency** | 应急 | 充电宝+手电+收音机+工具组合 | 户外应急电源套装 |
| **Survival** | 生存 | 户外生存工具包、净水器 | 72小时生存背包 |
| **Outage** | 停电/中断 | 备用照明、手摇发电设备 | 停电应急灯 |
| **Preparedness** | 防备 | 灾难预备套装 | 地震/飓风应急箱 |
| **Disaster** | 灾难 | 灾后恢复工具 | 防水文件袋+应急毯 |

**规律**：场景越紧急，用户价格敏感度越低。"Emergency"类产品可以大胆做功能组合。

### 2.2 替换/维修类（高频复购）

> 用户必须定期更换的消耗品，剃须刀模式

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Replacement** | 替换件 | 滤网、刷头、刀片、配件 | 空气净化器替换滤网 |
| **Refill** | 补充装 | 洗手液、墨水、肥皂 | 自动出液器补充装 |
| **Repair** | 维修 | 维修工具包、修复胶 | 屏幕修复套装 |
| **Spare** | 备件 | 备用零件、备用电池 | 备用遥控器 |
| **Upgrade** | 升级 | 原厂配件的升级替代 | 更好的替换刷头 |

**规律**：Replacement 类产品一旦进入用户的购买循环，复购极稳定。关键是找到"原厂贵、替代便宜"的价格差。

### 2.3 功能/动作类

> 描述产品"做什么"，一个词根下往往有多个细分场景

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Calculate** | 计算 | 细分计算器（厨房、编织、建筑、税） | 编织针数计算器 |
| **Convert** | 转换 | 适配器、转换器（电压、接口、单位） | 欧洲旅行电源转换器 |
| **Correct / Correction** | 矫正 | 体态、视力、牙齿、脚趾 | 儿童矫正鞋 |
| **Detect / Detector** | 检测 | 水质、甲醛、辐射、金属 | 家用水质检测笔 |
| **Sterilize / Sterilizer** | 消毒 | UV消毒盒、蒸汽消毒 | 手机UV消毒盒 |
| **Compress / Compression** | 压缩 | 收纳袋、运动护具、医疗 | 旅行压缩收纳袋 |
| **Filter / Filtration** | 过滤 | 水过滤、空气过滤、油过滤 | 龙头净水器 |
| **Dispense / Dispenser** | 分配 | 肥皂、食物、水、纸巾 | 宠物自动喂食器 |
| **Seal / Sealing** | 密封 | 真空封口、防水密封 | 食物真空封口机 |
| **Insulate / Insulation** | 隔离 | 隔热、隔音、保温 | 窗户隔热膜 |
| **Charge / Charger** | 充电 | 细分充电场景 | 太阳能背包充电器 |
| **Mount / Mounting** | 固定/安装 | 手机架、显示器臂、工具挂架 | 床头手机支架 |
| **Amplify / Amplifier** | 放大/增强 | 声音、信号、视觉 | 老年人电视声音放大器 |
| **Automate / Automation** | 自动化 | 定时器、智能开关、自动装置 | 自动浇花器 |

**规律**：一个动作 × 不同场景 = 多个产品机会。"Calculate"不只是计算器，是"编织计算器"、"面包烘焙计算器"、"混凝土配比计算器"。

### 2.4 人群/健康类

> 针对特定人群设计，小众但刚需

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Ergonomic** | 人体工学 | 办公椅、键盘、鼠标、厨房工具 | 站立办公桌转换器 |
| **Orthopedic** | 骨科/矫形 | 鞋垫、枕头、坐垫、护膝 | 狗狗骨科床 |
| **Corrective** | 矫正 | 姿势矫正器、脚趾分离器 | 驼背矫正带 |
| **Therapeutic** | 治疗 | 热敷、冷敷、按摩 | 颈椎治疗仪 |
| **Pediatric** | 儿科 | 婴儿专用产品 | 婴儿指甲修剪器 |
| **Senior / Geriatric** | 老年 | 辅助设备、安全产品 | 老年人防滑洗澡椅 |
| **Rehab / Rehabilitation** | 康复 | 术后恢复、运动康复 | 膝盖康复训练器 |

**规律**：健康类产品的共同特征——用户不会比价，因为"健康没有价格"。毛利极高。

### 2.5 属性/形态类

> 在任何品类上叠加这些属性，都能创造差异化

| 词根 | 含义 | 叠加逻辑 | 案例 |
|------|------|---------|------|
| **Portable** | 便携 | 任何产品都有便携版本的需求 | 便携洗衣机 |
| **Foldable** | 可折叠 | 节省空间 = 小户型/旅行刚需 | 折叠菜板 |
| **Collapsible** | 可收缩 | 同 foldable，更极端节省 | 收缩硅胶水壶 |
| **Waterproof** | 防水 | 户外、浴室、海洋场景 | 防水蓝牙音箱 |
| **Solar** | 太阳能 | 离网场景 = 便携+应急 | 太阳能充电宝 |
| **Adjustable** | 可调节 | 一款产品覆盖多个用户 | 可调节桌面高度 |
| **Heavy-duty** | 重型 | 专业/工业级 = 高价 | 重型犬链 |
| **Compact** | 紧凑 | 小空间优化 | 紧凑型空气净化器 |

**规律**：属性词根不单独使用，要和品类词根组合搜索。例如 "portable + washing machine"、"foldable + cutting board"。

## 3. 组合搜索公式

```
场景词根 + 品类 = 产品机会
功能词根 + 人群 = 细分市场
属性词根 + 品类 = 差异化
替换词根 + 品牌/型号 = 复购市场
```

**示例**：

| 公式 | 搜索词 | 可能发现 |
|------|--------|---------|
| 场景 + 品类 | emergency + radio | 手摇发电收音机 |
| 功能 + 人群 | calculate + knitting | 编织针数计算器 |
| 属性 + 品类 | portable + washing machine | 旅行迷你洗衣机 |
| 替换 + 型号 | replacement + dyson filter | 戴森替换滤网（通用替代） |
| 人群 + 功能 | senior + amplify | 老年人声音放大器 |

## 4. 筛选标准

发现产品后，用以下标准判断值不值得做：

| 维度 | 值得做 | 不值得做 |
|------|--------|---------|
| 毛利 | > 50% | < 30% |
| 竞争 | 搜索结果 < 1000 | 大品牌垄断 |
| 功能组合空间 | 能叠加 2-3 个功能 | 功能单一 |
| 复购/关联购买 | 有消耗品或配件 | 一次性购买 |
| 用户价格敏感度 | 低（健康、安全、应急） | 高（日用品比价） |
