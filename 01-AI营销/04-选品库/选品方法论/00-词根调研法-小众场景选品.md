# 词根调研法：小众场景 × 功能组合选品

> **核心思路**：用英文词根在亚马逊/电商平台搜索，发现小众场景下被忽视的高利润产品。小众场景竞争少、用户价格敏感度低、功能组合创新空间大。（来源：[小渔聊独立站](https://mp.weixin.qq.com/s/jMkpc8iDj--vyUI9PA5PnQ)）

---

## 1. 方法论

```
选一个词根 → 在 Amazon 搜索 → 逐个翻看产品 → 记录高毛利/低销量/功能组合机会 → 筛选验证
```

**为什么词根有效**：
- 词根 = 场景/功能/问题的英文表达，直接命中用户搜索意图
- 同一词根下往往覆盖多个品类，跨品类发现机会
- 小众词根的搜索结果少，竞争低，容易被忽视

**选词根的原则**：
1. 描述一个**具体问题或场景**（emergency、survival）
2. 描述一个**重复发生的动作**（calculate、filter、dispense）
3. 描述一个**需要周期性更换的消耗**（replacement、refill）
4. 描述一个**特定的用户群体**（orthopedic、ergonomic）
5. 描述一个**可定制的情感/纪念载体**（map、print、engrave）→ 见 2.6 节

**词根的变现力分级**：
词根不仅看搜索量，还要看**用户搜索时的购买意图强度**：

| 变现力 | 词根类型 | 特征 | 举例 |
|--------|---------|------|------|
| **高** | 定制/纪念类 | 搜索 = 想买，天然带交易意图 | custom map, personalized gift |
| **高** | 替换/复购类 | 必须买，不买设备不能用 | replacement filter, refill |
| **中** | 场景/人群类 | 有需求但会比价 | ergonomic chair, survival kit |
| **低** | 纯工具/动作类 | 用户来查信息，不一定要买 | calculator, converter |

**实操提示**：同样 2 万条搜索量，"map"（购买意图强）的变现价值远高于 "calculator"（工具属性强）。选品时优先考虑高变现力词根。

## 2. 词根清单

### 2.1 场景/问题类

> 描述一个"出问题时需要什么"的场景

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Emergency** | 应急 | 充电宝+手电+收音机+工具组合 | 户外应急电源套装 |
| **Survival** | 生存 | 户外生存工具包、净水器 | 72小时生存背包 |
| **Outage** | 停电/中断 | 备用照明、手摇发电设备 | 停电应急灯 |
| **Preparedness** | 防备 | 灾难预备套装 | 地震/飓风应急箱 |
| **Disaster** | 灾难 | 灾后恢复工具 | 防水文件袋+应急毯 |

**规律**：场景越紧急，用户价格敏感度越低。"Emergency"类产品可以大胆做功能组合。

### 2.2 替换/维修类（高频复购）

> 用户必须定期更换的消耗品，剃须刀模式

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Replacement** | 替换件 | 滤网、刷头、刀片、配件 | 空气净化器替换滤网 |
| **Parts** | 配件/零部件 | 家电配件、水暖配件、工具配件 | sink drain parts、vacuum parts |
| **Refill** | 补充装 | 洗手液、墨水、肥皂 | 自动出液器补充装 |
| **Repair** | 维修 | 维修工具包、修复胶 | 屏幕修复套装 |
| **Spare** | 备件 | 备用零件、备用电池 | 备用遥控器 |
| **Upgrade** | 升级 | 原厂配件的升级替代 | 更好的替换刷头 |

**规律**：Replacement 类产品一旦进入用户的购买循环，复购极稳定。关键是找到"原厂贵、替代便宜"的价格差。

**Parts 词根案例：家电配件独立站**（来源：[小渔聊独立站](https://mp.weixin.qq.com/s/)）

- 核心关键词：`xxx parts`、`xxx replacement`，搜索量尚可，**竞争度很低**
- 实测案例站：月流量仅 4.3K，但 **Traffic Cost 达 5.3K**（很多 10K+ 流量的信息站都达不到）
- 商业意图占比：Commercial + Transactional 词 **超过 73%**，用户都是奔着采购来的
- 竞品 SEO 水平差：On-page SEO 没做、文案没专门编辑、产品从阿里系搬运 → **优化空间大**
- 品类扩展：水槽配件（sink drain parts）、下水管配件、交通工具配件、家居设备配件，搜索量大但竞争不多
- **核心逻辑**：流量不高没关系，关键是**每一条流量的商业价值高**。用 CPC 投放拿同样的高意图流量成本不菲，而 SEO 几乎免费

> **实操提示**：如果你对某个产品领域（家电、水暖、汽配、电动工具）比较熟悉，用 "Parts" + 熟悉品类 做关键词调研，很可能发现低竞争高价值的配件市场。

### 2.3 功能/动作类

> 描述产品"做什么"，一个词根下往往有多个细分场景

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Calculate** | 计算 | 细分计算器（厨房、编织、建筑、税） | 编织针数计算器 |
| **Convert** | 转换 | 适配器、转换器（电压、接口、单位） | 欧洲旅行电源转换器 |
| **Correct / Correction** | 矫正 | 体态、视力、牙齿、脚趾 | 儿童矫正鞋 |
| **Detect / Detector** | 检测 | 水质、甲醛、辐射、金属 | 家用水质检测笔 |
| **Sterilize / Sterilizer** | 消毒 | UV消毒盒、蒸汽消毒 | 手机UV消毒盒 |
| **Compress / Compression** | 压缩 | 收纳袋、运动护具、医疗 | 旅行压缩收纳袋 |
| **Filter / Filtration** | 过滤 | 水过滤、空气过滤、油过滤 | 龙头净水器 |
| **Dispense / Dispenser** | 分配 | 肥皂、食物、水、纸巾 | 宠物自动喂食器 |
| **Seal / Sealing** | 密封 | 真空封口、防水密封 | 食物真空封口机 |
| **Insulate / Insulation** | 隔离 | 隔热、隔音、保温 | 窗户隔热膜 |
| **Charge / Charger** | 充电 | 细分充电场景 | 太阳能背包充电器 |
| **Mount / Mounting** | 固定/安装 | 手机架、显示器臂、工具挂架 | 床头手机支架 |
| **Amplify / Amplifier** | 放大/增强 | 声音、信号、视觉 | 老年人电视声音放大器 |
| **Automate / Automation** | 自动化 | 定时器、智能开关、自动装置 | 自动浇花器 |

**规律**：一个动作 × 不同场景 = 多个产品机会。"Calculate"不只是计算器，是"编织计算器"、"面包烘焙计算器"、"混凝土配比计算器"。

### 2.4 人群/健康类

> 针对特定人群设计，小众但刚需

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Ergonomic** | 人体工学 | 办公椅、键盘、鼠标、厨房工具 | 站立办公桌转换器 |
| **Orthopedic** | 骨科/矫形 | 鞋垫、枕头、坐垫、护膝 | 狗狗骨科床 |
| **Corrective** | 矫正 | 姿势矫正器、脚趾分离器 | 驼背矫正带 |
| **Therapeutic** | 治疗 | 热敷、冷敷、按摩 | 颈椎治疗仪 |
| **Pediatric** | 儿科 | 婴儿专用产品 | 婴儿指甲修剪器 |
| **Senior / Geriatric** | 老年 | 辅助设备、安全产品 | 老年人防滑洗澡椅 |
| **Rehab / Rehabilitation** | 康复 | 术后恢复、运动康复 | 膝盖康复训练器 |

**规律**：健康类产品的共同特征——用户不会比价，因为"健康没有价格"。毛利极高。

### 2.5 属性/形态类

> 在任何品类上叠加这些属性，都能创造差异化

| 词根 | 含义 | 叠加逻辑 | 案例 |
|------|------|---------|------|
| **Portable** | 便携 | 任何产品都有便携版本的需求 | 便携洗衣机 |
| **Foldable** | 可折叠 | 节省空间 = 小户型/旅行刚需 | 折叠菜板 |
| **Collapsible** | 可收缩 | 同 foldable，更极端节省 | 收缩硅胶水壶 |
| **Waterproof** | 防水 | 户外、浴室、海洋场景 | 防水蓝牙音箱 |
| **Solar** | 太阳能 | 离网场景 = 便携+应急 | 太阳能充电宝 |
| **Adjustable** | 可调节 | 一款产品覆盖多个用户 | 可调节桌面高度 |
| **Heavy-duty** | 重型 | 专业/工业级 = 高价 | 重型犬链 |
| **Compact** | 紧凑 | 小空间优化 | 紧凑型空气净化器 |

**规律**：属性词根不单独使用，要和品类词根组合搜索。例如 "portable + washing machine"、"foldable + cutting board"。

### 2.6 定制/纪念类（高溢价、高意图）

> 天然带购买意图，用户搜索 = 想定制一个东西，信息简单但溢价极高

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Map** | 地图 | 地理位置、游戏地图、攻略地图、纪念地图 | 权力的游戏攻略地图 |
| **Custom / Customize** | 定制 | 任何品类的定制版本 | 定制宠物项圈 |
| **Personalized** | 个性化 | 刻字、印照片、专属设计 | 个性化名字项链 |
| **Engrave / Engraving** | 刻字 | 首饰、手表、酒瓶、奖牌 | 刻字结婚对戒 |
| **Print / Printed** | 印制 | T恤、海报、手机壳、帆布画 | 宠物定制手机壳 |
| **Memorial** | 纪念 | 纪念品、追悼品、纪念日 | 宠物骨灰纪念项链 |
| **Keepsake** | 纪念物 | 出生、婚礼、毕业纪念 | 婴儿手足印套装 |
| **Monogram** | 字母组合 | 刺绣、压印、雕刻 | 字母组合浴巾 |
| **Commemorative** | 纪念性 | 限量版、周年纪念 | 毕业纪念相框 |
| **Dedicated** | 专属 | 为特定对象设计 | 专属星座水晶套装 |

**规律**：定制类产品虽然"信息简单"（本质上就是一段文字+一张图），但因为满足垂直小众的情感需求，溢价空间极大。产品形态多样（纸质、画框、3D立体、花束等），同一需求可以有多种载体。

**案例：Custom Map 词根验证**（来源：[小渔聊独立站](https://mp.weixin.qq.com/s/jZsqP3pf7nLdiGuvWOAqTw)）
- 用 "map" 词根广泛匹配，排除地理位置词条，设置低竞争 + 搜索意图过滤
- 结果仍有 **2 万+ 条关键词**
- 细分方向：游戏地图（权游）、攻略地图、纪念地图、交互地图
- 产品形态：纸质、画框、3D 立体、花束 —— 同一需求，多种载体 = 多个 SKU

## 3. 组合搜索公式

```
场景词根 + 品类 = 产品机会
功能词根 + 人群 = 细分市场
属性词根 + 品类 = 差异化
替换词根 + 品牌/型号 = 复购市场
定制词根 + 品类/场景 = 高溢价产品    ← 新增
```

**示例**：

| 公式 | 搜索词 | 可能发现 |
|------|--------|---------|
| 场景 + 品类 | emergency + radio | 手摇发电收音机 |
| 功能 + 人群 | calculate + knitting | 编织针数计算器 |
| 属性 + 品类 | portable + washing machine | 旅行迷你洗衣机 |
| 替换 + 型号 | replacement + dyson filter | 戴森替换滤网（通用替代） |
| 人群 + 功能 | senior + amplify | 老年人声音放大器 |
| **定制 + 品类** | **custom map + game of thrones** | **权游攻略地图** |
| **定制 + 场景** | **personalized + wedding** | **定制婚礼纪念品** |

### 3.1 进阶：词根 × 自家产品 交叉法

> 不是用词根找新品，而是**给已有产品发现新的词根入口**

**场景**：你已经在做一个品类的产品（如水晶手链），想找到更多能带来流量的词根，和自己的产品做关联。

**步骤**：
1. 在 Semrush/Ahrefs 输入一个和你产品**相关但不直接竞争**的词根（如 map、print、quiz）
2. 设置广泛匹配 + 低竞争过滤 + 搜索意图筛选
3. 找到搜索量大但竞品没覆盖的关键词
4. 思考：这个关键词背后的用户需求，能不能和我的产品结合？

**示例**：
- 水晶手链 + map 词根 → "crystal energy map"（水晶能量地图，个性化定制）
- 水晶手链 + calculator 词根 → "crystal bracelet size calculator"（已规划）
- 水晶手链 + quiz 词根 → "what crystal should I get quiz"（已规划）

**核心逻辑**：词根不只是选品工具，也是**给现有产品发现新流量入口**的工具。一个产品可以和多个词根交叉，每个交叉都可能产生新的内容/工具/产品方向。

## 4. 筛选标准

发现产品后，用以下标准判断值不值得做：

| 维度 | 值得做 | 不值得做 |
|------|--------|---------|
| 毛利 | > 50% | < 30% |
| 竞争 | 搜索结果 < 1000 | 大品牌垄断 |
| 功能组合空间 | 能叠加 2-3 个功能 | 功能单一 |
| 复购/关联购买 | 有消耗品或配件 | 一次性购买 |
| 用户价格敏感度 | 低（健康、安全、应急） | 高（日用品比价） |
| **Traffic Cost / 月** | **高（流量不大但 Traffic Cost 高）** | **低（流量大但都是信息词）** |
| **商业意图占比** | **Commercial + Transactional > 60%** | **Informational > 70%** |

**Traffic Cost 是什么**：在 Ahrefs/SEMrush 中，表示"如果用 CPC 广告购买同样的流量，需要花多少钱"。它反映的不是流量**多少**，而是流量的**商业价值**。

**为什么重要**：一个月流量 4.3K 但 Traffic Cost 5.3K 的配件站，比月流量 10K 但 Traffic Cost 1K 的信息站更赚钱。因为配件站的用户 73% 都是来买的，信息站的用户 73% 都是来查资料的。

> **一句话**：选品时不要只看搜索量，要看 **Traffic Cost 和商业意图占比**。低流量高意图 > 高流量低意图。

## 7. SEO 工具站词根大全（51 个）

> **来源**：哥飞付费社群分享（[原文链接](https://r2quepastz.feishu.cn/wiki/Ig6ewVJMOiTY1ikc3Ljc3vLUnlc)），51 个英文词根，适合用来搭建 SEO 工具站/功能页面。
>
> **与第 2 节的区别**：第 2 节侧重**选品**（用户搜完会买），本节侧重**建站/工具页面**（用户搜完会用）。两类词根可以交叉使用——用选品词根找产品方向，用工具词根给产品站做引流工具页。

**核心洞察**：工具/功能类页面本质是**决策辅助**，解决用户"具体的、动态的"决策需求。这类需求大媒体通常覆盖不到（需要代码逻辑，不只是文本），正是独立站开发者的机会。

### 7.1 生成/创建类

> 用户想"造一个东西出来"

| 词根 | 含义 | 常见搭配 | 建站方向 |
|------|------|---------|---------|
| **Generator** | 生成器 | Password Generator, Name Generator, QR Code Generator | 随机内容生成工具站 |
| **Maker** | 制作器 | Logo Maker, Video Maker, Music Maker | 在线制作工具 |
| **Creator** | 创作者 | Content Creator, Logo Creator, Game Creator | 内容创作平台 |
| **Builder** | 构建器 | App Builder, Website Builder, Game Builder | 可视化搭建工具 |
| **Constructor** | 构建者 | Website Constructor, Product Constructor | 同 Builder，更技术向 |

### 7.2 转换/处理类

> 用户想把 A 变成 B

| 词根 | 含义 | 常见搭配 | 建站方向 |
|------|------|---------|---------|
| **Translator** | 翻译器 | Google Translator, Language Translator | 在线翻译工具 |
| **Converter** | 转换器 | Currency Converter, PDF Converter, Image Converter | 格式/单位转换 |
| **Convert** | 转换 | PDF to Word Convert, Video Convert | 同 Converter，动词形式 |
| **Processor** | 处理器 | Data Processor, Image Processor, Word Processor | 数据处理工具 |
| **Formatter** | 格式化 | Text Format, Image Format | 数据格式化工具 |
| **Compiler** | 编译器 | C++ Compiler, Java Compiler | 代码编译在线工具 |
| **Interpreter** | 解释器 | Python Interpreter, JavaScript Interpreter | 在线代码运行 |

### 7.3 检查/验证类

> 用户想确认"对不对"、"好不好"

| 词根 | 含义 | 常见搭配 | 建站方向 |
|------|------|---------|---------|
| **Checker** | 检查器 | Grammar Checker, Plagiarism Checker, SEO Checker | 在线验证工具 |
| **Detector** | 检测器 | Virus Detector, AI Content Detector | 问题检测工具 |
| **Analyzer** | 分析器 | Data Analyzer, Text Analyzer, Performance Analyzer | 数据/文本分析 |
| **Evaluator** | 评估器 | Product Evaluator, Website Evaluator | 评估/打分工具 |
| **Verifier** | 验证器 | Identity Verifier, Code Verifier, Data Verifier | 身份/数据验证 |
| **Comparator** | 比较器 | Price Comparator, Product Comparator | 价格/产品对比 |
