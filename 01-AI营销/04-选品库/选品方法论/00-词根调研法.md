# 词根调研法

> **核心思路**：英文词根 = 场景/功能/问题的精准表达。选品词根发现"卖什么"，工具站词根发现"做什么工具引流"，修饰词矩阵发现"做什么类型的页面"。三类方法交叉使用。
>
> **来源**：[小渔聊独立站](https://mp.weixin.qq.com/s/jMkpc8iDj--vyUI9PA5PnQ)、[桃与SaaS](https://mp.weixin.qq.com/s/j4Pm5YPObHA9zEg80Thu4w)、[赫兹/droidHZ](https://mp.weixin.qq.com/s/PDF3Bbfx8ryShOmcMfgdbg)、[Ahrefs / Ryan Law](https://ahrefs.com/blog/the-free-tools-seo-strategy/)、[SEOmatic](https://seomatic.ai/blog/programmatic-seo-keyword-research)、[SE Ranking](https://seranking.com/blog/keyword-types/)
>
> **最后更新**：2026-07-22

## 1. 两类词根 + 变现力

| | 选品词根 | 工具站词根 |
|---|---|---|
| **用户行为** | 搜完会**买** | 搜完会**用** |
| **词根类型** | 场景/替换/功能/人群/属性/定制 | calculator/converter/generator/checker 等 |
| **目标** | 发现卖什么产品 | 做什么工具页面引流 |
| **交叉使用** | 用选品词根找产品方向，用工具词根给产品站做引流工具页 | |

**变现力分级**（词根不仅看搜索量，还要看购买意图强度）：

| 变现力 | 词根类型 | 举例 |
|---|---|---|
| **高** | 定制/纪念类 | custom map, personalized gift |
| **高** | 替换/复购类 | replacement filter, refill |
| **中** | 场景/人群类 | ergonomic chair, survival kit |
| **低** | 纯工具/动作类 | calculator, converter（但可做引流工具页） |

## 4. 工具站 SEO 策略（Ahrefs 方法论）

> 来源：[Ryan Law / Ahrefs, 2026-07-16](https://ahrefs.com/blog/the-free-tools-seo-strategy/)

### 4.1 AI 抗性

```
文章：AI 可以总结 → 用户不点击
工具：AI 可以描述"如何计算"，但不能替用户做计算 → 用户必须访问页面
```

### 4.2 硬数据案例

| 网站 | 工具类型 | 月访问量（美国） |
|---|---|---|
| Omni Calculator | 计算器库 | **230 万** |
| Ahrefs /writing-tools/ | 写作工具集 | 峰值 **~100 万** |
| FreeConvert | 文件转换器 | **150 万** |
| Coolors | 色彩生成器 | **59 万** |
| Adobe PDF-to-Word | 文档转换器 | **38.5 万** |
| Canva /templates/ | 模板库 | **50.5 万** |
| HubSpot 签名生成器 | 生成器 | **5.5 万** |
| Shopify 利润计算器 | 计算器 | 峰值 **2 万** |

非工具站品牌（Shopify、HubSpot、Adobe）也在用免费工具引流——工具是面向目标客户的"免费前门"。

### 4.3 关键词筛选法

**头部关键词打不过**：mortgage calculator (KD 81)、qr code generator (KD 90)

**可赢关键词**（低 KD，高潜力）：

| 关键词 | 月搜索量 | KD | 流量潜力 |
|---|---|---|---|
| etsy fee calculator | 2,000 | 7 | 4,000 |
| tattoo price calculator | 350 | 1 | 1,700 |
| yaml to json | 2,900 | 1 | 2,200 |
| band name generator | 1.1 万 | 7 | 1.4 万 |

**筛选公式**：`KD ≤ 30 + Lowest DR filter + 按 Traffic Potential 排序`

### 4.4 SERP 检查

| 排名页面类型 | 判断 | 行动 |
|---|---|---|
| **薄工具页** | 你可以做得更好 | ✅ 做 |
| **3000 字深度文章** | 用户想读不想做 | ❌ 不是工具机会 |

### 4.5 AI 构建 Prompt 模板

好的 Prompt = 输入字段 + 输出结果 + 页面文案（标题/说明/FAQ）。

**Ahrefs 实测**：$1.24 + 1 分钟生成可用计算器。

```
Build me a single-page "[thing] calculator" as one self-contained HTML
file. Inputs: [list fields with defaults]. Outputs: [list results]. Show
the working underneath each number. Make it fast and mobile-friendly, with
a clear title, a one-line explainer, and no login. Underneath the calculator,
add a short FAQ answering "[common question 1]" and "[common question 2]".
```

### 4.6 部署方案

| 平台 | 方式 | SEO 要点 |
|---|---|---|
| WordPress | Custom HTML block（`/tools/xxx/`） | 不用 iframe |
| React/Next.js | 直接集成 | 最佳 SEO |
| 快速上线 | 静态托管 + 主站链接 | 主站加链接传权重 |

## 5. 程序化 SEO 修饰词矩阵（Keyword Modifier Matrix）

> **和词根体系的关系**：词根解决"做什么产品/工具"，修饰词矩阵解决"做什么类型的页面"。
>
> **来源**：[SEOmatic](https://seomatic.ai/blog/programmatic-seo-keyword-research)、[SE Ranking](https://seranking.com/blog/keyword-types/)、[Search Engine Journal](https://www.searchenginejournal.com/how-to-organize-your-keyword-modifiers-to-create-long-tail-strategy/6526/)、[PageForge](https://pageforge.pro/tools/programmatic-seo-keyword-generator/)

### 5.1 核心公式

```
Head Term × Modifier = 页面模板
  Head Term = 你的产品/品类/主题（来自词根研究）
  Modifier = 用户搜索意图的前缀/后缀
```

### 5.2 按搜索意图分类的 Modifier

| 意图 | Modifier | 页面类型 |
|---|---|---|
| **Informational** | how to, what is, guide to, tips, ideas, examples, definition, meaning, benefits | 教程/指南页 |
| **Commercial** | best, top, vs, review, comparison, alternatives, cheap, affordable, premium | 评测/对比页 |
| **Transactional** | buy, order, price, cost, deal, discount, coupon, near me, for sale, wholesale | 购买/落地页 |
| **Navigational** | brand names, login, official site, alternative to, substitute for | 品牌导航页 |

### 5.3 完整 Modifier 清单

**前缀**：
```
best / top / cheap / affordable / free / premium / professional
alternative to / substitute for / similar to
how to / what is / why / when to
diy / homemade / natural / organic
used / refurbished
```

**后缀**：
```
for beginners / for kids / for seniors / for pregnancy / for travel
for sale / online / near me / in [city]
cost / price / under $50 / under $100
review / reviews / reddit / vs
wholesale / bulk / custom / personalized
template / checklist / guide / pdf
meaning / benefits / properties / uses
```

**动词型**：
```
how to [clean/use/charge/identify] [品类]
is [品类] [safe/real/worth it]
can [品类] [go in water/be in sun/go through airport]
does [品类] [work/help/fade]
```

### 5.4 矩阵交叉示例

```
Head Term: amethyst

amethyst meaning           → 含义页（Informational）
amethyst vs rose quartz    → 对比页（Commercial）
amethyst for anxiety       → 用例页（Informational→Transactional）
amethyst near me           → 本地页（Transactional）
amethyst cost              → 定价页（Transactional）
how to clean amethyst      → 教程页（Informational）
is amethyst safe in water  → FAQ 页（Informational）
```

**一个 Head Term × 20 个 Modifier = 20 个页面机会**。

### 5.5 关键词聚类避免同质化

> 来源：[Search Engine Zine - Programmatic Keyword Clustering Framework](https://searchenginezine.com/keywords/long-tail/programmatic-keyword-clustering/)

**问题**：Head Term × Modifier 矩阵可能产生大量相似页面 → Google 判定 thin content → 降权。

**解决**：
1. 把所有 Modifier 关键词按语义聚类
2. 同一个聚类内的关键词合并到一个页面（而非每词一页）
3. 不同聚类之间确保内容差异足够大

```
示例聚类：
  聚类 A："amethyst for sleep / amethyst for insomnia / amethyst under pillow"
  → 合并为一个页面："Amethyst for Sleep: Benefits, How to Use & Science"

  聚类 B："amethyst for anxiety / amethyst for stress / amethyst for calming"
  → 合并为一个页面："Amethyst for Anxiety: Does It Work?"

  A 和 B 内容差异足够大 → 不触发同质化
```

### 5.6 pSEO 风险控制

> 与 [18-AI批量内容风险与爬取经济学](../../03-SEO与GEO/18-AI批量内容风险与爬取经济学.md) 呼应。

- pSEO 不是无限生成页面——**页面数量必须与域名权重匹配**
- 验证每个 Modifier 的搜索量后再生成页面
- Google 已明确打击 programmatic spam
- 新站分批发布，每批配套外链 + 社媒信号

## 6. 组合搜索公式

```
场景词根 + 品类 = 产品机会
功能词根 + 人群 = 细分市场
属性词根 + 品类 = 差异化
替换词根 + 品牌/型号 = 复购市场
定制词根 + 品类/场景 = 高溢价产品
工具词根 + 自家产品 = 引流工具页
```

### 词根 × 自家产品 交叉法

不是用词根找新品，而是给已有产品发现新的流量入口。在 Ahrefs/Semrush 输入相关词根，找搜索量大但竞品没覆盖的关键词。

## 7. 筛选标准

### 7.1 选品筛选

| 维度 | 值得做 | 不值得做 |
|------|--------|---------|
| 毛利 | > 50% | < 30% |
| 竞争 | 搜索结果 < 1000 | 大品牌垄断 |
| 复购 | 有消耗品或配件 | 一次性购买 |
| **Traffic Cost** | **高（低流量高意图）** | **低（高流量低意图）** |
| **商业意图占比** | **Commercial + Transactional > 60%** | **Informational > 70%** |

### 7.2 工具站筛选

| 维度 | 值得做 | 不值得做 |
|------|--------|---------|
| KD | ≤ 30 | > 50 |
| SERP 页面类型 | 薄工具页 | 深度文章 |
| CPC | 高（健康/金融/开发） | 低（娱乐） |
| AI 抗性 | 高（用户必须使用工具） | 低（AI 可总结） |
| 开发成本 | AI 生成 < $5 | 需要开发团队 |
