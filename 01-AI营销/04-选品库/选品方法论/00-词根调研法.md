# 词根调研法：选品词根 × 工具站词根 × AI 构建方法

> **核心思路**：英文词根 = 场景/功能/问题的精准表达。选品词根发现"卖什么"，工具站词根发现"做什么工具引流"。两类词根交叉使用 = 选品 + 引流一体化。
>
> **来源**：[小渔聊独立站](https://mp.weixin.qq.com/s/jMkpc8iDj--vyUI9PA5PnQ)（选品词根）、[桃与SaaS](https://mp.weixin.qq.com/s/j4Pm5YPObHA9zEg80Thu4w) + [赫兹/droidHZ](https://mp.weixin.qq.com/s/PDF3Bbfx8ryShOmcMfgdbg)（工具站词根）、[Ahrefs / Ryan Law](https://ahrefs.com/blog/the-free-tools-seo-strategy/)（工具 SEO 策略 + AI 构建）
>
> **最后更新**：2026-07-22（合并原 00 + 03 文档 + Ahrefs 工具 SEO 策略增量）

## 1. 方法论

### 1.1 两类词根，两个目标

| | 选品词根 | 工具站词根 |
|---|---|---|
| **用户行为** | 搜完会**买** | 搜完会**用** |
| **词根类型** | 场景/替换/功能/人群/属性/定制 | calculator/converter/generator/checker 等 |
| **目标** | 发现卖什么产品 | 做什么工具页面引流 |
| **变现方式** | 产品销售 | 广告 / 引流到产品页 |
| **交叉使用** | 用选品词根找产品方向，用工具词根给产品站做引流工具页 | |

### 1.2 免费站 vs 付费站的需求逻辑

| | 付费站 (SaaS) | 免费站 (广告变现) |
|---|---|---|
| **逻辑** | 洞察隐性痛点 → 创造新产品 | 发现搜索数据 → 满足显性需求 |
| **起点** | 自身/周边人/已有产品 | 数据分析平台（搜索词） |
| **验证** | 需要主动验证用户是否需要 | 搜索量 = 需求已被验证 |

**关键认知**：做免费工具站不要从自身痛点出发（搜索量极低），要从搜索数据出发。

### 1.3 变现力分级

词根不仅看搜索量，还要看**用户搜索时的购买意图强度**：

| 变现力 | 词根类型 | 特征 | 举例 |
|---|---|---|---|
| **高** | 定制/纪念类 | 搜索 = 想买，天然带交易意图 | custom map, personalized gift |
| **高** | 替换/复购类 | 必须买，不买设备不能用 | replacement filter, refill |
| **中** | 场景/人群类 | 有需求但会比价 | ergonomic chair, survival kit |
| **低** | 纯工具/动作类 | 用户来查信息，不一定要买 | calculator, converter |

**实操提示**：同样 2 万条搜索量，"map"（购买意图强）的变现价值远高于 "calculator"（工具属性强）。但工具词根可以用来做引流工具页，再导流到产品页。

## 2. 选品词根清单（6 类）

### 2.1 场景/问题类

> 描述一个"出问题时需要什么"的场景

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Emergency** | 应急 | 充电宝+手电+收音机+工具组合 | 户外应急电源套装 |
| **Survival** | 生存 | 户外生存工具包、净水器 | 72小时生存背包 |
| **Outage** | 停电/中断 | 备用照明、手摇发电设备 | 停电应急灯 |
| **Preparedness** | 防备 | 灾难预备套装 | 地震/飓风应急箱 |
| **Disaster** | 灾难 | 灾后恢复工具 | 防水文件袋+应急毯 |

**规律**：场景越紧急，用户价格敏感度越低。"Emergency"类产品可以大胆做功能组合。

### 2.2 替换/维修类（高频复购）

> 用户必须定期更换的消耗品，剃须刀模式

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Replacement** | 替换件 | 滤网、刷头、刀片、配件 | 空气净化器替换滤网 |
| **Parts** | 配件/零部件 | 家电配件、水暖配件、工具配件 | sink drain parts、vacuum parts |
| **Refill** | 补充装 | 洗手液、墨水、肥皂 | 自动出液器补充装 |
| **Repair** | 维修 | 维修工具包、修复胶 | 屏幕修复套装 |
| **Spare** | 备件 | 备用零件、备用电池 | 备用遥控器 |
| **Upgrade** | 升级 | 原厂配件的升级替代 | 更好的替换刷头 |

**规律**：Replacement 类产品一旦进入用户的购买循环，复购极稳定。关键是找到"原厂贵、替代便宜"的价格差。

**Parts 词根案例**（来源：小渔聊独立站）：
- 月流量仅 4.3K，但 **Traffic Cost 达 5.3K**（很多 10K+ 流量的信息站都达不到）
- 商业意图占比：Commercial + Transactional 词 **超过 73%**
- **核心逻辑**：流量不高没关系，关键是**每条流量的商业价值高**

### 2.3 功能/动作类

> 描述产品"做什么"，一个词根下往往有多个细分场景

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Calculate** | 计算 | 细分计算器（厨房、编织、建筑、税） | 编织针数计算器 |
| **Convert** | 转换 | 适配器、转换器（电压、接口、单位） | 欧洲旅行电源转换器 |
| **Correct / Correction** | 矫正 | 体态、视力、牙齿、脚趾 | 儿童矫正鞋 |
| **Detect / Detector** | 检测 | 水质、甲醛、辐射、金属 | 家用水质检测笔 |
| **Sterilize / Sterilizer** | 消毒 | UV消毒盒、蒸汽消毒 | 手机UV消毒盒 |
| **Compress / Compression** | 压缩 | 收纳袋、运动护具、医疗 | 旅行压缩收纳袋 |
| **Filter / Filtration** | 过滤 | 水过滤、空气过滤、油过滤 | 龙头净水器 |
| **Dispense / Dispenser** | 分配 | 肥皂、食物、水、纸巾 | 宠物自动喂食器 |
| **Seal / Sealing** | 密封 | 真空封口、防水密封 | 食物真空封口机 |
| **Insulate / Insulation** | 隔离 | 隔热、隔音、保温 | 窗户隔热膜 |
| **Charge / Charger** | 充电 | 细分充电场景 | 太阳能背包充电器 |
| **Mount / Mounting** | 固定/安装 | 手机架、显示器臂、工具挂架 | 床头手机支架 |
| **Amplify / Amplifier** | 放大/增强 | 声音、信号、视觉 | 老年人电视声音放大器 |
| **Automate / Automation** | 自动化 | 定时器、智能开关、自动装置 | 自动浇花器 |

**规律**：一个动作 × 不同场景 = 多个产品机会。

### 2.4 人群/健康类

> 针对特定人群设计，小众但刚需

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Ergonomic** | 人体工学 | 办公椅、键盘、鼠标、厨房工具 | 站立办公桌转换器 |
| **Orthopedic** | 骨科/矫形 | 鞋垫、枕头、坐垫、护膝 | 狗狗骨科床 |
| **Corrective** | 矫正 | 姿势矫正器、脚趾分离器 | 驼背矫正带 |
| **Therapeutic** | 治疗 | 热敷、冷敷、按摩 | 颈椎治疗仪 |
| **Pediatric** | 儿科 | 婴儿专用产品 | 婴儿指甲修剪器 |
| **Senior / Geriatric** | 老年 | 辅助设备、安全产品 | 老年人防滑洗澡椅 |
| **Rehab / Rehabilitation** | 康复 | 术后恢复、运动康复 | 膝盖康复训练器 |

**规律**：健康类产品的共同特征——用户不会比价，因为"健康没有价格"。毛利极高。

### 2.5 属性/形态类

> 在任何品类上叠加这些属性，都能创造差异化

| 词根 | 含义 | 叠加逻辑 | 案例 |
|------|------|---------|------|
| **Portable** | 便携 | 任何产品都有便携版本的需求 | 便携洗衣机 |
| **Foldable** | 可折叠 | 节省空间 = 小户型/旅行刚需 | 折叠菜板 |
| **Collapsible** | 可收缩 | 同 foldable，更极端节省 | 收缩硅胶水壶 |
| **Waterproof** | 防水 | 户外、浴室、海洋场景 | 防水蓝牙音箱 |
| **Solar** | 太阳能 | 离网场景 = 便携+应急 | 太阳能充电宝 |
| **Adjustable** | 可调节 | 一款产品覆盖多个用户 | 可调节桌面高度 |
| **Heavy-duty** | 重型 | 专业/工业级 = 高价 | 重型犬链 |
| **Compact** | 紧凑 | 小空间优化 | 紧凑型空气净化器 |

**规律**：属性词根不单独使用，要和品类词根组合搜索。

### 2.6 定制/纪念类（高溢价、高意图）

> 天然带购买意图，用户搜索 = 想定制一个东西

| 词根 | 含义 | 产品方向 | 案例 |
|------|------|---------|------|
| **Map** | 地图 | 地理位置、游戏地图、攻略地图、纪念地图 | 权力的游戏攻略地图 |
| **Custom / Customize** | 定制 | 任何品类的定制版本 | 定制宠物项圈 |
| **Personalized** | 个性化 | 刻字、印照片、专属设计 | 个性化名字项链 |
| **Engrave / Engraving** | 刻字 | 首饰、手表、酒瓶、奖牌 | 刻字结婚对戒 |
| **Print / Printed** | 印制 | T恤、海报、手机壳、帆布画 | 宠物定制手机壳 |
| **Memorial** | 纪念 | 纪念品、追悼品、纪念日 | 宠物骨灰纪念项链 |
| **Keepsake** | 纪念物 | 出生、婚礼、毕业纪念 | 婴儿手足印套装 |
| **Monogram** | 字母组合 | 刺绣、压印、雕刻 | 字母组合浴巾 |
| **Commemorative** | 纪念性 | 限量版、周年纪念 | 毕业纪念相框 |
| **Dedicated** | 专属 | 为特定对象设计 | 专属星座水晶套装 |

**规律**：定制类产品虽然"信息简单"，但因为满足垂直小众的情感需求，溢价空间极大。

**Custom Map 验证案例**（来源：小渔聊独立站）：
- 用 "map" 词根广泛匹配，排除地理位置词条
- 结果仍有 **2 万+ 条关键词**
- 产品形态：纸质、画框、3D 立体、花束 —— 同一需求，多种载体 = 多个 SKU

## 3. 工具站词根清单

> 工具/功能类页面本质是**决策辅助**，解决用户"具体的、动态的"决策需求。这类需求大媒体通常覆盖不到（需要代码逻辑，不只是文本），正是独立站开发者的机会。

### 3.12 CPC 筛选策略

**三个最高频词根**：`calculator`、`converter`、`generator`——每天被搜索几百万次，但竞争也最激烈，需要找长尾词。

**不要碰的词根**（CPC 太低，娱乐性质）：`maker`（娱乐偏多）、`randomizer`（无商业价值）、`picker`（娱乐）、`inverter`（娱乐）

**正确做法**：不做泛词（如 "calculator"），做具体场景的长尾词：
- `sleep calculator`（月访问量 1.3M，靠广告年收入 50 万+）
- `etsy fee calculator`（KD 7，流量潜力 4,000）
- `twitch bits to usd`（KD 0，流量潜力 2,900）

### 3.13 场景价值评估框架

做免费工具站要考虑：**广告主愿意为什么站付钱？**

| 场景 | 人群 | CPC | 评估 |
|------|------|-----|------|
| 健康/金融计算器 | 高价值决策者 | 高 | 优先做 |
| 开发者工具（formatter/validator） | 技术人群 | 中 | 可做 |
| 图片压缩/编辑 | 普通用户 | 低-中 | 批量做 |
| 随机取名/抽奖 | 娱乐用户 | 低 | 不做 |

## 4. 工具站 SEO 策略（Ahrefs 方法论）

> 来源：[Ryan Law / Ahrefs, 2026-07-16](https://ahrefs.com/blog/the-free-tools-seo-strategy/)

### 4.1 AI 抗性：为什么工具比文章更值得做

```
文章：AI Overview / ChatGPT 可以总结 → 用户不点击
工具：AI 可以描述"如何计算利润率"，但不能替用户做计算
     → 用户必须访问页面才能使用工具
```

工具是终极的 AI 抗性内容——AI 无法替代用户使用工具这个动作。

### 4.2 硬数据案例

| 网站 | 工具类型 | 月访问量（美国） | 核心关键词 |
|---|---|---|---|
| Omni Calculator | 计算器库（数千个） | **230 万** | test grade calculator (5,900/月) |
| Ahrefs /writing-tools/ | 写作工具集 | 峰值 **~100 万** | — |
| FreeConvert | 文件转换器 | **150 万** | mp3 converter (26.3 万/月) |
| Coolors | 色彩生成器 | **59 万** | color palette generator (8.4 万/月) |
| Adobe PDF-to-Word | 文档转换器 | **38.5 万** | pdf to word (18.3 万/月) |
| Template.net | 模板库 | **70 万** | doctors note template (8,600/月) |
| Canva /templates/ | 模板库 | **50.5 万** | canva templates (2 万/月) |
| HubSpot 签名生成器 | 生成器 | **5.5 万** | email signature (4.3 万/月) |
| Shopify 利润计算器 | 计算器 | 峰值 **2 万** | margin calculator (5.5 万/月) |

**核心规律**：非工具站品牌（Shopify、HubSpot、Adobe、Gusto）也在用免费工具引流。工具是面向目标客户的"免费前门"。

### 4.3 关键词研究：KD 筛选法

**头部关键词（打不过）**：

| 关键词 | 月搜索量 | KD |
|---|---|---|
| mortgage calculator | 222 万 | 81 |
| qr code generator | 77.6 万 | 90 |
| calorie calculator | 86.8 万 | 86 |

**可赢关键词（低 KD，高潜力）**：

| 关键词 | 月搜索量 | KD | 流量潜力 |
|---|---|---|---|
| etsy fee calculator | 2,000 | 7 | 4,000 |
| tattoo price calculator | 350 | 1 | 1,700 |
| yaml to json | 2,900 | 1 | 2,200 |
| stl to gcode | 1,100 | 1 | 1,500 |
| band name generator | 1.1 万 | 7 | 1.4 万 |
| wedding seating chart template | 1,900 | 7 | 1.1 万 |

**筛选公式**：
```
Keyword Difficulty (KD) ≤ 30
+ Lowest DR filter（确保有低权重站排上去 = 你也有机会）
+ 按 Traffic Potential 或 Volume 排序
```

### 4.4 SERP 检查方法

点击进入 SERP Overview，看排名第一页是什么类型：

| 排名页面类型 | 判断 | 行动 |
|---|---|---|
| **薄工具页** | 机会——你可以做得更好 | ✅ 做 |
| **3000 字深度文章** | 用户想读不想做 | ❌ 不是工具机会 |
| **大品牌垄断** | 打不过 | ❌ 跳过 |

### 4.5 AI 构建 Prompt 模板

好的 Prompt 包含：输入字段 + 输出结果 + 页面文案（帮助排名的标题、说明、FAQ）。

**Ahrefs 实测 Prompt 示例**（$1.24 + 1 分钟生成可用工具）：

```
Build me a single-page "deck material calculator" as one self-contained
HTML file. Inputs: deck length and width (in feet), the deck board width
(in inches, default 5.5), the gap between boards (in inches, default
0.25), joist spacing (12, 16, or 24 inches), and an editable price per
board. Outputs: the number of deck boards needed, total linear feet of
decking, the number of joists, and an estimated materials cost. Show the
working underneath each number so people can see how it's calculated,
not just the final total. Make it fast and mobile-friendly, with a clear
title, a one-line explainer at the top, and no login or sign-up.
Underneath the calculator, add a short FAQ answering "how much decking
do I need?" and "how many deck boards per square foot?" so the page can
rank for those searches.
```

**不同工具类型的 Prompt 要点**：

| 工具类型 | Prompt 必须包含 |
|---|---|
| Calculator | 输入字段 + 计算逻辑 + 结果展示 + FAQ |
| Converter | 粘贴/上传框 + 转换逻辑 + 下载结果 |
| Generator | "生成"按钮 + 结果复制/导出方式 |
| Template | 可填写的表单 + "下载为 PDF"按钮 |

### 4.6 部署方案

| 平台 | 部署方式 | SEO 要点 |
|---|---|---|
| WordPress | Custom HTML block 或独立 landing page（`/tools/xxx/`） | 工具可见在 HTML 中，不用 iframe |
| Webflow/Framer | Embed HTML 组件 | 确保可爬，不被 iframe 隐藏 |
| React/Next.js | 直接集成到页面 | 最佳 SEO，但需开发时间 |
| 快速上线 | 静态托管 + 主站链接 | 从主站加链接传递权重 |

**SEO 基础**：描述性 URL + 标题 + 工具在 HTML 中可见（不用 iframe）+ 移动端快速加载 + FAQ 文字块。

## 5. 组合搜索公式

```
场景词根 + 品类 = 产品机会
功能词根 + 人群 = 细分市场
属性词根 + 品类 = 差异化
替换词根 + 品牌/型号 = 复购市场
定制词根 + 品类/场景 = 高溢价产品
工具词根 + 自家产品 = 引流工具页
```

**示例**：

| 公式 | 搜索词 | 可能发现 |
|------|--------|---------|
| 场景 + 品类 | emergency + radio | 手摇发电收音机 |
| 功能 + 人群 | calculate + knitting | 编织针数计算器 |
| 属性 + 品类 | portable + washing machine | 旅行迷你洗衣机 |
| 替换 + 型号 | replacement + dyson filter | 戴森替换滤网 |
| 定制 + 品类 | custom map + game of thrones | 权游攻略地图 |
| **工具 + 自家产品** | **crystal + quiz** | **水晶推荐测试** |

### 5.1 进阶：词根 × 自家产品 交叉法

不是用词根找新品，而是**给已有产品发现新的词根入口**。

**步骤**：
1. 在 Semrush/Ahrefs 输入一个和你产品**相关但不直接竞争**的词根
2. 设置广泛匹配 + 低竞争过滤 + 搜索意图筛选
3. 找到搜索量大但竞品没覆盖的关键词
4. 思考：这个关键词背后的用户需求，能不能和我的产品结合？

**示例**（水晶项目）：
- 水晶 + map → "crystal energy map"（水晶能量地图）
- 水晶 + calculator → "crystal bracelet size calculator"
- 水晶 + quiz → "what crystal should I get quiz"

## 6. 筛选标准

### 6.1 选品筛选标准

| 维度 | 值得做 | 不值得做 |
|------|--------|---------|
| 毛利 | > 50% | < 30% |
| 竞争 | 搜索结果 < 1000 | 大品牌垄断 |
| 功能组合空间 | 能叠加 2-3 个功能 | 功能单一 |
| 复购/关联购买 | 有消耗品或配件 | 一次性购买 |
| 用户价格敏感度 | 低（健康、安全、应急） | 高（日用品比价） |
| **Traffic Cost / 月** | **高（流量不大但 Traffic Cost 高）** | **低（流量大但都是信息词）** |
| **商业意图占比** | **Commercial + Transactional > 60%** | **Informational > 70%** |

**Traffic Cost 是什么**：在 Ahrefs/SEMrush 中，表示"如果用 CPC 广告购买同样的流量，需要花多少钱"。它反映的不是流量**多少**，而是流量的**商业价值**。

### 6.2 工具站筛选标准

| 维度 | 值得做 | 不值得做 |
|------|--------|---------|
| KD | ≤ 30 | > 50 |
| SERP 页面类型 | 薄工具页 | 深度文章 |
| CPC | 高（健康/金融/开发） | 低（娱乐/随机） |
| AI 抗性 | 高（用户必须使用工具） | 低（AI 可以总结） |
| 开发成本 | AI 生成 < $5 | 需要开发团队 |
| 目标用户匹配 | 和你的产品受众重合 | 无关流量 |

## 8. 程序化 SEO 修饰词矩阵（Keyword Modifier Matrix）

> **来源**：[PageForge](https://pageforge.pro/tools/programmatic-seo-keyword-generator/)、[SEOmatic](https://seomatic.ai/blog/programmatic-seo-keyword-research)、[Terry C. Power / LinkedIn](https://www.linkedin.com/posts/nc-localseo_sultanofseo-activity-7396453945424113666-Ej9d)
>
> **和词根体系的关系**：词根解决"做什么产品/工具"，修饰词矩阵解决"做什么类型的页面"。两者交叉使用：Head Term（来自词根研究） × Modifier = 批量页面。

### 8.1 核心概念

```
Head Term × Modifier = 页面模板
  Head Term = 你的产品/品类/主题
  Modifier = 用户搜索意图的前缀/后缀

示例：
  amethyst × best = "best amethyst for sleep" → 榜单页
  amethyst × vs = "amethyst vs clear quartz" → 对比页
  amethyst × for = "amethyst for anxiety" → 用例页
  crystal shop × near me = "crystal shop near me" → 本地页
```

### 8.2 五类 Modifier

| Modifier 类型 | 常见词 | 意图 | 页面模板 |
|---|---|---|---|
| **比较型** | vs、alternative to、compared | 商业调查 | 对比页 |
| **榜单型** | best、top、review | 商业调查 | 榜单/评测页 |
| **用例型** | for [人群/场景/问题] | 信息型→交易型 | 用例落地页 |
| **本地型** | near me、in [城市] | 交易型 | 地理位置落地页 |
| **价格型** | cost、price、cheap、affordable、free | 交易型 | 定价/对比页 |

### 8.3 完整 Modifier 清单

**前缀 Modifier**：

```
best / top / cheap / affordable / free / premium / professional
alternative to / substitute for / similar to
how to / what is / why / when to
diy / homemade / natural / organic
used / refurbished / second hand
```

**后缀 Modifier**：

```
for beginners / for kids / for seniors / for pregnancy / for travel
for sale / online / near me / in [city]
cost / price / under $50 / under $100
review / reviews / reddit / vs / comparison
wholesale / bulk / custom / personalized
template / checklist / guide / pdf
```

**动词 Modifier**：

```
how to [clean/use/charge/clean/identify] [品类]
is [品类] [safe/real/worth it]
can [品类] [go in water/be in sun/go through airport]
does [品类] [work/help/fade]
```

### 8.4 矩阵交叉法

取一个 Head Term，和所有 Modifier 交叉，批量发现页面机会：

```
Head Term: amethyst

best amethyst          → 榜单页
amethyst vs rose quartz → 对比页
amethyst for anxiety   → 用例页
amethyst for sleep     → 用例页
amethyst near me       → 本地页
amethyst cost          → 定价页
how to clean amethyst  → 教程页
is amethyst safe in water → FAQ 页
amethyst for beginners → 新手指南
cheap amethyst         → 低价选项页
```

**一个 Head Term × 20 个 Modifier = 20 个页面机会**。
