## 2. 网站内容结构

### 2.1 道德采购差异化（核心亮点）

**Moonrise Crystals 的核心差异化 = 道德采购(Ethical Sourcing)**

```
采购准则（8-9条）：
├── Traceable 来源可追溯
├── Ethical mining practices 道德采矿
├── Fair labor 公平劳动
├── Environmental responsibility 环保责任
├── No conflict minerals 无冲突矿物
├── Direct relationships with miners 与矿工直接关系
├── Quality over quantity 质量优先
├── Transparency 透明度
└── Sustainability 可持续性
```

**对 LuckyCrystals 的启发**：
- 道德采购是水晶行业的**强差异化信号**
- 消费者越来越关注产品的来源和伦理
- "Traceable" 和 "Ethical" 关键词本身有SEO价值
- 但需要真正落实，不能仅做营销噱头

## 3. 内容策略分析

### 3.1 245+ 水晶完整指南

**全网最完整的水晶教育内容库之一**：

```
规模: 245+ 篇 Complete Guides to Healing Crystals
URL: moonrisecrystals.com/crystal-guides/

每篇指南内容：
├── 水晶概述与简介
├── 矿物学信息（硬度/化学式/产地）
├── 疗愈属性（身体/情感/灵性）
├── 冥想与使用方法
├── 搭配建议
├── 采购建议
└── 相关产品推荐（连接到电商）
```

**关键发现**：
- 245篇指南 ≈ Tiny Rituals 200-250篇博客的规模
- 但 Moonrise 的内容更**聚焦和系统化**（每篇都是完整的水晶百科）
- 每篇指南直接连接到该水晶的**购买页面**
- 这是"内容×电商"的另一种实现方式

### 6.2 Moonrise Crystals 的弱点

| 弱点 | 说明 |
|------|------|
| **流量规模小** | ~35K/月，相比Tiny Rituals的800K差距巨大 |
| **DR较低** | 35，SEO竞争力有限 |
| **变现模式单一** | 仅实物销售，无数字产品/课程/订阅 |
| **内容类型单一** | 只有水晶指南，缺少天使号码/星座等高流量内容 |
| **品牌知名度有限** | 虽有NYT引用，但整体知名度不高 |

### 6.3 道德采购的策略价值

```
Moonrise Crystals 证明了一个关键点：
- 道德采购本身不足以驱动大规模流量
- 但它是强有力的品牌差异化信号
- 需要与其他策略（内容SEO/互动工具/订阅制）结合

LuckyCrystals 的策略：
1. 道德采购作为"基础层"（必须有）
2. 内容SEO作为"流量层"（学习Tiny Rituals）
3. 互动工具作为"留存层"（学习The Crystal Council）
4. 数字产品+订阅作为"增值层"（差异化）
```
