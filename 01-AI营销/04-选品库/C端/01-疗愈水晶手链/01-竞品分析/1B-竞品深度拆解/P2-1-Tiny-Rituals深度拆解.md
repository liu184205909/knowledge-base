## 3. 内容策略分析

### 3.1 SEO内容策略

Tiny Rituals 是"内容×电商双驱动"的典范：用免费内容获取SEO流量，自然引导到产品购买。

**博客规模**：9页+（每页约25-30篇）= 估计 **200-250篇博客文章**

**内容类型分布**：

| 内容类型 | 占比 | 示例 | SEO价值 |
|----------|------|------|---------|
| **水晶功效指南** | 30% | "Crystals for Sleep", "Crystals for Money", "Crystals for Beginners" | 极高，高搜索量 |
| **颜色/种类百科** | 25% | "Blue Gemstones", "Green Gemstones", "Yellow Crystal Names" | 高，信息型搜索 |
| **使用教程** | 20% | "How to Clean Quartz", "How to Activate Crystals", "Crystal Grid Patterns" | 高，长尾词 |
| **天使号码** | 15% | "Angel Number 25", "707 Angel Number" | **极高**，单篇流量最高 |
| **送礼/场合** | 5% | "Crystal Gifts for Men", "Crystal Gifts for Her" | 中，季节性 |
| **诞生石** | 5% | "June Birthstones", "February Birthstone" | 中，周期性 |

## 6. SEO策略分析

### 6.1 内容×电商双驱动模式

```
内容获取流量 → 产品页承接转化

具体路径：
1. 用户搜索 "amethyst meaning" → 到达博客文章
2. 文章内嵌 Amethyst 产品推荐/链接
3. 用户点击 → 到达 Amethyst 集合页(/collections/amethyst-crystals)
4. 浏览手链/原石 → 加入购物车 → 购买

关键：博客文章和产品页之间的无缝衔接
```

### 6.2 关键词矩阵

```
品牌词: "tiny rituals" / "tinyrituals"
产品词: "crystal bracelets" / "gemstone bracelets" / "worry stones"
水晶词: "[crystal name] meaning" × 34种 = 34个高价值入口
功效词: "crystals for [sleep/money/anxiety/love]" × N = 大量长尾
天使号码: "angel number [XXX] meaning" × 100+ = 爆款流量源
教程词: "how to [clean/charge/activate] crystals" = 教程型流量
```

### 6.3 内容矩阵乘法

```
34种水晶 × 5种功效维度(sleep/money/love/anxiety/protection) = 170+ 交叉页面
+ 天使号码 × 100+ = 100+ 文章
+ 颜色百科 × 10+ (blue/green/red/yellow/black/purple/...) = 10+ 综合页
+ 诞生石 × 12月 = 12个月度页面
+ 教程 × 10+ = 10+ 教程页
= 总计300+ 高质量SEO页面
```
