# 1C 关键词研究说明

> **RLM步骤1C输出**：把 1B 的关键词相关原始数据加工成可执行关键词库。

## 数据来源线

| 来源线 | 上游位置 | 说明 |
|------|---------|------|
| 竞品反推线 | 1B 竞品SEO明细（Top Pages / Top Keywords / 关键词排名明细） | 从竞品已经获得流量的页面和关键词反推需求 |
| 种子词扩展线 | Semrush Keyword Magic / Keyword Overview 导出 | 从产品词、类目词、场景词、用户问题扩展关键词 |
| 人工/AI候选线 | 项目种子关键词、内容策略候选方向 | 只能作为候选，必须后续验证数据 |

## 当前项目口径

- `semrush volume>100` Google Sheet 是当前水晶品类关键词库，包含 4,391 条关键词数据。
- 本目录的 `Semrush排除关键词列表.md` 用于 Semrush 导出前过滤明显无关词。
- 关键词研究的最终产物不是 Semrush 原始导出，而是经过去重、筛选、聚类、意图判断和页面类型映射后的关键词库。

## 待补充

当前关键词库主要覆盖水晶品类。项目简报中已标注：天使数字、MBTI、星座配对、月相、塔罗、灵性等非水晶跨界内容，执行前需要单独补充种子词导出和筛选。
