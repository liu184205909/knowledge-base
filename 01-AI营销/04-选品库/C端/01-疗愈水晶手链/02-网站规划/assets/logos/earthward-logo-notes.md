# EarthWard Logo Draft

## Current Direction

Use `EarthWard` as the primary wordmark:

- `Earth` = charcoal `#2B2A27`
- `Ward` = moss green `#596B4C`
- accent line = muted copper `#A66A43`
- primary light logo/icon background = white `#FFFFFF`
- warm ivory `#F7F3EA` remains available as a site background color

## Files

- `earthward-logo-header-light.svg` — primary 1920x600 header logo for light backgrounds
- `earthward-logo-header-dark.svg` — reversed 1920x600 logo for dark footer/header areas
- `earthward-favicon-icon.svg` — standalone favicon/social icon draft

## Notes

The PNG files in this folder are AI-generated visual explorations. The SVG files are the first editable website-ready drafts based on that direction.
