## 站点 A: Bad Dream Dictionary (baddreamictionary.com)

### 基本面
- **栈**:React/Vite SPA(`/assets/index-CXdRnO3v.css` + `manifest.json` + `favicon.svg`),移动友好
- **变现**:**Google AdSense**(`ca-pub-4139230886215582`)—— 主力
- **内容**:"31+ expert articles" + AI 分析器 + 符号字典(规模小)
- **关键词堆砌**:`keywords` meta 塞 9 个词(dream interpretation/analysis/meaning/symbols/AI analyzer/free dictionary/nightmare/lucid/psychology)—— 老式 SEO 痕迹
- **og 图**:180×180(直接复用 apple-touch-icon,粗糙)
- **主题色**:#4f46e5(indigo,现代)

### 优势
- 现代 React 栈,移动友好
- AI + 内容双轨(虽规模小)

### 缺口
- **E-E-A-T 最弱**:无作者 bio、无 about、无专家、无方法论透明页
- 内容规模小(31+ 篇 vs 大站千页级)
- AdSense 变现(低 RPM)
- og 图粗糙(复用图标)

### 候选策略
- **反面教材**:无权威 + 小内容 + AdSense = 我们要反着做(绑主理人 + 大规模 + Raptive)

## 站点 B: DreamPower (dreampower.app) ⭐ 方法论亮点

### 基本面
- **栈**:**Replit 构建**(`cdn.replit.com/agent/bootstrap-agent-dark-theme.min.css`)+ 原生 JS + voice-input css
- **方法论**(行业小站里最强):**荣格心理学 + Processwork(Arnold Mindell 博士)+ 原住民(Aboriginal-Informed)三栈**
- **定位**:明确 **"This Is Not a Dream Dictionary"**——批判字典"snake=fear, water=emotions"的一刀切,主张**个性化 3 问**(你的感受/联想/符号)→ 基于你的回答生成解读
- **多语言**:**5 语**(PL/IT/ES/EN/PT,alternate hreflang 全配)
- **8 大主题**:Snake / Relationship / Water / People / Threat & Crisis / Body / Animal / Place & Setting
- **特色功能**:
  - **语音输入**(audio transcription)
  - **Day Review**(日回顾 4 区:做了什么/接触谁/身体在做什么/大背景 → 找"soft spot"门槛点)
  - **Blog**:三视角解读(Jungian / Processwork / Aboriginal-Informed)

### 优势(值得对标)
1. **方法论透明 + 引用学者**(Mindell)——E-E-A-T 加分
2. **"非字典"反定位**——精准踩中"字典千篇一律"的痛点
3. **个性化 3 问**(感受/联想/符号)——比纯输入框深一层
4. **三视角解读**(荣格/过程工作/原住民)——独家多框架
5. **5 语布局**——欧洲市场覆盖(我们 TP+13 语有同思路)
6. **语音输入**——移动端友好

### 缺口
- 流量小(serp 未进头部)
- 无变现(免费,无显性付费层)
- 无 app、无实物
- 设计朴素(Replit 模板痕)

### 候选策略(高价值)
1. **"非字典"反定位 + 个性化 3 问** — 证据:homepage 原文。适用:我们解梦工具不做"输入梦→吐字典释义",而做"3 问(感受/联想/符号)→ 个性化解读 + 水晶推荐",差异化于 dreamslytic/dreaminterpreter.ai 的纯输入框
2. **三视角解读(心理学/灵性/水晶能量)** — 证据:blog 三视角。适用:每篇解梦给三层视角,覆盖理性 + 感性 + 实物用户
3. **方法论透明 + 学者引用(Mindell 对标)** — 证据:FAQ 引用。适用:我们引荣格 + 现代睡眠科学 + 矿物学,建立权威

## 站点 C: Centre of Excellence (centreofexcellence.com)

### 基本面
- **类型**:**课程电商平台**(梦是其众多垂直之一),非解梦专精站
- **栈**:Symfony 风格 coe-encore 构建 + jQuery UI + Google Tag Manager
- **变现**:**课程销售**(Dream Analysis & Therapy Diploma **£29**)+ 免费试听课引流(邮件列表)
- **内容**:灵性梦指南(13 个 H2 章节:Visitation/Prophetic/Chased/Falling/Flying/Water/Animals/Recurring/How to Interpret/FAQ)
- **水晶关联**(关键):**FAQ "Can crystals help with spiritual dreams?" 明确提 amethyst(增强梦回忆)+ labradorite(直觉 + 预言梦)**——直接水晶×解梦交叉点

### 优势
- 课程变现(高客单 vs 广告)
- 内容做深(13 章节 + FAQ)
- 机构认证背书(Diploma)

### 缺口
- 非解梦专精(梦是众多垂直之一)
- 内容为课程引流(非纯内容站)
- 无 AI 工具、无实物电商

### 候选策略
1. **课程变现层(Diploma/认证)** — 证据:£29 Diploma。适用:我们除电商 + 工具外,可做"水晶顾问认证"课程,高客单 B2B + B2C(对标 dreamtending Certification)
