## 2. 产品分层

tarotap 把"抽牌"和"解读"拆成**两个独立产品**：

```
tarotap.com/zh/
├── /draw-cards  ← 抽牌工具（免费，纯抽牌无解读）
│   └── "此功能仅提供塔罗抽牌体验，不包含AI解读"
└── /（首页）     ← AI 塔罗占卜（深度解读）
    └── "获得专业AI塔罗师的深度解读"
```

**可抄点（重要）**：goearthward 塔罗工具应明确分层 —
- **抽牌工具**（免费引流，纯抽牌+牌义）
- **AI 深度解读**（差异化/变现，结合水晶推荐）

这种分层避免"免费工具吃掉付费价值"。

## 3. 抽牌交互（核心拆解）

### 3.1 抽牌配置

webReader 实测，三步快速占卜 + 多配置项：

```
塔罗抽牌：在线三步快速占卜
├── 静心洗牌（点击「开始洗牌」，系统随机打乱 78 张）
├── 选择张数（1〜8 张，可启用正逆位与大阿尔卡纳模式）
└── 抽牌查看（点牌堆抽取，立即或统一显示）
```

**配置项**：
- 牌数：**1-8 张**（比 tarothall 的 1-12 窄，比 gomedia 固定5张灵活）
- **正逆位开关**（可关，去逆位简化）
- **大阿尔卡纳开关**（22 张模式）
- **自定义牌背**

### 3.2 自定义牌背

- 支持用户自定义牌背样式
- **可抄点**：个性化牌背是差异化体验，goearthward 可提供 3-5 套牌背主题（如水晶主题牌背，呼应站点调性）。

### 3.3 即时 vs 统一翻牌

两种翻牌模式：
- **即时翻牌**：抽一张翻一张
- **统一翻牌**：抽完所有牌一次性翻开

**可抄点**：统一翻牌模式强化仪式感（一次性揭晓），goearthward 默认统一翻牌 + 提供"即时"选项。

### 3.4 仪式感评估

| 仪式要素 | tarotap | 备注 |
|----------|---------|------|
| 洗牌交互 | ✓「开始洗牌」按钮 | 78 张随机打乱 |
| 切牌 | ✗ 无 | — |
| 翻牌模式 | ✓✓ 即时/统一可选 | 灵活 |
| 自定义牌背 | ✓✓ 个性化 | 差异化 |
| 牌数灵活 | ✓ 1-8 张 | — |
| 静心引导 | ✓ FAQ 引导 | "深呼吸三次" |
