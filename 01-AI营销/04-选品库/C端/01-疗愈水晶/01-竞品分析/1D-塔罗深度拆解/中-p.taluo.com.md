## 3. 抽牌交互（核心拆解）

### 3.1 抽牌流程

单牌阵页（以 `/1/` 单张为例，result.json 实测文本）：

```
单张牌占卜法
适合是非判断 & 单日运程
─────────────────────
[韦特塔罗-无牌阵单张 牌背展示区]

1  无牌阵单张：请专注问题,并保持静心
                    [开始抽牌]
```

**流程**：选牌阵 → 进牌阵页 → 静心引导文案 → 点"開始抽牌" → 翻牌 → 看牌义。

### 3.2 牌阵卡片展示

首页每个牌阵卡片（截图 `ptaluo--landing--viewport.png` 实测）采用**左右结构**：

```
┌─────────────┬──────────────────────────┐
│  牌阵示意图  │ 三张牌解析法              │ ← 大标题
│  (紫色牌位   │ 适合综合分析 & 单事解析   │ ← 橙色副标题（场景）
│   1 2 3     │ 本牌阵是萬能型…（说明）   │ ← 灰色描述
│   装饰花纹)  │            [開始抽牌]    │ ← 黑色按钮
└─────────────┴──────────────────────────┘
```

**关键 UI 元素**：
- **牌阵示意图**：装饰花纹边框 + 紫色牌位方块 + 数字标注（直观展示牌位排列）
- **场景副标题**：橙色小字点明"适合 XX"（降低选择成本）
- **每牌阵独立"开始抽牌"按钮**（非统一入口）

**可抄点**：牌阵示意图 + 场景副标题的组合，是牌阵选择页的最高转化设计，goearthward 应 1:1 复刻。

### 3.3 仪式文案引导

抽牌前强仪式话术（`/1/` 实测）：

> "占卜是一件嚴肅的事情,請認真對待,在抽牌之前請靜下心來,清除雜念,放空自己.然後點擊'開始抽牌'按鈕."

**配套文案矩阵**（侧栏）：
- 塔罗牌如何帮助你？
- 怎样提高解析的准确性？
- 如何快速而正確的記憶牌義？
- 怎麼解讀逆位的塔羅牌？

**可抄点**：仪式文案 + 学习引导双轨，既造仪式感又做内容留存。
