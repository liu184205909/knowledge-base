# 模板-水晶配对文章框架（`/crystal-combinations/{a}-and-{b}/`）

> **定位**：水晶×水晶配对。承接 "[stone A] and [stone B] together" / "can I wear X and Y together" 长尾。
> **实体**：post。URL `/crystal-combinations/{a}-and-{b}/`（66 组，12 颗水晶两两组合），category `crystal-combinations`。
> **对标**：Solacely 配对系列 + Navratan 配对文 + **差异化模块 "Three Perspectives" + 站内算法透明**。
> **依据**：竞品 sitemap 分析（Solacely 20+ 篇配对文）+ Google Suggest（"crystals that work well together" 等高意图词）。
> **创建**：2026-06-25

---

## 一、66 组配对说明

12 颗核心水晶（Amethyst / Rose Quartz / Clear Quartz / Citrine / Black Tourmaline / Carnelian / Selenite / Moonstone / Tiger's Eye / Labradorite / Pyrite / Jade）两两组合：C(12,2) = **66 组**。

含 6 组传统冲突（Carnelian+Amethyst / Moonstone+Citrine / Black Tourmaline+Jade / Selenite+Red Jasper / Rose Quartz+Obsidian / Turquoise+Malachite）——其中 Red Jasper 和 Obsidian/Turquoise 不在 12 颗内，实际冲突约 3-4 组。

URL 顺序固定（按字母序），如 `/crystal-combinations/amethyst-and-citrine/`。

## 二、文章结构（10 个 H2 模块）

```
# [A] and [B] Together: Benefits, Meaning + How to Use

## 1. Quick Answer（40-60 词）
   能不能一起戴？一句话回答 + 兼容性评分。
   "[A] and [B] can [be worn together / should be used separately]. [一句原因]. Compatibility score: [X]/100."

## 2. About [A]（100-150 词）
   颜色/元素/脉轮/核心传统用途。内链 /gemstone/[a]-meaning/。
   矿物参数一行（Mohs / 成分）+ genuine / ethically sourced。

## 3. About [B]（100-150 词）
   同上。

## 4. Can You Wear [A] and [B] Together?（200-250 词）⭐核心
   三视角分析：
   - 科学：元素互动 + 矿物硬度 + 颜色心理
   - 灵性：脉轮对应 + 能量场互动 + 传统搭配
   - 心理：intention 协同 + 仪式感锚点
   边界声明："traditionally associated with" / "many people find"

## 5. Benefits of [A] + [B] Combination（400-500 词）⭐差异化
   5-7 个具体好处，每个 60-80 词：
   基于共享 intention tags + 元素互动 + 脉轮对应生成。
   如 Amethyst + Citrine →
   - Balanced Energy（calm + abundance 平衡）
   - Crown + Solar Plexus Alignment
   - Spiritual Growth + Manifestation
   - Emotional Clarity
   - Creativity Boost
   - Protection + Prosperity

## 6. How to Use [A] and [B] Together（300-400 词）
   4 种方式（每种 60-80 词）：
   - Wear：bracelet / pendant / ring（左右手建议）
   - Meditate：持冥（各持一颗 / 放脉轮）
   - Crystal Grid：几何排列 + 意图设定
   - Place：卧室 / 工作台 / 玄关

## 7. Caring for [A] + [B]（150-200 词）
   净化/充电/存放注意事项。
   重点关注两颗水晶各自的安全要求（Water/Sun/Salt 三档）。
   内链 /blog/how-to-cleanse-crystals/

## 8. Shop [A] + [B]（产品导购）
   wd-products 块 + 套装推荐。
   "Bundle any 3 and save 20%"

## 9. FAQ（5-7 问，Schema）
   高频问题（从 Google Suggest / PAA 提取）：
   - Can I wear [A] and [B] together every day?
   - Which hand should I wear them on?
   - How do I cleanse them?
   - Can I add a third stone?
   - What if they're [conflicting]?
   FAQPage Schema（技术 SEO 差异化）

## 10. Related（内链集群）
   → /gemstone/[a]-meaning/ + /gemstone/[b]-meaning/
   → /crystal-combinations/[a]-and-[X]/（同系列配对）
   → /tools/crystal-compatibility-checker/
   → /crystals-for-[intention]/（场景页）
```

### 评分逻辑

复用 [compatibility-engine.js](../../07-互动工具/02-数据层/scripts/compatibility-engine.js) 的 `pairScore()` 函数：
- 元素矩阵 base score
- + 共享 chakra × 5
- + 共享 intention × 5
- + 站内 pairing 命中 +15
- 冲突库命中 → 25（强制）

### 12 颗水晶的属性

从 [crystal-attributes.json](../../07-互动工具/02-数据层/crystal-attributes.json) 提取：element / chakra / intentions / safety / pairings。

## 五、批量生成策略

1. **脚本生成配对数据**：从 12 颗水晶两两组合，用 pairScore() 计算评分（66 组，5 分钟）
2. **参数化模板**：模块 1/2/3/7/8/9/10 用脚本填充
3. **Agent 差异化**：模块 4/5/6 由 agent 逐对写（三视角 + Benefits + How to Use）
4. **QA Gate**：Kill List / 边界声明 / 相似度 / 冲突组合特殊处理

---

## 六、冲突组合特殊处理

6 组传统冲突中的 3-4 组在 12 颗内：
- Carnelian + Amethyst
- Moonstone + Citrine
- Black Tourmaline + Jade

冲突组合的文章框架调整：
- Quick Answer 明确说 "should be used separately"
- 模块 4 解释为什么冲突（元素/chakra/intention 对立）
- 模块 5 改为 "What Happens If You Combine Them" + "How to Use Them Separately"
- FAQ 增加 "Can I still wear them on different days?"
