## 三、vs 月运框架差异（模块级，证明需独立框架）

| | 月运 | 年运（本框架） |
|---|---|---|
| 组织骨架 | 月度主题+日期时间线 | **外行星长周期 transit/生活领域** |
| 时间粒度 | 单月某几天 | 全年：上下半年+行星过宫月份+季节 |
| 核心叙事 | "这个月能量/日期/事业" | "这一年 Jupiter/Saturn 在哪个宫→哪个领域" |
| 关键日期 | 当月某日 | 全年区间+精确相位日 |
| 占星深度 | 新月满月/内行星 | **外行星换座+守护星全年+相位** |
| 分区 | 否 | **上下半年** |

**年运 ≠ 月运拉长版**：年运是不同信息架构（外行星换座叙事+上下半年分区），直接派生月运会丢失年运核心。**共享复用**：Life Areas 分块 / Shop / FAQ / Crystal 仪式骨架（结构复用，内容按年运粒度重写）。
