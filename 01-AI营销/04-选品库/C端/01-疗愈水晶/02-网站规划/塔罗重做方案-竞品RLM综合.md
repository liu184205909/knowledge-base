## 三、重做方案

### 3.1 工具交互方案（evatarot 选牌模式 + love-healing 仪式 + tarotoo 反馈）

**核心选牌流程（5 步仪式，用户认可）**：

```
(1) 选牌阵（5 大场景分类 + 12 种 P0 牌阵）
       ↓
(2) 【开始洗牌】+ "默念你的问题" 引导文（love-healing 静心文案）
       ↓  Shuffle 按钮（牌背横向铺开洗牌动画）
(3) 点选牌堆切牌位置（love-healing 切牌点选，仪式感核心）
       ↓  Cut 按钮
(4) 按牌阵提示选牌（牌背横向一排铺开 + Choose N cards 引导 + 倒计数反馈）
       ↓  选满 → Validate 确认 → 统一翻牌（仪式感）
(5) AI 解读（牌阵+牌义+水晶推荐三合一）+ 3 个预设追问按钮
```

**交互细节（抄谁）**：
- **牌背横向一排铺开** ← evatarot（金棕复古花纹，我们改**水晶主题牌背**呼应站点调性）
- **Choose N cards 引导** ← evatarot（顶部明确提示选牌数）
- **Shuffle / Cut / Validate 三按钮** ← evatarot（仪式三段式）
- **点选切牌位置** ← love-healing（用户掌控感最强的一步，区别于西方自动切）
- **倒计数 Select 5/4/3/2/1** ← tarotoo（每选一张递减，视觉反馈）
- **统一翻牌模式**（默认）+ 即时翻牌选项 ← tarotap
- **可选问题框** ← evatarot（optional，喂给 AI）
- **可选逆位开关 + 大阿尔卡纳缩到 22 张**（新手模式）← trustedtarot/tarotap
