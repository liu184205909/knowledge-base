# T17 水晶手链定制工具 - 商品与素材表

> **状态**：待补全  
> **用途**：单独管理 T17 工具内所有可售/可选部件、素材来源、价格、履约字段和上线状态。  
> **关联**：[T17水晶手链定制工具-主文档.md](T17水晶手链定制工具-主文档.md)

---

## 1. 表格原则

- 本表是商品、图片、价格和履约字段的执行表，不再塞回 UI 规范。
- Crystals / Accessories / Charms / Packaging / Cord / Ready-made Presets 分开维护。
- 水晶珠子图优先保证转化视觉一致性；生产环境可参考或采用竞品同类珠子视觉快速落地，后续再替换为品牌自有资产。
- 其他 Tab 不能继续使用占位内容：必须补齐图片、价格、是否可选、是否进 3D、是否占槽位、订单 meta 字段。
- Ready-made Presets 是工具下方的预设产品列表，不等同于普通水晶首饰栏目。

---

## 2. 字段标准

| 字段 | 说明 |
|---|---|
| tab | Crystals / Accessories / Charms / Packaging / Cord / Ready-made Presets |
| item_id | 前端和订单中使用的稳定 ID |
| display_name | 前台英文名 |
| cn_note | 中文备注，仅内部使用 |
| image_source | 图片来源：competitor / own-site / supplier / generated / pending |
| image_url_or_path | 图片 URL 或本地路径 |
| price_6mm | 水晶用；非水晶可留空 |
| price_8mm | 水晶用；非水晶可留空 |
| price_10mm | 水晶用；非水晶可留空 |
| price_12mm | 水晶用；非水晶可留空 |
| fixed_price | 非水晶固定价格 |
| enters_3d | 是否进入手链 3D/2D 圆环 |
| slot_weight | 占几个槽位 |
| order_meta_key | 写入订单的字段名 |
| status | live / draft / pending_image / pending_price / disabled |
| notes | 履约、视觉或竞品参考备注 |

## 9. Checkout 后订单必须保存的信息

付款后订单中必须能看到并用于履约：

- 预览图 URL
- wrist_cm
- bead_size_mm
- cord
- packaging
- charms 数量和名称
- accessories 数量和名称
- beads_by_type：每种珠子数量
- sequence：逐槽顺序
- frontend_total
- server_recalc_total
- human_readable_summary

隐藏 Woo 承载产品只作为 line item 容器；不为每次用户定制生成新的 Woo 商品。
