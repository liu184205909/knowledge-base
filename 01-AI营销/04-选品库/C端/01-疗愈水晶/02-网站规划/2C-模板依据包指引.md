## 二、实例页最小差异记录格式

每个实例页需要记录"与模板的差异"，格式：

```json
{
  "slug": "amethyst-meaning",
  "template": "Crystal-Meaning文章框架",
  "differences": {
    "primary_keyword": "amethyst meaning",
    "crystal_name": "Amethyst",
    "poetic_name": "The Artist's Stone",
    "chakra": ["Third Eye", "Crown"],
    "zodiac": ["Aquarius", "Pisces"],
    "color": "Purple",
    "element": "Air",
    "hardness": "7",
    "chemical_formula": "SiO2",
    "key_properties": ["calm", "intuition", "spiritual awareness"],
    "product_recommendations": ["amethyst-bracelet", "amethyst-cluster"],
    "image_set": "1.crystal-meaning/amethyst/",
    "custom_sections": [
      {
        "title": "Amethyst in History",
        "reason": "历史价值高（古希腊酒神传说）",
        "source": "crystal-attributes.json > history"
      }
    ]
  },
  "brief_locked": true,
  "qc_status": "passed"
}
```

## 三、自动化生成差异记录

大部分差异记录可以从数据源自动生成，不需要手写。

### Crystal Single（390 篇）

```bash
# 从 crystal-attributes.json 自动生成差异记录
node 07-互动工具/_shared/scripts/extract-crystal-profile.js \
  --input _shared/crystal-attributes.json \
  --output 02-网站规划/pages/crystal-single/diff-records/
```

输出：390 个 JSON 文件，每个对应一种水晶的差异记录。

### 其他模板矩阵页

同样从对应数据源生成。需要编写生成脚本（未来任务）。
