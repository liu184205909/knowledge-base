## 六、AI 占星师 MVP（皇冠深度入口）

**工具名**：`Astrology Crystal Reading`（或 `Cosmic Crystal Reading`）
**URL**：`/tools/astrology-crystal-reading/`

### 流程
1. 用户输入：出生日期 + 时间 + 地点（+ 可选当前困扰：爱情/事业/焦虑…）
2. swisseph-js 算完整盘 → 渲染 SVG 盘图
3. LLM 生成个性化解读（太阳/月亮/上升 + 关键相位 + 当前行运）
4. **核心差异化**：基于盘的行星/宫位/element 缺失 → 从 390 水晶库匹配 3-5 颗"补能量"水晶，每颗给占星依据
5. 每颗水晶 → Shop CTA（三级降级）
6. 可选：兼容性报告（输入两人 → 关系水晶）

### 技术方案
| 出生盘计算库 | 语言 | 精度 | 适用 |
|-------------|------|------|------|
| **swisseph-js**（首选）| TypeScript | 天文级（Swiss Ephemeris 封装）| 站是 Node/JS 栈，直接用 |
| pyswisseph | Python | 行业标准 | Python 后端时 |
| Kerykeion | Python | 高级封装+SVG 图 | 快速出盘图 |
| AstrologyAPI.com | 商业 API | 300+ 端点+内置 AI | 最快上线但持续成本 |

**LLM + 占星架构**：计算层（swisseph-js → 10 行星+12 宫+相位 JSON）→ 知识层（390 水晶 JSON）→ LLM 层（占星 JSON + 用户问题 → 解读 + 选 3-5 石）→ 导购层（Shop CTA 三级降级）。

**成本**：每次解读 ~5,774 tokens，$0.01-0.03/次（远低于水晶客单价）。免费层限额（每日 3 次）+ 邮箱注册换额外。

**MVP 开发量**：盘计算+表单+SVG（3-5天）+ LLM prompt+水晶映射（3-5天）+ 结果页+Shop 联动（2-3天）= **约 2 周**（复用 _shared 工具层 + 390 水晶数据 + Crystal Checker）。

**SEO 目标词**：'AI astrology reading' / 'birth chart crystal reading' / 'crystals for my birth chart'（Google Suggest 有真实搜索量且竞争低）。

## 八、商业化策略（单次付费 vs 订阅）

> **核心结论**：用"单次付费报告 $9.99 + 邮件自动化流"组合，可在不引入订阅复杂度的前提下，实现接近订阅的 LTV（$80-120/年/用户），且与电商主业务强协同。**订阅留到流量 5 万 UV / 邮件列表 5,000 人时再做**，钩子是"个人化月运行运报告"而非静态占星报告。

### 1. 单次付费深度报告（强推荐）

| 维度 | 分析 |
|------|------|
| **成本结构** | LLM $0.01-0.03/次 vs 售价 $9.99 → **毛利率 99.7%**（远高于实物电商 50%） |
| **已有规划** | L3 已列"Personalized Crystal Prescription Report $9.99"，AI 占星师是天然的内容生产引擎 |
| **竞品验证** | hint.app $29.99/月、Nummi $7.99/月证明用户愿为个性化占星付费；但他们没水晶，我们有 |
| **用户画像匹配** | "灵性探索型" 15% 流量占比，转化率 3-5%，客单价 $39-99/月——愿意为深度付费 |
| **不影响主漏斗** | 免费层照常引流 Shop；付费墙只在"想看更深度"时出现，是纯增量 |

**Freemium 切分**：

| 层级 | 内容 | 目的 |
|------|------|------|
| **免费** | 完整盘图 SVG 渲染（视觉钩子）+ 太阳/月亮/上升三句话概述 + 1 颗主推荐水晶 + Shop CTA（三级降级） | 引流 + 留资 |
| **付费 $9.99** | 10 行星 + 12 宫位完整解读 + 关键相位分析 + 当前行运 + 5-8 颗水晶匹配（主推荐 3 颗 + 扩展 2-5 颗，每颗含占星依据）+ PDF 导出 + 30 天后"行运更新"提醒邮件 | 深度变现 |

**切分原则**：盘图免费（视觉钩子），解读深度付费（信息钩子）。

### 2. 为什么现阶段不做订阅

| 反对理由 | 详细 |
|---------|------|
| **占星报告是 snapshot 非 stream** | 出生盘一辈子不变，看一次就够。强行订阅 = 用户疑惑"每月付钱看同样的盘？" |
| **流量基数不足** | 5,000 UV/月 × 2% 订阅转化 × $9.99 = $999/月毛收入；扣流失后稳定付费约 60-70 人，规模 < 订阅系统工程成本 |
| **与电商定位冲突** | 主营收入是水晶首饰 $35 客单价。订阅会把注意力从"买水晶"转移到"看运势"，**用户付 $9.99/月后可能不再花 $35 买水晶**，LTV 反而下降。hint/Co-Star 走订阅是因为没有实物可卖，我们有 |
| **运营复杂度高** | billing/退款/流失挽回/Stripe/PayPal/客服成本。早期应聚焦内容 SEO + 电商转化，不是 SaaS 化 |

### 3. 推荐流程：付费报告 + 邮件列表替代订阅

   ↓ 分流
   ├─→ Shop 直接购买（转化 1-2%）
   ├─→ 邮箱注册换"行运提醒"（留资 15-25%）
   └─→ "Unlock Full Report $9.99"（付费转化 2-5%）
       ↓
Layer 3: 付费深度报告（完整解读 + 3-5 水晶 + PDF）
   ↓
Layer 4: 邮件自动化流（替代订阅的核心）
   - D0 注册：盘图回顾 + 主推荐水晶
   - D3 未购买：限时 10% off
   - D7 占星事件：水逆 + 你盘受影响 + 缓解水晶
   - D30 月运推送：你星座本月重点 + 推荐水晶
   - D90 复购提醒：你 3 月前的 Amethyst 搭配 Rose Quartz 更佳
