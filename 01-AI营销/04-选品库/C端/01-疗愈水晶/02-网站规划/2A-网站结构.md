## 一、URL 总原则

1. **层级 ≤ 3**（域名后最多 2 层路径）
2. **类目名含关键词**（URL slug 包含目标英文关键词）
3. **产品用 `/products/`**，内容用 `/blog/` 或 Topic 前缀
4. **全部小写**，单词间用 `-` 连接
5. **禁止年份/日期**（月相/运势类除外）
6. **WooCommerce 产品页由系统自动生成**，不手动创建
7. **内容分类 spoke 页根级平铺**，Hub 索引页单独承担导航与内链中心；分类 spoke 与 Hub 的 slug 均带 `crystals`，锚定水晶商业意图

### 内容分类页（B2）⭐ 2026-06-17 更新

> **结构原则**：spoke 页（具体某个星座/脉轮/颜色/元素/功效的水晶页）**根级平铺**，URL 短、完整匹配搜索词；Hub 索引页（列出该类所有 spoke）单独承担导航和内链中心。
>
> **命名原则**：spoke 与 Hub 的 slug **均带 `crystals`**，目的是锚定"水晶"商业意图、过滤纯知识流量——用户搜 "root chakra crystals"（明确要水晶推荐）而非 "root chakra"（混杂了解脉轮的流量，且与纯脉轮知识站正面竞争）。不使用不带 crystals 的 `/chakras/`、`/zodiac/`。
>
> **功效页 vs 意图页不冲突**：`/crystals-for-anxiety/`（内容推荐页，"哪些水晶适合抗焦虑"）与 `/calm-mindfulness/`（意图页，"平静正念水晶"，WordPress page + 产品聚合）是不同页面，分工明确。注意：意图页是根级 WordPress page（/calm-mindfulness 等8个），**不是** /collections/（那是 Shopify 术语，本站无 collection 页）。

## 四、内链规则（写文章 / 锁定 Brief 时用）

> 本节面向"写一篇文章时该加哪些内链"的实操场景，保持简洁。规则按页面类型给出"该链到哪 + 位置 + 锚文本"，不绑定具体 Section 编号（模块结构会演进）。

### 4.1 总原则

| 原则 | 说明 |
|------|------|
| 上下层级链接 | 百科 → 分类 → 产品，各层双向链接 |
| 主题关联链接 | 同主题下的内容互链 |
| 意图递进链接 | 信息型 → 商业考察型 → 交易型 |
| 锚文本描述性 | 用描述性锚文本，不用"点击这里""learn more" |
| 每页内链上限 | 正文内链 5–15 个（视篇幅），避免过度 |
| 每个 CTA 必须有 | 每个内容页至少 1 个 CTA 指向产品/商店 |

### 4.2 各页面类型的内链清单

**Crystal Single（`/gemstone/{slug}-meaning/`）** — 写水晶百科时：

| 链接到 | 位置 | 锚文本示例 | 数量 |
|--------|------|----------|------|
| Crystal Guide Index | 面包屑/页脚 | "Crystal Guide" | 1 |
| 对应 Crystals by Condition | 功效描述处 | "Crystals for Anxiety" | 1–2 |
| 对应 Intention Category | 产品触点/CTA | "Shop Anxiety Relief" | 1 |
| 相关 Crystal Single | Pairings 模块 | "Amethyst pairs beautifully with Rose Quartz" | 2–3 |
| Shop by Stone（该水晶） | sidebar CTA | "Explore Amethyst Jewelry" / "Shop Amethyst Crystals" | 1 |
| Crystals by Color（该水晶颜色） | 属性区 | "See all purple crystals" | 0–1 |
| Crystals by Zodiac（对应星座） | 属性区 | "Amethyst for Pisces" | 0–1 |

**Crystals by Condition（`/crystals-for-{condition}/`）** — 写功效页时：

| 链接到 | 位置 | 锚文本示例 | 数量 |
|--------|------|----------|------|
| Crystal Guide Index | 面包屑 | "Crystal Guide" | 1 |
| 涉及的 Crystal Single | 每张水晶卡片 | 水晶名（链到 meaning 页） | 按推荐数 |
| 对应 Intention Category | 产品区 CTA | "Shop Anxiety Relief Crystals" | 1 |
| 相关 Condition 页 | 相关推荐 | "Also helpful: Crystals for Stress" | 1–2 |
| 对应 Hub（如有） | 顶部/面包屑 | — | 0–1 |

**Intention Page（`/{intention-slug}`，如 `/calm-mindfulness`）** — 写意图页时：

| 链接到 | 位置 | 锚文本示例 | 数量 |
|--------|------|----------|------|
| Crystal Guide Index | 面包屑 | "Crystal Guide" | 1 |
| 涉及的 Crystal Single | 水晶卡片 | 每张卡片链到百科 | 3–8 |
| Shop by Stone | CTA | "Shop All Crystals" | 1 |
| 相关 Intention Category | 相关推荐 | "Explore Crystals for Sleep" | 1–2 |
| 对应 Crystals by Condition | 内容引导 | "Learn: Best Crystals for Anxiety" | 1 |

**博客文章（`/blog/{slug}`）** — 写博客时：

| 链接到 | 位置 | 锚文本示例 | 数量 |
|--------|------|----------|------|
| 提到的 Crystal Single | 正文首次提及 | "Amethyst, the stone of peace" | 按内容 |
| 相关 Crystals by Condition / Intention | 文末相关推荐 | "Best Crystals for Anxiety" | 1–2 |
| 产品页 / Shop by Stone | CTA | "Shop Crystal Jewelry" | 1 |

**分类 spoke（Zodiac / Chakra / Color / Element）** — 写分类页时：

| 链接到 | 位置 | 数量 |
|--------|------|------|
| Crystal Guide Index | 面包屑 | 1 |
| 对应 Hub 索引页 | 顶部/同类导航 | 1 |
| 涉及的 Crystal Single | 每张卡片 | 按涉及水晶数 |
| 相关分类 spoke | 交叉推荐 | 1–3 |

### 4.3 锚文本规范

| 类型 | ✅ 正确 | ❌ 错误 |
|------|--------|--------|
| 水晶百科 | "Amethyst's calming properties" | "click here" |
| 功效/分类页 | "Best crystals for anxiety" | "learn more" |
| 产品页 | "Explore Amethyst Jewelry" / "Shop Amethyst Crystals" | "buy now" |
| 相关内容 | "How to cleanse your crystals" | "read this" |

### 4.4 意图递进链路（信息 → 购买）

用户从信息搜索到购买的典型路径，**每个 Crystal Single 必须完整实现这条链路**：

```
Google 搜 "amethyst meaning"
  → Crystal Single: /gemstone/amethyst-meaning/        （信息型）
    → 功效描述: "Crystals for Anxiety"                  （商业考察型）
      → Crystals by Condition 或 Intention Category     （考察加深）
        → Shop CTA: "Shop Anxiety Relief Crystals"     （交易型）
          → Product Page: /products/xxx                 （转化）
```
