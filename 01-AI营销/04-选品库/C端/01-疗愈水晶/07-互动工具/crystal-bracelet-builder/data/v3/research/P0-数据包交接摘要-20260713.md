# T17 v3 P0 数据包交接摘要

日期：2026-07-13。范围：只读竞品/来源研究与非生产候选准备。未修改 T17 前端、WordPress 插件、WooCommerce、线上页面或 `approved-production-catalog.import.csv`。

## 数据线边界（已锁定）

- **素材数据线**：服务于 T17 独立素材库。珠子、配件、绳材、包装的来源、图片、规格、价格与生产审核均在此线完成；单个素材或 Variant 不创建 Woo 产品。
- **官方预设产品数据线**：服务于不懂搭配用户的完整官方设计候选。只有完成素材、配方、成品图、价格和运营审核后，候选才可被单独创建为 Woo Official Design Product，并支持直接购买或导入编辑器修改。
- 竞品素材、竞品预设、用户设计广场作品和素材库单件都不会自动成为 Woo 产品。本 P0 包只做研究、候选整理与审核准备。

## 已核验

- **来源可读**：Annflora、MoonArq Studio、Lucid Beads、Crystalis，以及 MyAstris 的指定 Web Archive 归档页。
- **图片 URL 可见**：Annflora 的多款珠子，Lucid 的部分工作区珠子，MyAstris 归档中的部分珠子/配件。它们均明确标为竞品/归档参考，不是 Earthward 可生产主图。
- **可按 Variant 读取的海外价格样本**：Annflora 13 条，含 Clear Quartz 6/8/10/12mm 的 USD 3/5/8/12；MoonArq 4 条 10mm 单颗增量样本。详情以标准化表为准。
- **不可混用样本**：Lucid USD 29.99 与 MoonArq USD 35 是 builder 起始价；MyAstris 的部分 USD 1-4 卡片未显示尺寸。这些都没有写成单颗材料价格。

## 仅可见但未取得

- MoonArq 公共 builder 的材料名、尺寸和加价可见，但本轮未取得独立原图或完整单颗基础价格。
- Crystalis 的场景分组可见，但未在观察页取得可用于材料单价的卡片数据。
- MyAstris 当前主站返回访问拒绝；指定归档页可读，归档资产仅作为参考。

## 无法读取 / 待用户提供

- 珠了个珠、晶奥之境、记珠、拾光水晶、灵感石：已记录用户提供的小程序深链接，但没有发现可合规读取的公开目录路线。需要用户提供已授权截图、导出、公开网页或浏览器已打开的可读页面。
- 所有生产候选仍缺：供应商、SKU、成本、图片权利、规格/材质确认、库存、方向规则、适配珠径、包装规格。

## 价格与版本建议

- 推荐研究草案：`t17-price-research-20260713-r0`。
- 当前可给出的结论是**参考价格区间 / 待成本确认**，不是 Earthward 最终零售价。
- `approved` 的条件是：供应成本、损耗、人工、包装、履约、支付/售后预留、目标毛利、审核人和审核时间完整；在此之前所有候选保持 `research-only`。

## 主线下一步

1. 从 17 条来源台账中选定第一批真实供应商与拥有图片权利的素材，生成稳定 `material_key` / `variant_key`。
2. 用实际成本填充研究价格表，按定价公式形成待审核单价。
3. 从 32 个候选中筛选首批 Official Design，并在所有 Variant 已 live、配方和成品图已审核后，再创建 Woo 产品和可 Customize 配方。
4. 任何未通过上述审核的行都不得进入生产 catalog 或 Woo。
