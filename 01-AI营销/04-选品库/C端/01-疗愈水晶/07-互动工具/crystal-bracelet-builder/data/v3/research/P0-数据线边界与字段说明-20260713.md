# T17 v3 P0 数据线边界与字段说明

状态：`research-only`。日期：2026-07-13。

本文件只约束 P0 数据采集与审核准备。它不修改 T17 前端、插件、WooCommerce、WordPress 页面、`approved-production-catalog.import.csv` 或 v3 主需求基线。

## P0-A 素材数据线

对象：珠子、配件、绳材、包装等单个零件；目标是 T17 独立素材库候选。

最低字段：名称、类别、颜色、尺寸、图片取得状态、来源、公开价格、适配关系、方向规则、库存或售罄状态、备注。

边界：单个素材与其 Variant 均不是 Woo 产品。竞品素材图片仅为研究参考，除非已取得自有、供应商授权或明确许可，否则不可标记为 Earthward 生产主图。

## P0-B 官方预设产品数据线

对象：国内小程序及海外重点竞品中的完整设计款、模板或设计师设计成品；目标是 Earthward Official Design Product 的研究候选。

每条完整设计样本必须记录：来源与直接证据链接、设计名称、完整图状态、主场景/寓意/标签、可见售价、手围或佩戴信息、珠子/配件/包装配方证据、是否可由 Earthward 已批准 Variant 重建、缺失项、证据状态。

边界：竞品设计款和用户设计广场作品都是 **Earthward 官方预设的候选来源**，但不自动创建 Woo 产品；“来源为用户设计广场”不等同于“已是 Earthward 官方预设”。候选必须先完成素材映射、配方复核、Earthward 价格重算、使用自有工具重新生成成品图，以及运营审核；完成后才能创建为 Woo Official Design Product，并支持直接购买或导入编辑器继续修改。竞品/用户原图仅作采集证据和重绘参考，不作 Earthward 商品主图。

当前 `official-design-candidates-20260713.csv` 中的 32 条为内部场景/名称研究槽位，不含逐条完整设计证据、可生产配方或已批准 Variant 映射。因此：已核验完整设计样本为 0；可由已批准 Variant 重建的官方预设为 0。

## P0-C 价格与审核数据线

对象：国内四个小程序及 MyAstris、MoonArq、Annflora、Lucid Beads、Crystalis 的公开价格，以及 Earthward 研究候选审核行。

口径：珠子按“单颗 + 尺寸”；配件按“单件”；整串与构建器起始价使用独立计价类型。竞品公开价只能用于价格区间校验，不能直接成为 Earthward 售价。

公式：

```text
建议零售价 =
（供应成本 + 损耗 + 人工 + 包装 + 履约 + 支付/售后预留）
÷（1 - 目标毛利率）
```

没有供应成本时，审核行只能标记“待成本确认”或 `research-only`，不得填写为已批准最终零售价。

## 统一状态词

- `verified-public`：公开页面或用户授权会话中已实际核验。
- `visible-not-obtained`：可见，但没有取得可复核的原图、价格或配方证据。
- `access-not-available`：没有合规的公开或已授权读取路径。
- `pending-user-input`：需要用户提供授权截图、导出、公开链接或已打开的可读页面。
- `pending-review` / `research-only`：资料已登记，但尚未满足生产或产品审核门槛。
