## 联动机制

22 工具通过 `related-html.js` 实现双向 CTA 联动：
- 每个工具结果页推荐 2-3 个相关工具
- 联动 MAP 定义在 `_shared/related-tools/related-html.js`
- 同时需同步修改 `snippet.php`（WordPress 端注入）

---

## 重新生成命令

```bash
# 单个工具重新生成
cd 07-互动工具/{tool-name}/build/
node generate.js

# 更新到线上
node ../../_shared/scripts/update-page.js --slug={tool-slug}

# 联动 CTA 重新注入
cd ../../_shared/related-tools/
node related-html.js
```
