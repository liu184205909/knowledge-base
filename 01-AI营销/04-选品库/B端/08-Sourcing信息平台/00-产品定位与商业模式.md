# Sourcing 信息平台 — 产品定位与商业模式

> 创建时间: 2026-05-29
> 项目类型: B端 SaaS / 信息服务平台
> 关联资产: sourcingagents.net（已有 sourcing agent 服务网站）
> 灵感来源: 有机食品供应商/买家聚合平台模式（展会数据 → 结构化目录 → 订阅付费）

---

## 一、产品定义

**一句话定位**：基于 sourcingagents.net 的数据引擎，为海外买家提供中国供应商结构化信息与智能匹配服务。

```
核心逻辑：

  传统 sourcingagents.net          Sourcing 信息平台（升级）
  ─────────────────────         ──────────────────────────
  服务型（人工对接）       →     平台型（数据驱动 + 人工增值）
  1对1 定制服务           →     1对多 自助查询 + 深度报告
  靠佣金/服务费           →     订阅 + 佣金 + 增值服务 三层变现
  被动等待询盘            →     主动 SEO/内容引流
```

---

## 二、为什么选 Direction A（帮海外买家找中国供应商）

### 2.1 两个方向对比

| 维度 | A. 海外买家 → 中国供应商 | B. 中国供应商 → 海外买家 |
|------|--------------------------|--------------------------|
| **与现有业务协同** | 直接协同（sourcingagents.net 已在做） | 无协同，需独立冷启动 |
| **客户付费意愿** | 高（节省采购时间 = 省钱） | 中（国内供应商对信息付费敏感） |
| **竞争格局** | 分散，中大型玩家少 | 国内数据服务商已成熟（企查查/1688） |
| **壁垒** | 已有网站 + 英文能力 + 跨境经验 | 需从零建海外买家流量 |
| **变现路径** | 订阅 → 佣金 → 增值服务三层 | 主要靠会员订阅 |
| **SEO 优势** | 英文长尾关键词竞争低 | 中文 B2B 关键词竞争激烈 |
| **数据获取** | 展会名录/海关数据/工厂认证（已有渠道） | 需要海外买家数据（难获取） |

### 2.2 决策

**选择 Direction A**，原因：
1. sourcingagents.net 已有品牌基础和 SEO 权重
2. 信息平台作为数据层叠加在现有服务上，自然延伸
3. 海外买家付费意愿更强，LTV 更高
4. 英文 SEO 长尾词空间大

## 三、产品架构

### 3.1 三层产品模型

```
┌─────────────────────────────────────────────────┐
│  Layer 3: 增值服务（高利润）                       │
│  ├── 一对一 Sourcing 顾问（对接 sourcingagents）    │
│  ├── 工厂验厂报告（QC 服务）                        │
│  ├── 物流方案定制                                   │
│  └── 定制供应链报告                                 │
│                                                   │
│  Layer 2: 付费订阅（核心收入）                      │
│  ├── 供应商深度档案（认证/产能/合作客户）            │
│  ├── 智能匹配推荐（AI 匹配需求 vs 供应商能力）       │
│  ├── 价格趋势监控                                   │
│  └── 导出/批量查询                                  │
│                                                   │
│  Layer 1: 免费层（引流）                           │
│  ├── 供应商目录浏览（基础信息）                      │
│  ├── 品类市场概览                                   │
│  ├── 采购指南文章（SEO 内容）                       │
│  └── 免费样品申请                                   │
└─────────────────────────────────────────────────┘
```

### 3.2 核心功能模块

| 模块 | 功能 | 优先级 |
|------|------|--------|
| 供应商目录 | 按品类/地区/认证/产能筛选 | P0 |
| 智能搜索 | 关键词 + 参数规格搜索 | P0 |
| 供应商档案 | 公司信息/产品线/认证/评价/合作记录 | P0 |
| 采购需求匹配 | 买家发布需求 → 系统匹配供应商 | P1 |
| 价格参考 | 品类价格区间/趋势 | P1 |
| 内容库 | 采购指南/品类分析/工厂考察报告 | P1 |
| 用户后台 | 收藏/对比/询盘记录/订阅管理 | P0 |
| 数据导出 | CSV/Excel 批量导出供应商清单 | P2 |

## 四、数据来源策略

### 4.1 供应商数据获取渠道

| 渠道 | 成本 | 数据质量 | 可行性 |
|------|------|---------|--------|
| **国内数据服务商** | 低-中 | 高（工商/海关/认证数据） | ✅ 最可行 |
| 展会名录（广交会/华交会等） | 低 | 中（联系方式可能过时） | ✅ 可行 |
| 1688/阿里巴巴公开数据 | 低 | 中 | ✅ 可行 |
| 工厂认证数据库（ISO/UL/CE） | 中 | 高 | ✅ 可行 |
| 海关出口数据 | 中-高 | 高 | P1 |
| 爬取公开企业信息 | 低 | 中 | 需合规评估 |
| 供应商自主入驻 | 低 | 取决于审核 | P1（平台上线后） |

### 4.2 核心数据字段

```
供应商基础信息：
  - 公司名（中英文）
  - 地区/地址
  - 成立年份
  - 员工规模
  - 年营收范围
  - 出口国家

产品能力：
  - 主营品类
  - 产品线清单
  - OEM/ODM 能力
  - MOQ
  - 产能（月/年）

认证与合规：
  - ISO 9001 / 14001
  - 行业认证（UL/CE/FDA 等）
  - 验厂报告（BSCI/Sedex）

商务信息：
  - 联系方式（公开层 vs 付费层）
  - 合作客户案例
  - 出口记录摘要
  - 价格参考区间
```

## 五、变现模型

### 5.1 定价策略

| 层级 | 价格 | 包含内容 | 目标客户 |
|------|------|---------|---------|
| Free | $0 | 基础搜索 + 目录浏览 + 5个供应商详情/月 | 引流 |
| Starter | $29-49/月 | 50个供应商详情/月 + 基础导出 | 小型买家 |
| Professional | $99-199/月 | 无限详情 + 智能匹配 + 价格趋势 + 批量导出 | 中型买家 |
| Enterprise | $499+/月 | 定制报告 + 专属顾问 + API 接入 | 大型买家 |
| 单次报告 | $49-149/份 | 单品类/地区深度供应商报告 | 一次性需求 |

## 六、与 sourcingagents.net 的关系

### 6.1 产品定位关系

```
sourcingagents.net（母站）
  ├── 现有服务: Product Sourcing / QC / DropShipping / Freight / Fulfillment
  │
  └── 信息平台（子产品）
      ├── 独立子域名: suppliers.sourcingagents.net 或 sourcing.data platform
      ├── 共享: 品牌信任 / 用户体系 / 供应商数据库
      └── 转化路径: 免费用户 → 付费订阅 → sourcing 服务（回到母站）
```

### 6.2 协同效应

| 协同点 | 说明 |
|--------|------|
| **流量闭环** | 信息平台 SEO 引流 → 免费使用 → 付费 → 采购服务（母站） |
| **数据飞轮** | 服务客户积累的供应商数据 → 信息平台内容 → 吸引更多客户 |
| **信任背书** | sourcingagents.net 已有 500+ specialists、60+ industries 的信任基础 |
| **运营复用** | 客服/运营团队可同时服务两个产品线 |

## 七、关键假设与风险

### 7.1 需验证的假设

| # | 假设 | 验证方式 |
|---|------|---------|
| 1 | 海外买家愿意为结构化供应商信息付费 | 预售/Landing Page 测试 |
| 2 | 国内数据服务商的数据质量满足需求 | 采购样本数据验证 |
| 3 | 英文 SEO 能在 6 个月内获取有效流量 | 竞品关键词分析 |
| 4 | 供应商数据更新频率可维持（月更/季更） | 数据维护成本估算 |
| 5 | 3% 付费转化率可达成 | 对标竞品（ThomasNet/Kompass） |

### 7.2 风险与应对

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| 数据获取成本超预期 | 中 | 高 | 先从 1-2 个品类验证，再扩展 |
| 付费转化率低于预期 | 中 | 高 | 增加免费层价值，拉长转化漏斗 |
| 竞品模仿 | 低 | 中 | 数据深度 + 服务整合是壁垒 |
| 数据合规风险 | 低 | 高 | 仅使用公开/授权数据，法务审核 |
| SEO 排名起效慢 | 高 | 中 | 配合 Google Ads + 社媒引流 |
