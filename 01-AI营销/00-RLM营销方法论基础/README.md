## 📖 什么是RLM营销方法论?

**RLM = 递归语言模型 = 让AI学会"分而治之"**

### 传统方式 vs RLM方式

| 传统方式 | RLM方式 |
|---------|--------|
| 用户 → AI → 一次性输出 ❌ | 用户 → AI → 分阶段执行 ✅ |
| 容易遗漏、泛泛而谈 | 每步专业处理、深入分析 |
| 依赖AI训练数据(过时) | MCP工具实时搜索 |
| 难以复用 | 流程文档可复用 |
| 2-3天 | 2-3分钟 |

### 核心原理

```
复杂任务: "制定营销策略"
  ↓ AI分而治之
阶段1: 市场机会洞察
  ├─ 产品评估(100分制)
  ├─ 竞品发现(搜索15+)
  ├─ 竞品分析(深度5-10家)
  └─ 用户洞察(画像+痛点)
  ↓ 输出: 市场分析报告

阶段2: 差异化定位策略
  ├─ 差距分析(竞品vs用户)
  ├─ 独特价值主张
  ├─ 内容架构设计
  └─ SEO布局策略
  ↓ 输出: 差异化策略

汇总: 完整营销方案
```
