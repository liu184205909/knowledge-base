# 社媒内容生产全流程

**版本**: v1.0
**创建时间**: 2026-01-16
**基于**: RLM递归思想

---

## 🎯 核心价值

> **从创意到发布的完整社媒内容生产**

基于 **RLM递归思想**的社媒内容生产流程，系统化完成：
1. ✅ 平台选择 (IG/FB/TikTok/小红书/微博)
2. ✅ 内容规划 (图文/视频/直播策略)
3. ✅ 短内容创作 (50-800字)
4. ✅ 视觉设计 (图片/视频/缩略图)
5. ✅ 发布优化 (最佳时间 + 互动策略)

**专注于社交媒体平台的内容营销**

---

## 🔄 工作流程

### 阶段 1: 平台选择与策略

**输入**: 产品/品牌 + 目标受众
**输出**: 平台选择报告 + 内容策略

#### 主流社媒平台特性

**Instagram (IG)**
- **内容形式**: 图片、IGTV、Reels、Stories
- **内容风格**: 视觉化、美学、生活方式
- **受众特征**: 18-34岁，女性偏多
- **最佳发布时间**: 11am-1pm, 7pm-9pm
- **互动率**: 高 (视觉驱动)

**Facebook (FB)**
- **内容形式**: 图文、视频、直播、群组
- **内容风格**: 信息丰富、故事性强
- **受众特征**: 25-54岁，广泛分布
- **最佳发布时间**: 1pm-4pm
- **互动率**: 中等

**TikTok**
- **内容形式**: 短视频 (15-60秒)
- **内容风格**: 创意、娱乐、趋势驱动
- **受众特征**: 16-24岁，Z世代
- **最佳发布时间**: 7pm-9pm, 10pm-12am
- **互动率**: 极高 (算法驱动)

**小红书**
- **内容形式**: 图文、短视频、直播
- **内容风格**: 种草、评测、生活方式
- **受众特征**: 18-35岁，女性为主
- **最佳发布时间**: 9pm-11pm
- **互动率**: 高 (种草文化)

**微博**
- **内容形式**: 图文、视频、直播
- **内容风格**: 热点话题、明星效应
- **受众特征**: 18-40岁，广泛分布
- **最佳发布时间**: 12pm-2pm, 8pm-10pm
- **互动率**: 中等

#### 平台选择策略

**单平台专注**:
```
如果资源有限 → 选择1个最匹配的平台深耕
```

**多平台布局**:
```
如果资源充足 → 选择2-3个平台组合
```

**关键输出**:
- 选择哪些平台 (1-3个)
- 每个平台的内容策略
- 内容形式优先级 (图文/视频/直播)
- 受众匹配度分析

### 阶段 2: 内容规划

**输入**: 平台选择 + 产品/品牌
**输出**: 内容日历 + 主题列表

#### 内容类型分类

**1. 产品展示类** (30%)
- 产品特写
- 功能演示
- 使用场景
- 包装开箱

**2. 教育内容类** (25%)
- 使用教程
- 知识科普
- 技巧分享
- FAQ解答

**3. 用户故事类** (20%)
- 用户评价
- 成功案例
- 使用心得
- 晒单激励

**4. 品牌文化类** (15%)
- 幕后故事
- 团队介绍
- 品牌价值观
- 慈善活动

**5. 互动娱乐类** (10%)
- 投票/问答
- 挑战活动
- 趋势热点
- 节日祝福

#### 内容日历规划

**一周内容规划 (示例)**:

```yaml
周一:
  平台: Instagram + 小红书
  主题: 产品展示
  形式: 图文
  时间: 19:00

周二:
  平台: TikTok
  主题: 创意视频
  形式: 15-60秒视频
  时间: 20:00

周三:
  平台: Facebook + 微博
  主题: 教育内容
  形式: 图文
  时间: 14:00

周四:
  平台: Instagram Stories
  主题: 用户故事
  形式: Stories系列
  时间: 12:00, 19:00

周五:
  平台: 小红书 + TikTok
  主题: 种草/测评
  形式: 视频
  时间: 21:00

周六:
  平台: Instagram + 微博
  主题: 互动娱乐
  形式: 投票/问答
  时间: 20:00

周日:
  平台: 全平台
  主题: 品牌文化
  形式: 幕后故事
  时间: 19:00
```

#### 关键输出

- 一周内容日历
- 30天内容主题列表
- 每个平台的内容形式分布
- 发布时间建议

### 阶段 3: 短内容创作

**输入**: 内容主题 + 目标平台
**输出**: 社媒文案 (50-800字)

#### 不同平台文案特点

**Instagram文案** (150-300字符):
- 开头吸引注意 (Hook)
- 中间价值传递 (Value)
- 结尾互动引导 (CTA)
- 使用10-15个Hashtags

**Facebook文案** (100-300字):
- 故事化表达
- 问题-解决方案结构
- 情感共鸣
- 链接到网站/产品

**TikTok脚本** (100-200字，15-60秒):
- 前3秒抓住注意力
- 中间展示核心内容
- 结尾互动/关注引导
- 配合BGM和特效

**小红书笔记** (500-800字):
- 标题吸引点击 (20-30字)
- 正文详细描述
- 配图5-9张
- Hashtags 5-10个

**微博文案** (100-200字):
- 热点话题切入
- 简洁有力
- 话题标签 (#话题#)
- @相关账号

#### 文案创作框架

**AIDA模型** (注意-兴趣-欲望-行动):

```markdown
# 注意 (Attention)
{开头3秒，抓住注意力}

# 兴趣 (Interest)
{提出问题或痛点，引发兴趣}

# 欲望 (Desire)
{展示产品/解决方案，激发欲望}

# 行动 (Action)
{明确的CTA：点击链接/评论/分享/关注}
```

**PAS模型** (问题-煽动-解决):

```markdown
# 问题 (Problem)
{描述用户面临的问题}

# 煽动 (Agitation)
{放大问题的痛苦和影响}

# 解决 (Solution)
{提供解决方案：你的产品}
```

#### 关键输出

- 针对每个平台的定制文案
- Hook (开头吸引注意)
- CTA (行动号召)
- Hashtags/话题标签
- @提及账号

### 阶段 4: 视觉设计

**输入**: 文案内容 + 平台要求
**输出**: 视觉设计方案

#### 图片设计指南

**Instagram图片**:
- **尺寸**: 1080x1080 (正方形), 1080x1350 (竖版)
- **风格**: 统一的滤镜和色调
- **内容**: 高质量产品图、生活方式图
- **文字**: 简洁的文字叠加 (可选)
- **数量**: Feed 1-9张, Stories 3-10张

**小红书图片**:
- **尺寸**: 1080x1440 (3:4竖版)
- **风格**: 真实、生活化、细节丰富
- **内容**: 产品图、使用场景、对比图
- **文字**: 详细说明和标注
- **数量**: 5-9张

**Facebook图片**:
- **尺寸**: 1200x630 (横版), 1080x1080 (正方形)
- **风格**: 信息丰富、清晰明了
- **内容**: 产品图、信息图、活动海报
- **文字**: 重要信息突出

**微博图片**:
- **尺寸**: 任意，推荐1080x1080
- **风格**: 热点、话题、视觉冲击
- **内容**: 表情包、梗图、产品图

#### 视频制作指南

**TikTok视频**:
- **时长**: 15-60秒 (最佳15-30秒)
- **尺寸**: 9:16竖版
- **格式**: MP4, H.264编码
- **内容**: 前3秒抓住注意力
- **BGM**: 使用热门音乐
- **字幕**: 必须有字幕
- **特效**: 适度使用转场和特效

**Instagram Reels**:
- **时长**: 15-60秒
- **尺寸**: 9:16竖版
- **内容**: 创意、教程、娱乐
- **音乐**: 使用Instagram音乐库

**直播内容**:
- **时长**: 30-60分钟
- **内容**: 产品介绍、问答、教程
- **互动**: 及时回复评论
- **准备**: 提前准备脚本和产品

#### 缩略图设计

**原则**:
- 高对比度，吸引点击
- 包含关键信息或产品
- 使用大号文字
- 保持品牌一致性

#### 关键输出

- 图片规格和数量
- 视频时长和格式
- 视觉风格指南
- 缩略图设计方案

---

### 阶段 5: 发布优化

**输入**: 内容准备完成
**输出**: 发布策略 + 互动策略

#### 最佳发布时间

**Instagram**:
- **最佳时间**: 11am-1pm, 7pm-9pm (工作日)
- **周末**: 10am-12pm, 8pm-10pm
- **频率**: 每天1-3条

**Facebook**:
- **最佳时间**: 1pm-4pm (工作日)
- **周末**: 12pm-3pm
- **频率**: 每天1-2条

**TikTok**:
- **最佳时间**: 7pm-9pm, 10pm-12am
- **高峰**: 周五、周六晚上
- **频率**: 每天1-3条

**小红书**:
- **最佳时间**: 9pm-11pm
- **高峰**: 工作日晚上、周末全天
- **频率**: 每天1-2条

**微博**:
- **最佳时间**: 12pm-2pm, 8pm-10pm
- **高峰**: 热点事件发生时
- **频率**: 每天2-5条

#### 互动策略

**发布后互动**:
- **前30分钟**: 快速回复所有评论
- **前2小时**: 持续互动，提升曝光
- **24小时内**: 回复所有问题和私信

**提升互动的技巧**:
1. **提问式文案**: "你觉得...？"
2. **投票/问卷**: "A还是B？"
3. **挑战活动**: "@3个朋友"
4. **赠品活动**: "关注+评论+分享"
5. **UGC激励**: "晒单有奖"

#### 数据监控

**关键指标**:
- **曝光量**: Reach
- **互动率**: Engagement Rate (点赞+评论+分享/曝光)
- **点击率**: CTR (如果有链接)
- **粉丝增长**: Followers Growth
- **转化率**: Conversion (如果有产品链接)

**优化策略**:
- A/B测试不同发布时间
- 测试不同内容类型的互动率
- 分析高互动内容的共同点
- 迭代优化内容策略

#### 关键输出

- 发布时间表
- 互动响应计划
- 数据监控指标
- 优化建议

## 🔄 RLM递归思想说明

### 这个流程如何体现RLM?

```
传统方式:
用户 → AI → 一次性输出社媒内容
         (缺乏平台适配、无发布策略、低互动)

RLM方式:
用户 → 主控流程 (社媒内容生产)
       ↓
       分解任务 → 平台选择策略
       ↓
       继续分解 → 内容规划
       ↓
       继续分解 → 短内容创作
       ↓
       继续分解 → 视觉设计
       ↓
       继续分解 → 发布优化
       ↓
       汇总所有结果 → 完整社媒内容包
```

### 优势

| 对比维度 | 传统方式 | RLM方式 |
|---------|---------|---------|
| **平台适配** | ❌ 一刀切 | ✅ 每个平台定制 |
| **视觉设计** | ❌ 缺乏指导 | ✅ 详细设计方案 |
| **发布策略** | ❌ 随意发布 | ✅ 最佳时间+互动策略 |
| **效果监控** | ❌ 无优化 | ✅ 数据驱动优化 |
