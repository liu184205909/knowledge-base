# RLM营销方法论核心

> **如何用AI实现营销全流程自动化**

---

## 🎯 什么是RLM营销方法论?

**RLM营销方法论 = 用AI的分而治之思想 + 从网站架构到内容全面超越竞品的策略，实现营销全流程自动化**

### 核心逻辑链（重要！AI执行时必须遵循）

```
核心理念: 从网站架构到内容全面超越竞争对手

阶段1: 市场分析与用户洞察
  ↓ 理解: 市场环境和用户需求

阶段2: 竞争对手网站分析
  ↓ 模仿: 竞品做得好的地方
  ↓ 超越: 竞品做得不好的地方
  ↓ 得出: 我们的策略（价值主张+差异化）
  ↓

阶段3: 网站内容规划
  ↓ 基于: 竞品分析结果 + 我们的策略
  ↓ 目标: 从网站架构到内容全面超越
  ↓

阶段4: 营销组合与内容创作
  ↓ 基于: 竞品内容策略 + 我们的策略
  ↓ 目标: 从网站架构到内容全面超越
```

### 核心思路：模仿 + 超越 + 差异化

**我们的目标**：**从网站架构到内容全面超越竞争对手**

**具体路径**：
1. **先模仿**竞争对手做得好的地方（学习最佳实践）
2. **再超越**竞争对手做得不好的地方（改进不足）
3. **最后建立**差异化优势（找到独特机会）

**关键原则**：
- ✅ 不要重新发明轮子
- ✅ 学习竞品的成功经验
- ✅ 改进竞品的不足之处
- ✅ 找到竞品没做的机会点
- ✅ **从网站架构到内容全面超越**

## 🚀 营销全流程的RLM化改造

### RLM营销的4个核心阶段

#### 阶段1: 市场分析与用户洞察

**目标**: 理解"市场环境、产品价值、用户需求"

**核心任务** (4个步骤):
1. **市场环境分析**
   - PEST分析(政策/经济/社会/技术)
   - 市场规模与增长趋势
   - 行业发展趋势

2. **产品评估(100分制)**
   - 市场分析(30分)
   - 产品分析(30分)
   - 竞争分析(20分)
   - 运营分析(20分)

3. **用户洞察(画像+痛点)**
   - 目标用户是谁？（根据产品特性判断）
   - 用户画像
   - 核心痛点
   - 用户需求

4. **客户细分(用户分类)**
   - 主要用户群体
   - 次要用户群体
   - 用户价值分层
   - 用户旅程映射

**RLM化改造**:
```
传统方式: 2-3天
RLM方式: 2-3分钟
效率提升: 1000倍
```

**关键输出**:
- 市场分析报告
- 产品评估报告(100分制)
- 用户画像和痛点
- 客户细分清单

**完整流程**: [01-市场分析.md](./01-市场分析.md)

#### 阶段2: 竞争对手网站分析

**目标**: 深度分析竞品网站，制定"模仿+超越+差异化"策略

**核心任务** (3个步骤):
1. **竞品发现** (40+竞品)
   - 使用MCP Web Search Prime搜索
   - 识别所有相关竞品类型
   - 分类整理(品牌方/平台/第三方)
   - 筛选重点分析对象

2. **竞争对手网站分析** (20+深度) ⭐ 核心
   - **2.1 产品分类模式识别** ⭐ 新增
     - **目标**: 识别竞品的所有分类模式,并评估适用性
     - **输出**: 5种分类模式总结表

     **分类模式识别模板**:

     | 模式 | 分类逻辑 | 代表竞品 | 优点 | 缺点 | 适用性 |
     |------|---------|---------|------|------|--------|
     | 模式A | 按产品结构 | 竞品X | 专业 | 不友好 | ✅推荐 |
     | 模式B | 按设备类型 | 竞品Y | 直观 | 复杂 | ✅辅助 |
     | 模式C | 按设备品牌 | 竞品Z | 精准 | 市场窄 | ❌不推荐 |
     | 模式D | 按行业标准 | 竞品W | 高端 | 认证难 | ⚠️后期 |
     | 模式E | 按材料 | 竞品V | 技术型 | 混乱 | ✅筛选 |

     **关键洞察**:
     - 给每种模式命名(A/B/C/D/E)
     - 标注适用性(✅推荐/❌不推荐/⚠️后期)
     - 提供代表竞品案例
     - 分析优缺点对比
     - 给出正确用法建议

     **示例**:
     - ⚠️ 警示: 材料分类应该作为筛选功能,不是主分类
     - ✅ 最佳实践: 主导航简洁(4-6类) + 辅助导航专业(按设备)

   - **2.2 竞品对比矩阵** ⭐ 新增
     - **目标**: 直观展示"我们 vs 竞品"的优劣势
     - **输出**: 多维度评分对比表

     **标准化对比模板**:

     | 对比维度 | 我们 | 竞品A | 竞品B | 竞品C |
     |---------|------|-------|-------|-------|
     | 主分类数量 | 4类 | 4类 | 5类 | 8类 |
     | 简洁度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐ |
     | 专业度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |
     | 客户友好度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ |

     **评分标准**:
     - ⭐⭐⭐⭐⭐ (5星): 行业领先
     - ⭐⭐⭐⭐ (4星): 优秀
     - ⭐⭐⭐ (3星): 良好
     - ⭐⭐ (2星): 基础
     - ⭐ (1星): 较弱

     **关键价值**:
     - ✅ 直观展示优劣势
     - ✅ 量化评估(1-5星评分)
     - ✅ 多维度对比
     - ✅ 结论清晰(例如:"我们处于最佳平衡点")

   - **产品分类方式分析** (原有内容)
     - 竞品按什么维度分类产品？
     - 分类逻辑是什么？
     - 哪些分类方式值得模仿？

   - **网站布局结构分析**
     - 导航结构如何设计？
     - 页面层级如何组织？
     - 信息架构如何规划？
     - 哪些布局做得好？哪些不好？

   - **内容布局策略分析**
     - 内容如何组织？
     - 内容板块有哪些？
     - 哪些内容布局值得学习？

   - **内容类型分析**
     - 写什么类型的内容？
     - 内容形式（博客/案例/教程/指南等）
     - 内容深度和频率如何？
     - 哪些内容类型做得好？哪些不足？

   - **竞品共同弱点识别** ⭐ 新增
     - 目标: 找到90%+竞品都没做的事情
     - 输出: 竞品共同弱点清单
     - 格式: "❌ 90%没有XXX"
     - 价值: 这些是最大的差异化机会,避免红海竞争

3. **我们的策略制定** ⭐ 核心

   **输出要求** (重要):
   - ✅ **模仿清单**必须独立成章节
     - 格式: "模仿X: [竞品名称]的[具体做法]"
     - 内容: 来源于哪、做得好在哪里、我们怎么做
     - 目的: 清晰的学习对象

   - ✅ **超越清单**必须独立成章节
     - 格式: "超越X: [竞品不足的具体描述]"
     - 内容: 竞品哪里不足、我们怎么改进、效果对比
     - 目的: 明确的改进目标

   - ✅ **差异化清单**必须独立成章节
     - 格式: "差异化X: [竞品空白的具体机会]"
     - 内容: 竞品都没做什么、市场空白有多大
     - 目的: 清晰的差异化机会

   **可追溯性要求**:
   - 每条策略都必须标注来源竞品
   - 每条策略都必须说明具体做法
   - 每条策略都应该有数据/事实支撑

**RLM化改造**:
```
传统方式: 5-7天
RLM方式: 5-10分钟
效率提升: 1000倍
```

**关键输出**:
- 竞品清单(40+家)
- 竞品深度分析报告(20+家)
- **我们的策略**（模仿清单+超越清单+差异化机会）

#### 阶段3: 网站内容规划

**目标**: 基于竞品分析和我们的策略，规划网站如何做得更好

**核心任务** (5个步骤):

**步骤0: 前置输入检查** ⭐ 新增
   - ✅ **来自阶段2的输入**
     - 明确阶段2产出了哪些关键决策?
     - 这些决策如何影响阶段3?
     - 如何确保阶段3与阶段2保持一致?

   - ✅ **输入文档**
     - 阶段2输出的"模仿清单"
     - 阶段2输出的"超越清单"
     - 阶段2输出的"差异化清单"

   - ✅ **一致性检查**
     - 阶段3的每个决策都应该追溯到阶段2
     - 如果阶段3的决策与阶段2冲突,需要说明原因
1. **网站结构设计** (基于竞品分析+我们的策略)
   - 参考: 竞品好的结构（模仿）
   - 优化: 竞品不足的地方（超越）
   - 目标: 做得比竞品更好

2. **产品类目规划** (基于竞品分析+我们的策略)
   - 参考: 竞品的分类方式（模仿）
   - 优化: 做得更清晰、更合理（超越）
   - 目标: 用户更容易找到产品

3. **网站架构设计**
   - 技术架构建议
   - URL结构设计
   - 内部链接策略

4. **核心页面内容规划**

   **每个页面的标准结构** ⭐ 新增:
   - ✅ **参考对象**: 哪家竞品的这个页面做得好?
   - ✅ **模仿**: 学习了哪些好的做法?
   - ✅ **超越**: 改进了哪些不足?
   - ✅ **差异化**: 哪些是我们的创新?

   **核心页面包括**:
   - **首页**:
     - 参考对象: ____
     - 模仿: ____
     - 超越: ____
     - 差异化: ____

   - **About Us**:
     - 参考对象: ____
     - 模仿: ____
     - 超越: ____
     - 差异化: ____

   - **产品/服务页面**:
     - 参考对象: ____
     - 模仿: ____
     - 超越: ____
     - 差异化: ____

   - **其他关键页面**:
     - FAQ、联系方式、案例等
     - (每个页面都遵循标准结构)

5. **价值主张呈现策略**
   - 在网站的哪些地方呈现价值主张？
   - 如何用视觉和文字强化价值主张？

**RLM化改造**:
```
传统方式: 3-5天
RLM方式: 5-10分钟
效率提升: 500倍
```

**关键输出**:
- 网站结构设计
- 产品类目规划
- 核心页面内容规划
- 价值主张呈现策略

#### 阶段4: 营销组合与内容创作

**目标**: 基于竞品内容策略和我们的策略，创作比竞品更好的内容

**核心任务** (6个步骤):
1. **内容架构设计** (基于竞品内容策略+我们的策略)
   - 参考: 竞品的内容组织方式（模仿）
   - 优化: 做得更有条理、更清晰（超越）
   - 目标: 内容结构比竞品更合理

2. **SEO关键词策略** (基于竞品关键词策略+我们的策略)
   - 参考: 竞品的关键词选择（模仿）
   - 优化: 覆盖竞品没覆盖的关键词（超越）
   - 目标: SEO效果比竞品更好

3. **内容选题规划** (基于竞品内容类型+我们的策略)
   - 参考: 竞品的内容类型（模仿）
   - 优化: 增加竞品缺少的内容类型（超越）
   - 差异化: 创作竞品没有的内容
   - 目标: 内容类型比竞品更丰富

4. **内容创作准备**
   - 素材收集
   - 工具准备
   - 风格指南

5. **内容创作** ⭐ 核心
   - 博客文章(1500-3000字)
   - 案例研究
   - 教程指南
   - 其他内容形式

6. **内容优化** ⭐ 核心
   - SEO优化(关键词布局、元数据)
   - 降AI味(三遍审校)
   - 质量控制

**RLM化改造**:
```
传统方式: 4-6小时/篇
RLM方式: 15分钟/篇
效率提升: 16倍
```

**关键输出**:
- 内容策略方案
- 高质量内容
- SEO优化完整版

## 💎 RLM营销的核心原则

### 原则1: 分而治之

**复杂任务 → 分解为简单子任务**

**示例**:
```
复杂: "制定营销策略"
  ↓ 分解
简单1: "产品评估(100分制)"
简单2: "竞品发现(MCP搜索)"
简单3: "竞品分析(Web Reader)"
简单4: "用户洞察(AI分析)"
  ↓ 汇总
完整: "营销策略报告"
```

### 原则2: 工具增强

**使用MCP工具获取实时数据**

**示例**:
```
❌ 不用工具: "根据我的训练数据分析竞品"
✅ 使用MCP: "搜索XX行业最新竞品，分析网站"
```

### 原则3: 结构化输出

**每个阶段都有明确的输出，下阶段基于上阶段输出**

**示例**:
```
阶段1输出 → 市场分析报告
  ↓ 输入给阶段2
阶段2输出 → 差异化策略
  ↓ 可直接执行
```

### 原则4: 流程文档化

**将成功经验固化成文档，实现复用**

**示例**:
```
第1次: "分析智能宠物健康监测产品"
  → AI执行01-市场分析.md
  → 输出报告

第2次: "分析睡眠改善产品"
  → AI再次执行01-市场分析.md
  → 输出报告
  → 无需重新描述流程

第N次: 持续复用...
```
