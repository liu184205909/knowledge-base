## 📊 执行流程

### 阶段1: 关键词研究

**目标**: 挖掘高商业价值关键词

**执行步骤**:
1. 使用MCP Web Search搜索行业关键词
2. 分析关键词搜索量和竞争程度
3. 识别长尾关键词机会
4. 评估关键词商业意图

**输出**:
```
关键词清单表格:
| 关键词 | 搜索量 | 竞争度 | 商业意图 | 优先级 |
|-------|-------|--------|---------|--------|
| pet health monitor | 1200 | 中 | 高 | 高 |
| smart pet tracker | 800 | 低 | 中 | 高 |
...
```

---

### 阶段2: 竞品分析

**目标**: 分析竞品SEO策略和排名情况

**执行步骤**:
1. 识别主要竞品（3-5个）
2. 使用MCP Web Reader分析竞品网站
3. 分析竞品关键词布局
4. 分析竞品内容和外链策略

**输出**:
```
竞品SEO分析报告:
- 竞品列表及其排名关键词
- 竞品内容策略分析
- 竞品外链来源分析
- 可超越的SEO机会点
```

---

### 阶段3: 内容优化

**目标**: 生成SEO优化的内容建议

**执行步骤**:
1. 基于关键词研究，规划内容主题
2. 生成SEO优化的标题和描述
3. 生成内容大纲（包含关键词布局）
4. 提供内部链接建议

**输出**:
```
内容优化方案:
- 目标关键词页面列表
- 每个页面的SEO元数据（标题、描述、H1）
- 内容关键词布局建议
- 内部链接结构建议
```

---

### 阶段4: 技术SEO

**目标**: 网站技术审计和优化建议

**执行步骤**:
1. 检查网站技术问题（加载速度、移动友好性等）
2. 分析网站结构和URL
3. 检查网站可索引性
4. 提供技术优化建议

**输出**:
```
技术SEO优化清单:
- 页面加载速度优化建议
- 移动友好性改进建议
- 网站结构优化建议
- 元标签和结构化数据建议
```

---

### 阶段5: 外链建设

**目标**: 高质量外链获取策略

**执行步骤**:
1. 分析竞品外链来源
2. 识别外链机会
3. 制定外链建设策略（主动+被动）
4. 提供具体执行方案

**输出**:
```
外链建设方案:
- 竞品外链分析
- 外链机会清单
- 主动外链建设策略（客座博客、资源页面等）
- 被动链接资产建设策略（统计页面、工具页面等）
- 执行时间线和优先级
```
