# Parasite SEO 平台资源库

> 借力高权重第三方平台的内容页面参与 Google 排名，直接获取搜索流量（而非仅做外链跳板）
>
> 与传统外链建设的区别：外链建设目标是"链"，Parasite SEO 目标是"排名 + 流量"
>
> 最后更新：2026-06-01

---

## 平台筛选原则

1. **信息差优先**：Pinterest / Reddit / Quora / Medium / YouTube 人人皆知，不收录
2. **排名能力验证**：平台内容曾在 Google SERP 中出现过，或 DA ≥ 70 且内容可被 Google 索引
3. **可自主发布**：无需申请 contributor 身份（那种属于 Guest Post 范畴）
4. **标注实测状态**：本清单中的"排名能力"为预估，实际以 SERP 验证为准

## Tier 1：冷门但有效（推荐优先测试）

### 1. Instructables

| 项目 | 详情 |
|------|------|
| DA | 93 |
| URL | <https://www.instructables.com> |
| 内容类型 | DIY 教程（图文步骤式） |
| 链接属性 | 教程正文内可插入链接 |
| 注册门槛 | 免费，Autodesk 账号即可 |
| 排名能力 | **强**。教程类长尾词在 Google 排名优秀（如 "how to make crystal bracelet"） |
| 适合内容 | 手工制作教程、材料指南、步骤式 how-to |
| 品类匹配 | **极高**。"how to make healing crystal bracelet""DIY crystal jewelry" 等词直接匹配 |

**操作要点：**
- 每篇教程需要 ≥5 个步骤 + 配图，质量门槛低
- 在材料清单/推荐购买段落中植入 Amazon 联盟链接
- 教程标题精确匹配搜索词
- 参与 contests 可获得额外曝光

- 文章需 ≥1000 字，通过 QAP 后才会获得搜索流量分配
- 被选为 "Editor's Choice" 的文章排名显著提升

## 平台选择决策速查

```
你的目标关键词在 SERP 上，哪个 Tier 1-2 平台的内容出现了？
├─ 出现了 → 直接投入该平台，发同类内容
└─ 都没出现 → 逐个用 site:平台域名 + 关键词 验证

按内容格式选择：
├─ 教程/DIY → Instructables
├─ 产品清单/Gift Guide → Buzzfeed Community
├─ 深度评测 → HubPages (FeltMagnet)
├─ PDF/目录/杂志 → Issuu / FlipHTML5
├─ 演示文稿/信息图 → SlideShare
├─ 产品摄影 → Behance / Flickr / DeviantArt
├─ 快速测试 → Telegra.ph / Google Sites
└─ 生活方式/养生 → Vocal Media / NewsBreak
```
