## 使用说明

1. **新增平台**：先在[外链建设指南](../00-营销方法论基础/06-外链建设指南.md)了解策略，测试通过后录入"已验证平台清单"
2. **发布外链**：每次发布后在"已发布外链记录"中新增一行
3. **月度更新**：每月检查收录状态和引荐流量，更新"效果汇总"
4. **项目关联**：每条外链关联到具体选品项目，项目文件夹中也做记录
