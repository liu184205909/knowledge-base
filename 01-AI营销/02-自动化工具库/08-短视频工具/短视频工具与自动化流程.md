## 视频下载与转录

### yt-dlp

**支持平台**：YouTube / TikTok / Instagram 等主流平台

```bash
# 基本下载
yt-dlp "https://www.tiktok.com/@user/video/xxx"

# 下载 + 提取字幕
yt-dlp --write-sub --sub-lang en "https://www.youtube.com/watch?v=xxx"

# 指定格式（mp4）
yt-dlp -f "bestvideo[ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]" "URL"
```

### Vibe（离线语音转录）

- 基于 Whisper，100+ 语言，GPU 加速 3-5x
- 视频转文字后仅 ~12KB 文本，省 token
- 安装参考：`00-基础能力/03-Github开源项目.md`

### Fluent-M3U8

- 流媒体视频下载（跨平台）
- 安装参考：`00-基础能力/03-Github开源项目.md`

## 视频剪辑与创作

### video-use（自然语言剪辑）

```bash
git clone https://github.com/browser-use/video-use
cd video-use
ln -s "$(pwd)" ~/.claude/skills/video-use
pip install -e .
brew install ffmpeg  # macOS
# Windows: choco install ffmpeg
```

进入素材文件夹后对 Claude Code 说：`将这些素材剪辑成一条发布视频`

### 美图设计室（Seedance 2.0）

> 来源：袋鼠帝《用AI做了一批TikTok带货视频》

- **用途**：产品图一键生成带货视频
- **模板**：8 种（UGC种草/口播/开箱/痛点解决/TVC等）
- **特色**：支持多语言多市场，可定制专属模特，所有 SKU 共用同一视觉资产
- **成本**：~35 元/条（375 美豆）

### Codex + cutcli 批量混剪

> 来源：唐舰长《用 Codex 批量混剪视频》

- **链路**：中文素材 → 多语言版本全自动（下载→转录→切片→文案→配音→裁切→剪映草稿）
- **技巧**：截帧代替视频识别降 token
- **适合**：跨境矩阵号批量混剪

## Codex 数字人口播视频（HeyGen 插件链路）

> **来源**：唐舰长《用 Codex 做数字人口播，太丝滑了》
>
> **适用场景**：知识类/教程类口播视频（右下角挂人物，屏幕中间演示过程），全程在 Codex 内完成，不需要剪映。
>
> **前置条件**：Codex 已过插件验证（未过验证无法使用插件）。

### 工具链

| 环节 | 工具 | 说明 |
|------|------|------|
| 配音 | **Minimax 声音克隆 API** | 克隆 9.9 元/次 + 合成 2 元/万字，国内模型，白菜价 |
| 数字人 | **HeyGen 插件** | 给一张照片即可生成数字人，免费可用 |
| 视频框架/剪辑 | **HyperFrames by HeyGen 插件** | HTML+CSS+GSAP 动画创作，CLI 剪辑/预览/渲染 |
| 文件上传 | **Cloudinary 插件** | 本地文件秒转在线链接（HeyGen 需要在线 URL） |

### 制作流程

```
准备素材 → HyperFrames 生成视频框架 → HeyGen 加数字人 → 口述修改意见 → 最终成片
```

**Step 1：配音接入（Minimax）**

1. 去 [Minimax 平台](https://platform.minimaxi.com) 注册，左上角账户管理充值
2. 录制 20-30 秒自己的声音，读一段内容
3. 在 Codex 中告诉 AI：`用 Minimax 声音克隆 API，API key 是 xxx，音频文件路径是 xxx，音色 id 设为 xxx`
4. Codex 自动测试调用，确认接入成功

**Step 2：HyperFrames 出视频框架**

给 Codex 一篇完整文章 + 效果参考视频 + 配图素材，提出要求：
> 帮我生成口播文案视频，用 Minimax 帮我配音

引用 HyperFrames 插件，Codex 自动生成视频框架。首次效果可能一般。

**Step 3：HeyGen 加数字人**

对不满意的地方口述修改意见（哪里不行、字幕怎么改），然后提供自己的照片：
> 用 HeyGen 插件，这张照片做数字人

HeyGen 会自动处理头像分辨率、宽高比、语音选择。

**Step 4：迭代优化**

看完视频后，把发现的不足再告诉 Codex：
- 数字人形状调整（如长方形→圆形）
- 添加动态元素
- 字幕清理符号
- 其他排版细节

### 素材准备要点

| 素材类型 | 要求 |
|---------|------|
| **文字内容** | 丢完整文章/文档，不要只给模糊主题。内容太长先划重点段落 |
| **图片素材** | PNG/JPG，1080p 以上；3-5 分钟视频准备 8-15 张；**用语义化命名**（如 `01-配置界面.png`，不要 `IMG_0291.png`） |
| **视频素材** | MP4，单段 30 秒以内；同样语义化命名 |
| **给素材时配一句话** | 告诉 Codex 这些素材是干嘛的，上下文越明确出片越准 |

### 成本

| 项目 | 费用 |
|------|------|
| Minimax 声音克隆 | 9.9 元/次 |
| Minimax 语音合成 | 2 元/万字 |
