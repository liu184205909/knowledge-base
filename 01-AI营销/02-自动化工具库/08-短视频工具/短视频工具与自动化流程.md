# 短视频工具与自动化流程

> 基于 AI + 开源工具的短视频素材采集、拆解、创作、分发全链路
> 最后更新: 2026-05-04

---

## 全链路流程（参考顾小北三阶段模型）

```
阶段1：素材采集     阶段2：素材拆解        阶段3：创作与分发
刷到好内容           AI 自动拆解元素         AI 生成脚本/视频
  ↓                    ↓                     ↓
链接丢给AI → 入库   → 痛点/场景/钩子/镜头  → 分发到TikTok/YouTube/抖音
自动下载+转文字      写入多维表/数据库        数据分析→调整方向
```

### 核心工具

| 工具 | 用途 | 来源 |
|------|------|------|
| **yt-dlp** | YouTube/TikTok 视频下载 + 字幕提取 | `00-基础能力/05-Reddit的JSON功法.md` |
| **Vibe** | 本地语音转录，视频自动转文字（离线，Whisper） | `00-基础能力/03-Github开源项目.md` |
| **Fluent-M3U8** | 流媒体视频下载（跨平台） | `00-基础能力/03-Github开源项目.md` |
| **web-access Skill** | 浏览器自动化，复用登录态抓取社媒内容 | `00-基础能力/02-Skill设计与管理.md` |

### 采集流程

```
1. 手刷 TikTok/YouTube，看到好内容
2. 把链接丢给 AI，说"入库"
3. AI 自动：下载视频 → 转文字 → 关键帧截图 → 上传云存储
```

**关键技术点**：
- yt-dlp 支持 TikTok/YouTube/Instagram 等主流平台
- Vibe 离线转录，100+ 语言，GPU 加速 3-5x
- 视频转文字后仅 ~12KB 文本，省 token（video-use 的做法）

## 阶段2：素材拆解

### 拆解维度（Karpathy 知识库思路）

一条素材进来，拆成以下可复用元素：

| 维度 | 说明 | 示例 |
|------|------|------|
| 痛点 | 用户什么问题 | "焦虑到睡不着" |
| 场景 | 使用场景 | "工作压力大时" |
| 功效 | 产品效果 | "戴上3天明显感觉平静了" |
| 钩子 | 开头3秒抓人点 | "我以为这是智商税..." |
| 镜头结构 | 拍摄手法 | 口播→特写→佩戴→对比 |
| 激励点 | 促转化的点 | "限时买一送一" |
| 适合人群 | 目标用户 | 25-35岁职场女性 |
| 可复用元素 | 能拆出来复用的 | 开头句式、转场方式 |

### 数据存储方案

| 方案 | 适合场景 | 成本 |
|------|---------|------|
| **飞书多维表** | 团队协作，可视化好 | 免费 |
| **Notion 数据库** | 个人使用，灵活 | 免费 |
| **CSV + 本地文件夹** | 最简单，无依赖 | 0 |

## 阶段3：创作与分发

### AI 视频创作工具

| 工具 | 用途 | 说明 |
|------|------|------|
| **video-use** | 自然语言剪辑视频（Claude Code Skill） | 自动去口癖、加字幕、调色、渲染自评 |
| **Voicebox** | 本地语音克隆 + TTS | 免费版 ElevenLabs，全离线 |
| **AiToEarn** | 多平台一键分发 | 抖音/TikTok/YouTube/小红书/B站 |
| **美图设计室（Seedance 2.0）** | 产品图一键生成带货视频 | 8 种模板（UGC种草/口播/开箱/痛点解决/TVC等），支持多语言多市场，可定制专属模特，所有 SKU 共用同一视觉资产；~35 元/条 |
| **Codex + cutcli** | Agent 自动化批量混剪 | 中文素材 → 多语言版本全自动链路（下载→转录→切片→文案→配音→裁切→剪映草稿）；截帧代替视频识别降 token；适合跨境矩阵号 |

### Codex 数字人口播视频制作（HeyGen 插件链路）

> **来源**：唐舰长《用 Codex 做数字人口播，太丝滑了》
>
> **适用场景**：知识类/教程类口播视频（右下角挂人物，屏幕中间演示过程），全程在 Codex 内完成，不需要剪映。
>
> **前置条件**：Codex 已过插件验证（未过验证无法使用插件）。

#### 工具链

| 环节 | 工具 | 说明 |
|------|------|------|
| 配音 | **Minimax 声音克隆 API** | 克隆 9.9 元/次 + 合成 2 元/万字，国内模型，白菜价 |
| 数字人 | **HeyGen 插件** | 给一张照片即可生成数字人，免费可用 |
| 视频框架/剪辑 | **HyperFrames by HeyGen 插件** | HTML+CSS+GSAP 动画创作，CLI 剪辑/预览/渲染 |
| 文件上传 | **Cloudinary 插件** | 本地文件秒转在线链接（HeyGen 需要在线 URL） |

#### 制作流程

```
准备素材 → HyperFrames 生成视频框架 → HeyGen 加数字人 → 口述修改意见 → 最终成片
```

**Step 1：配音接入（Minimax）**

1. 去 [Minimax 平台](https://platform.minimaxi.com) 注册，左上角账户管理充值
2. 录制 20-30 秒自己的声音，读一段内容
3. 在 Codex 中告诉 AI：`用 Minimax 声音克隆 API，API key 是 xxx，音频文件路径是 xxx，音色 id 设为 xxx`
4. Codex 自动测试调用，确认接入成功

**Step 2：HyperFrames 出视频框架**

给 Codex 一篇完整文章 + 效果参考视频 + 配图素材，提出要求：
> 帮我生成口播文案视频，用 Minimax 帮我配音

引用 HyperFrames 插件，Codex 自动生成视频框架。首次效果可能一般。

**Step 3：HeyGen 加数字人**

对不满意的地方口述修改意见（哪里不行、字幕怎么改），然后提供自己的照片：
> 用 HeyGen 插件，这张照片做数字人

HeyGen 会自动处理头像分辨率、宽高比、语音选择。

**Step 4：迭代优化**

看完视频后，把发现的不足再告诉 Codex：
- 数字人形状调整（如长方形→圆形）
- 添加动态元素
- 字幕清理符号
- 其他排版细节

#### 素材准备要点

| 素材类型 | 要求 |
|---------|------|
| **文字内容** | 丢完整文章/文档，不要只给模糊主题。内容太长先划重点段落 |
| **图片素材** | PNG/JPG，1080p 以上；3-5 分钟视频准备 8-15 张；**用语义化命名**（如 `01-配置界面.png`，不要 `IMG_0291.png`） |
| **视频素材** | MP4，单段 30 秒以内；同样语义化命名 |
| **给素材时配一句话** | 告诉 Codex 这些素材是干嘛的，上下文越明确出片越准 |

#### 成本

| 项目 | 费用 |
|------|------|
| Minimax 声音克隆 | 9.9 元/次 |
| Minimax 语音合成 | 2 元/万字 |
| HeyGen 数字人 | 免费版可用 |
| HyperFrames 插件 | 免费（Codex 插件） |
| Cloudinary 插件 | 免费账号即可 |

**一条 3-5 分钟知识类口播视频的硬成本约 10-15 元**（主要是配音）。

## 各平台短视频策略要点

> 从多个选品项目的竞品分析中提炼

### TikTok

- 标签策略：#sleephacks(5000万+)、#crystals、#anxietyrelief 等
- 内容形式：15-60秒科普/开箱/佩戴展示
- 发布频率：每周 2-3 条
- 挑战赛：UGC 裂变（如 #MapToPoster、#CrystalChallenge）

### YouTube Shorts

- AI 引用数据：YouTube 在 Google AIO 中引用率 23.3%，Perplexity 中 13.9%
- YouTube 被引用频率是其他视频平台的 **200倍**
- 适合做：教程类、科普类长视频 + Shorts 切片

### 抖音（国内）

- 开箱评测、直播带货
- 站外投放：评测视频 → 淘宝/京东转化
