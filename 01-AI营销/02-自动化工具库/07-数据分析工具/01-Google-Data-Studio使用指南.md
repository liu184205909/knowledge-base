# Google Data Studio 使用指南

> Google Data Studio（原名 Looker Studio，2025年改回 Data Studio）是 Google 的免费数据可视化工具，可连接多种数据源搭建交互式仪表盘。本文档为基础使用指南，等网站上线有数据后再补充进阶分析和仪表盘模板。

---

## 一、产品概览

### 版本对比

| 维度 | Data Studio（免费版） | Data Studio Pro（$9/月） |
|------|----------------------|--------------------------|
| 价格 | 免费 | $9/用户/月 |
| 数据源连接 | 支持所有 Google 系 + 第三方连接器 | 同免费版 + 更多 BI 系统集成 |
| 协作 | 支持分享和协作 | 增加数据权限管理（按角色控制） |
| AI 功能 | 基础 | Gemini 深度集成，自然语言查询 |
| 数据预警 | 无 | 支持异常预警通知 |
| 适用场景 | 个人/小团队完全够用 | 团队/企业级数据管理 |

### 核心定位

Data Studio 正在成为 Google 数据分析体系的**中央入口**：
- 连接数据源（GA4、GSC、Sheets、BigQuery 等）
- 可视化分析（拖拽式仪表盘）
- 与 BigQuery 和 AI 工作流打通
- 整合 Gemini 进行自然语言数据分析

## 二、支持的数据源

### Google 系（原生支持，推荐）

| 数据源 | 用途 | 推荐优先级 |
|--------|------|-----------|
| **Google Analytics 4** | 网站流量、用户行为、转化数据 | **必接** |
| **Google Search Console** | 搜索排名、点击率、关键词表现 | **必接** |
| **Google Sheets** | 自定义数据、竞品分析表、手动数据 | **必接** |
| Google Ads | 广告投放数据 | 按需 |
| YouTube Analytics | 视频表现数据 | 按需 |
| BigQuery | 大规模数据分析、SQL 查询 | 进阶 |

### 第三方（通过连接器）

| 连接器 | 支持的数据源 |
|--------|-------------|
| Supermetrics | Facebook Ads、LinkedIn Ads、Semrush、Ahrefs 等 |
| PorterMetrics | Shopify、WooCommerce、Stripe 等 |
| 社区连接器 | 几乎所有主流平台 |

> **注意**：第三方连接器大多付费。初期用 Google 系免费数据源即可。

## 三、基础操作

### 3.1 创建报告

1. 打开 [datastudio.google.com](https://datastudio.google.com)
2. 点击「空白报告」或从模板库选择
3. 添加数据源 → 选择 GA4 / GSC / Sheets
4. 拖拽图表组件到画布

### 3.2 核心图表类型

| 图表 | 用途 | 典型场景 |
|------|------|---------|
| 时间序列图 | 趋势变化 | 日/周/月流量趋势 |
| 记分卡 | 核心指标 | 总访问量、转化率、收入 |
| 表格 | 详细数据 | 关键词排名列表、页面表现 |
| 饼图/环形图 | 占比分布 | 流量来源分布 |
| 地理图 | 地域分布 | 访客国家/城市分布 |
| 柱状图 | 对比分析 | 各页面流量对比 |
| 控制面板 | 交互筛选 | 日期范围、国家、设备筛选 |

### 3.3 数据筛选和交互

- **日期范围控制**：必须添加，所有报告都需要
- **下拉筛选器**：按国家、设备、渠道等维度筛选
- **图表交互**：点击某个数据点可联动其他图表

## 四、SEO 数据仪表盘模板（待搭建）

> 网站上线后，基于以下结构搭建仪表盘。

### 4.1 GSC 仪表盘

| 模块 | 指标 | 图表类型 |
|------|------|---------|
| 搜索概览 | 总点击、总展示、平均 CTR、平均排名 | 记分卡 + 时间序列 |
| 关键词表现 | Top 查询词、点击量、排名位置 | 表格 |
| 页面表现 | Top 页面、点击量、CTR | 表格 |
| 国家分布 | 各国点击量 | 地理图 |
| 设备分布 | Desktop/Mobile/Tablet 占比 | 饼图 |

### 4.2 GA4 流量仪表盘

| 模块 | 指标 | 图表类型 |
|------|------|---------|
| 流量概览 | 用户数、会话数、页面浏览量 | 记分卡 + 时间序列 |
| 流量来源 | Organic / Direct / Social / Referral | 饼图/柱状图 |
| 页面热度 | Top 页面浏览量 | 柱状图 |
| 用户地域 | 国家/城市分布 | 地理图 |
| 转化漏斗 | 关键转化事件 | 漏斗图 |

### 4.3 综合仪表盘（GSC + GA4）

将搜索数据和流量数据放在一个仪表盘中，便于观察：
- 搜索排名 → 点击流量 → 用户行为的完整链路
- 哪些关键词带来流量但跳出率高
- 哪些页面搜索表现好但转化低

## 五、与现有工具的配合

### 数据流闭环

```
竞品挖掘工具 → 竞品分析工具 → 内容分析工具 → 网站内容上线
                                                    ↓
                                              GSC/GA4 数据产生
                                                    ↓
                                          Data Studio 可视化分析
                                                    ↓
                                          数据驱动内容迭代决策
```

### 具体配合场景

| 工具 | 产出的数据 | 导入 Data Studio 的方式 |
|------|-----------|----------------------|
| 竞品挖掘工具 | SERP 竞品数据 | 导出 CSV → Google Sheets → Data Studio |
| 竞品分析工具 | Sitemap 页面结构 | 导出 CSV → Sheets → Data Studio |
| 内容分析工具 | 内容评分数据 | 导出 CSV → Sheets → Data Studio |
| GSC | 搜索排名数据 | 原生连接 |
| GA4 | 流量行为数据 | 原生连接 |

## 六、进阶功能（待数据积累后补充）

> 以下功能在网站上线运行 1-2 个月、有足够数据后启用。

### 6.1 BigQuery 集成

- GA4 原始数据导出到 BigQuery → Data Studio 直连 BigQuery
- 优势：不受 GA4 聚合限制，可做更细粒度的分析
- 成本：BigQuery 有免费额度（1TB/月查询），初期足够

### 6.2 Gemini AI 辅助分析

- Pro 版可用自然语言提问：「上个月哪个页面的转化率最高？」
- 自动生成图表建议
- 数据异常自动检测和预警

### 6.3 自动化报告

- 设置定时邮件发送（周报/月报）
- 数据异常预警通知（Pro 版）
- 嵌入 Google Slides 做汇报

### 6.4 自定义计算字段

- 创建派生指标（如：转化率 = 转化次数 / 会话数）
- 使用 CASE 语句做条件分类
- 正则表达式匹配和提取

## 七、快速上手步骤

1. **注册**：用 Google 账号登录 [datastudio.google.com](https://datastudio.google.com)
2. **连接数据源**：先连接 Google Sheets（最简单，可以用现有竞品分析数据练手）
3. **创建第一个报告**：选空白报告 → 添加图表 → 拖拽数据维度和指标
4. **添加交互**：添加日期范围控制器和筛选器
5. **分享**：分享链接给团队成员，支持多人实时协作
