# 短视频工具与自动化流程

> 基于 AI + 开源工具的短视频素材采集、拆解、创作、分发全链路
> 最后更新: 2026-05-04

---

## 全链路流程（参考顾小北三阶段模型）

```
阶段1：素材采集     阶段2：素材拆解        阶段3：创作与分发
刷到好内容           AI 自动拆解元素         AI 生成脚本/视频
  ↓                    ↓                     ↓
链接丢给AI → 入库   → 痛点/场景/钩子/镜头  → 分发到TikTok/YouTube/抖音
自动下载+转文字      写入多维表/数据库        数据分析→调整方向
```

## 阶段1：素材采集

### 核心工具

| 工具 | 用途 | 来源 |
|------|------|------|
| **yt-dlp** | YouTube/TikTok 视频下载 + 字幕提取 | `00-基础能力/05-Reddit的JSON功法.md` |
| **Vibe** | 本地语音转录，视频自动转文字（离线，Whisper） | `00-基础能力/03-Github开源项目.md` |
| **Fluent-M3U8** | 流媒体视频下载（跨平台） | `00-基础能力/03-Github开源项目.md` |
| **web-access Skill** | 浏览器自动化，复用登录态抓取社媒内容 | `00-基础能力/02-Skill设计与管理.md` |

### 采集流程

```
1. 手刷 TikTok/YouTube，看到好内容
2. 把链接丢给 AI，说"入库"
3. AI 自动：下载视频 → 转文字 → 关键帧截图 → 上传云存储
```

**关键技术点**：
- yt-dlp 支持 TikTok/YouTube/Instagram 等主流平台
- Vibe 离线转录，100+ 语言，GPU 加速 3-5x
- 视频转文字后仅 ~12KB 文本，省 token（video-use 的做法）

## 阶段2：素材拆解

### 拆解维度（Karpathy 知识库思路）

一条素材进来，拆成以下可复用元素：

| 维度 | 说明 | 示例 |
|------|------|------|
| 痛点 | 用户什么问题 | "焦虑到睡不着" |
| 场景 | 使用场景 | "工作压力大时" |
| 功效 | 产品效果 | "戴上3天明显感觉平静了" |
| 钩子 | 开头3秒抓人点 | "我以为这是智商税..." |
| 镜头结构 | 拍摄手法 | 口播→特写→佩戴→对比 |
| 激励点 | 促转化的点 | "限时买一送一" |
| 适合人群 | 目标用户 | 25-35岁职场女性 |
| 可复用元素 | 能拆出来复用的 | 开头句式、转场方式 |

### 数据存储方案

| 方案 | 适合场景 | 成本 |
|------|---------|------|
| **飞书多维表** | 团队协作，可视化好 | 免费 |
| **Notion 数据库** | 个人使用，灵活 | 免费 |
| **CSV + 本地文件夹** | 最简单，无依赖 | 0 |

## 阶段3：创作与分发

### AI 视频创作工具

| 工具 | 用途 | 说明 |
|------|------|------|
| **video-use** | 自然语言剪辑视频（Claude Code Skill） | 自动去口癖、加字幕、调色、渲染自评 |
| **Voicebox** | 本地语音克隆 + TTS | 免费版 ElevenLabs，全离线 |
| **AiToEarn** | 多平台一键分发 | 抖音/TikTok/YouTube/小红书/B站 |

### 已有的 Claude Code Skills（视频创作）

| Skill | 触发词 | 功能 |
|-------|--------|------|
| `video-outline-generation` | "视频大纲" / "策划视频" | 视频脚本大纲生成 |
| `video-script-collaborial` | "口语化" / "像说话一样" | 脚本口语化改写 |
| `video-thumbnail-check` | "封面" / "点击率" | 封面/CTR 检查 |

### video-use 安装

```bash
git clone https://github.com/browser-use/video-use
cd video-use
ln -s "$(pwd)" ~/.claude/skills/video-use
pip install -e .
brew install ffmpeg  # macOS
# Windows: choco install ffmpeg
```

进入素材文件夹后对 Claude Code 说：`将这些素材剪辑成一条发布视频`

## 各平台短视频策略要点

> 从多个选品项目的竞品分析中提炼

### TikTok

- 标签策略：#sleephacks(5000万+)、#crystals、#anxietyrelief 等
- 内容形式：15-60秒科普/开箱/佩戴展示
- 发布频率：每周 2-3 条
- 挑战赛：UGC 裂变（如 #MapToPoster、#CrystalChallenge）

### YouTube Shorts

- AI 引用数据：YouTube 在 Google AIO 中引用率 23.3%，Perplexity 中 13.9%
- YouTube 被引用频率是其他视频平台的 **200倍**
- 适合做：教程类、科普类长视频 + Shorts 切片

### 抖音（国内）

- 开箱评测、直播带货
- 站外投放：评测视频 → 淘宝/京东转化
