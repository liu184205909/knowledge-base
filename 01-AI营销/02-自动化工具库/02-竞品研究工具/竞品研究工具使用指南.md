# 竞品研究工具使用指南

> RLM 步骤 1A（竞品发现）+ 1B（数据采集与分级）的工具链 | 最后更新: 2026-05-11

---

## RLM 流程对应

```
步骤1A: 竞品发现 → 40+ 竞品域名（Google Sheets）
    │
    │  工具: serp_competitor_finder.py（SERP API 轮询）
    │
步骤1B: 数据采集 → 给竞品清单表格填数据
    │   ├── SEMrush 脚本 → AS + Organic Traffic + 截图（自动化）
    │   └── traffic.cv → 月访问量 + 流量渠道拆分（手动）
    │
步骤1B: 竞品分级 → 基于数据做 P0/P1/P2/P3 判断
    │
    └→ 深度拆解 P0 竞品
```

## 1A 竞品发现

### serp_competitor_finder.py

```bash
python serp_competitor_finder.py --keyword "crystal healing bracelet"
```

批量搜索关键词，从 SERP 结果中提取竞品域名，输出到竞品清单 Google Sheets。

## 1B 数据采集

### SEMrush 自动化脚本

```bash
# 前置：Tabbit 带 --remote-debugging-port=9222 --remote-allow-origins=* 启动，并登录 semrush.com

python semrush_to_sheets.py              # 全量查询
python semrush_to_sheets.py --start 4    # 从第4行开始
python semrush_to_sheets.py --limit 5    # 只查前5个
python semrush_to_sheets.py --dry-run    # 只提取不写回 Sheets
```

**功能：** 从 Google Sheets「竟对清单」读取域名 → SEMrush Domain Overview 提取数据 → 写回 Sheets

| 写入字段 | 列号 | 说明 |
|---------|------|------|
| AS | C | Authority Score（0-100） |
| Organic Traffic | D | 自然搜索流量 |
| SEMrush 截图 | E | Distribution 区域截图 |

### traffic.cv（手动）

浏览器访问 `https://traffic.cv/{domain}`，获取月访问量和流量渠道拆分。

| 字段 | 列号 | 格式 |
|------|------|------|
| 月访问量 | F | 如 233K、2.71M |
| 搜索% | G | 如 53.80% |
| 直接% | H | 如 14.63% |
| 社交% | I | 如 16.26% |
| 邮件% | J | 如 1.75% |
| 付费% | K | 如 0.17% |
