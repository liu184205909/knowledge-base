# 外链工具

> 最后更新：2026-05-26

---

## 当前状态：占坑

等外链建设指南手动验证通过 3-5 个平台后，再启动自动化工具开发。

## 计划方向

| 工具 | 功能 | 优先级 |
|------|------|--------|
| **PDF 批量分发工具** | 一份 PDF 自动分发到 Scribd/SlideShare/Issuu 等多个平台 | P0 |
| **外链收录检测工具** | 批量检查已发布外链是否被 Google 收录 | P1 |
| **平台账号管理工具** | 统一管理多个平台的账号和发布状态 | P2 |

## Embed Bait 工具

> 详见 [05-外链建设指南.md](../../00-营销方法论基础/05-外链建设指南.md) 5.6 节

| 工具 | 适合做什么 | 嵌入方式 | 费用 |
|------|-----------|---------|------|
| **Datawrapper** | 数据表格、对比表、排名表、地图 | 一键 embed + 自动署名链接 | 免费版 + 付费版 |
| **Flourish** | 动态可视化、竞赛图、趋势动画 | iframe embed | 免费版 + 付费版 |
| **Infogram** | 交互式图表、信息图 | embed | 免费版 + 付费版 |
| **Tableau Public** | 复杂数据分析仪表盘 | 公开嵌入 | 免费 |
| **Looker Studio** | 数据报告仪表盘 | 公开分享 + 嵌入 | 免费 |
| **Canva** | 信息图 Infographic | embed + 下载分享 | 免费版 + 付费版 |

## 前置条件

- [ ] 手动验证至少 3 个平台的发布流程和效果
- [ ] 确认哪些平台值得自动化（收录+权重+流量）
- [ ] 记录每个平台的 API 支持情况和反爬策略
