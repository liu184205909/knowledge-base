## 🔧 三种实现方式

### 方式1: MCP工具集成（最简单）⭐

**适用场景**: 快速测试，无需代码

**配置示例**:
```json
{
  "mcpServers": {
    "dalle": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-dalle"]
    }
  }
}
```

**优势**:
- ✅ 无需编程
- ✅ 在Claude Code中直接使用
- ✅ 快速上手

---

### 方式2: Python脚本（推荐，最灵活）

**适用场景**: 批量处理，完全控制

**代码示例**:
```python
import openai

# 批量生成图片
for i in range(10):
    response = openai.Image.create(
        prompt=f"Product image {i}",
        model="dall-e-3"
    )
    # 保存图片...
```

**优势**:
- ✅ 完全控制
- ✅ 批量处理
- ✅ 成本最低
