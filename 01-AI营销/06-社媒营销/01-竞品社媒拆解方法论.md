# 01-竞品社媒拆解方法论

> 拆解竞争对手的社媒账号是社媒营销的起点。不拆解竞品就发文，等于闭着眼睛打仗。
>
> **与 SEO 竞品分析的区别**：SEO 竞品分析看的是关键词排名和内容结构；社媒竞品拆解看的是内容形式、互动数据、粉丝画像、发布节奏。
>
> **自动化执行**：本文档的方法论已被 [04-社媒内容自动化流水线](./04-社媒内容自动化流水线.md) 的能力 1（自动存档）和能力 3（爆款自动拆解）覆盖。手动拆解是学习和验证阶段，生产阶段直接用流水线。

---

## 核心思路

```
发现竞品账号 → 结构化拆解 → 提炼可复用模式 → 适配到自己的账号
```

拆解不是抄，是理解"什么有效"然后用自己的品牌语言重新表达。

---

## 拆解维度

### 1. 账号基础信息

| 维度 | 关注点 |
|------|--------|
| 粉丝量级 | 判断竞品所处阶段（起步/成长/成熟） |
| 发布频率 | 日更/隔日/周更？找到最佳频率 |
| 内容形式占比 | 视频/图文/轮播/Story 各占多少 |
| 主推产品/内容 | 哪些内容被反复推广 |
| 账号定位 | 品牌号/个人号/话题号？ |

### 2. 内容分析

| 维度 | 关注点 |
|------|--------|
| 爆款内容特征 | 高互动帖的共同特征（主题/形式/长度/时间） |
| 内容主题分布 | 教育/娱乐/促销/UGC 各占比例 |
| 视觉风格 | 色调/排版/字体/拍摄手法 |
| 文案风格 | 正式/轻松/幽默/情感/争议性 |
| CTA 类型 | 引导评论/点击链接/保存/分享/购买 |

### 3. 互动分析

| 维度 | 关注点 |
|------|--------|
| 真实互动率 | (点赞+评论+分享) / 粉丝数，排除买粉水分 |
| 高互动内容特征 | 什么主题/形式/时间发布的内容互动最高 |
| 评论质量 | 评论是真实讨论还是灌水？粉丝痛点是什么？ |
| 回复策略 | 竞品是否回复评论？回复风格？ |

### 4. 发布策略

| 维度 | 关注点 |
|------|--------|
| 发布时间 | 哪个时段发布效果最好 |
| 发布节奏 | 是否有固定的内容系列/栏目 |
| Hashtag 策略 | 使用哪些标签？自创品牌标签？ |
| 跨平台策略 | 同一内容在不同平台如何差异化 |

## 拆解流程（SOP）

### Step 1：确定拆解对象（3-5 个竞品）

**选择标准**：
- 同品类直接竞品（如做水晶手链，找其他水晶品牌）
- 同受众间接竞品（如做疗愈产品，找冥想/瑜伽品牌）
- 品类标杆账号（不一定是竞品，但内容做得好的账号）

**TikTok 竞品发现**：
- 搜索品类关键词，按粉丝数排序
- 查看 `For You` 页面推荐的同品类账号
- 分析竞品的关注列表和互动账号

**Instagram 竞品发现**：
- 搜索品类标签（如 #crystaljewelry、#healingcrystals）
- 查看竞品的相似账号推荐
- 分析竞品的 tagged 照片

### Step 2：数据采集

**手动采集**（起步阶段，适合 3-5 个竞品）：

| 平台 | 采集方式 | 重点数据 |
|------|---------|---------|
| TikTok | 手动浏览 + 截图 | 播放量、点赞、评论、分享、视频时长 |
| Instagram | 手动浏览 + 截图 | 点赞、评论、保存、轮播页数 |
| YouTube | 浏览频道 + 分析工具 | 播放量、订阅转化、视频时长分布 |

**工具辅助采集**（量级增大后）：

| 工具 | 功能 | 参考 |
|------|------|------|
| Apify | 抓取 TikTok 账号数据（视频/互动/粉丝） | `02-自动化工具库/08-短视频工具/` |
| FastMoss | TikTok Shop 数据分析，2 亿+ 商品数据库 | `02-自动化工具库/08-短视频工具/短视频工具与自动化流程.md` |
| 分析竞品内容 | AI 提取字幕/文本，分析内容主题 | `02-自动化工具库/13-视频转文稿/` |

### Step 3：结构化分析

**用 AI 辅助分析竞品内容**：

```
你是一位社媒分析师。请分析以下竞品账号的内容数据，输出结构化报告：

## 输入数据
[粘贴竞品最近 20-30 条内容的数据：主题、形式、互动数据、发布时间]

## 分析要求
1. 内容主题分布（分类统计各主题占比）
2. 爆款特征分析（互动 top 5 内容的共同点）
3. 发布规律（最佳时间、最佳频率）
4. 文案/视觉风格总结
5. 可复用的内容模式（列出 5-10 个可直接借鉴的内容框架）

## 输出格式
用表格 + 要点列表，每个结论附数据支撑。
```

### Step 4：提炼可复用模式

从拆解中提炼出**内容模板**，而非具体内容：

| 提炼对象 | 示例 |
|---------|------|
| **钩子模板** | "I thought this was a scam until..." / "POV: you just discovered..." |
| **内容框架** | 问题 → 原因 → 解决方案 → CTA |
| **视觉模板** | 产品特写 → 佩戴效果 → 用户评价截图 |
| **互动模板** | 投票帖 / "Tag someone who..." / "Which one would you choose?" |

### Step 5：适配与执行

将提炼的模式适配到自己品牌：

```
竞品模式："POV: you discover crystal healing"（13M 播放）
  ↓ 适配
我的版本："POV: you find out your crystal matches your zodiac sign"（结合星座差异化）
```

## 拆解报告模板

```markdown
# [竞品名称] 社媒拆解报告

## 基础信息
- 平台：TikTok / Instagram / YouTube
- 粉丝：XX
- 总内容数：XX
- 平均互动率：XX%

## 内容概览
| 维度 | 数据 |
|------|------|
| 发布频率 | X 条/周 |
| 主内容形式 | 视频 XX% / 图文 XX% |
| 平均视频时长 | XX 秒 |
| 最常用 Hashtag | #xxx #xxx #xxx |

## 爆款分析（Top 5）
| 排名 | 主题 | 形式 | 时长 | 播放 | 点赞 | 评论 | 爆款原因 |
|------|------|------|------|------|------|------|---------|

## 可复用模式
1. [模式名称]：[描述] → 预计适配到我的账号方式
2. ...

## 差异化机会
- 竞品没覆盖的内容主题：[列表]
- 竞品内容短板（可以做更好的）：[列表]
```
