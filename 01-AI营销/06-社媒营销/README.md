# 06-社媒营销模块

> 社媒营销与 SEO 内容分发是两套不同的逻辑。本模块覆盖独立的社媒营销策略，而非博客内容的社媒分发。
>
> **本模块的核心是自动化流水线**（[04-社媒内容自动化流水线.md](./04-社媒内容自动化流水线.md)）：输入一条竞品爆款链接 → 自动完成存档、拆解、打标签、编译、去重 → 输出可直接拍摄的脚本。01/02/03 文档是方法论层，04 是把它们全自动串起来的引擎。

---

## 本模块 vs SEO 内容分发

| | SEO 内容分发（RLM 步骤4） | 社媒营销（本模块） |
|---|---|---|
| **起点** | 已写好的博客文章 | 竞品社媒账号拆解 / 平台热点 |
| **内容** | 博客拆成帖子（二手内容） | 原生社媒内容创作（一手内容） |
| **逻辑** | 一次创作、多平台复用 | 按平台算法+用户偏好独立策划 |
| **目标** | 外链 + 引流回网站 | 品牌曝光 + 粉丝增长 + 直接转化 |
| **流量类型** | 搜索流量（用户主动搜） | 推荐流量（算法推给用户） |

**关键区别**：SEO 内容分发是把博客"搬运"到社媒平台；社媒营销是围绕平台生态独立策划内容——拆解竞品账号、研究算法偏好、创作原生爆款、达人合作带货。

## 与外链资产库的关系

社媒营销的一个核心产出是**外链**——达人测评链接、社媒账号 Profile 链接、内容中的品牌链接，都是外链资产。详见 [05-外链资产库/README.md](../05-外链资产库/README.md) 的"外链获取全渠道总览"。

```
社媒营销产出的外链类型：
├── 达人测评/推荐 → 达人合作型外链
├── 联盟客内容 → 达人合作型外链
├── 社媒账号 Profile（Pinterest/Instagram） → 视觉/产品型外链
├── 社媒帖子中的链接 → 平台自助型外链
└── UGC 用户内容中的品牌提及 → 社区参与型外链
```
