# 02-短视频爆款创作与分发

> 短视频是社媒营销的核心战场。本文档覆盖**原生短视频**的创作策略——从竞品爆款拆解到原创内容制作到多平台分发。
>
> **与"博客→视频"分发的区别**：`01-营销方法论基础/04-社媒内容分发工作流.md` 中的 4D 阶段是把已有博客改写成 TTS 脚本再合成视频。本文档是**独立策划**短视频内容，不依赖博客。
>
> **自动化执行**：本文档的爆款创作流程已被 [04-社媒内容自动化流水线](./04-社媒内容自动化流水线.md) 的能力 5（内容自动编译）和能力 2（数据自动追踪）覆盖。手动创作是方法论学习阶段，批量生产阶段直接用流水线。

---

## 核心思路

```
竞品爆款拆解 → 提炼爆款公式 → 原生内容创作 → 批量生产 → 多平台分发 → 数据反馈
```

短视频的本质是**算法推荐**，不是搜索。关键不是"内容好不好"，而是"前 3 秒能不能让人停下来"。

---

## 一、爆款拆解与公式提炼

### 爆款视频的结构拆解

一条爆款视频通常包含以下可拆解元素：

| 元素 | 说明 | 示例 |
|------|------|------|
| **钩子（Hook）** | 前 3 秒决定生死 | "我以为这是智商税..." / "Stop scrolling if you..." |
| **痛点/悬念** | 制造情绪共鸣 | "焦虑到睡不着" / "你不知道你的出生石是什么" |
| **解决方案** | 提供价值 | 展示产品使用 + 效果 |
| **社交证明** | 建立信任 | 用户评价/前后对比/达人推荐 |
| **CTA** | 引导行动 | "Link in bio" / "Save this for later" / "Follow for more" |

### 爆款公式模板

| 公式名 | 结构 | 适合品类 |
|--------|------|---------|
| **反转型** | 反常识开场 → 证据 → 反转结论 | 教育/科普类 |
| **开箱型** | 惊喜表情 → 开箱过程 → 真实反应 | 电商/产品类 |
| **对比型** | Before/After → 产品展示 → 效果证明 | 美妆/穿搭/家居 |
| **痛点型** | 痛点共鸣 → 解决方案 → 效果展示 | 功能性产品 |
| **合集型** | "Top 5 / 3 ways to" → 逐一展示 | 知识/推荐类 |
| **Vlog型** | 日常生活 + 自然植入产品 | 个人IP/生活方式 |

### 拆解竞品爆款的 SOP

```
1. 找到竞品播放量 top 10 的视频
2. 逐条拆解：钩子类型 / 时长 / 节奏 / 视觉风格 / CTA
3. 找共性：top 10 中有多少用了同一种钩子？哪种 CTA 转化最好？
4. 提炼公式：共性 = 爆款公式
5. 适配创作：用自己的产品/品牌套入公式
```

## 二、原生内容创作

### 内容策划：从"拍什么"开始

**不要一上来就拍，先策划内容矩阵**：

| 内容类型 | 占比建议 | 目的 | 示例 |
|---------|---------|------|------|
| 教育科普 | 30% | 建立专业形象 | "水晶入门指南" / "如何选你的第一串水晶" |
| 产品展示 | 25% | 直接转化 | 开箱 / 佩戴效果 / 材质特写 |
| UGC/用户故事 | 20% | 社交证明 | 用户评价截图 / 使用前后对比 |
| 趋势热点 | 15% | 借势流量 | 结合节日/热门话题/平台挑战 |
| 人设/幕后 | 10% | 拉近距离 | 打包发货 / 选品过程 / 团队日常 |

### 脚本创作流程

```
选定主题 → 写 Hook（3秒钩子）→ 写正文（价值传递）→ 写 CTA → 控制时长
```

**脚本模板**：

```
[Hook - 3秒] 反常识/悬念/痛点，让人停下来
  ↓
[痛点展开 - 5-10秒] 让目标用户觉得"说的就是我"
  ↓
[解决方案 - 15-30秒] 产品展示 + 效果证明
  ↓
[CTA - 3-5秒] 明确引导（点击链接 / 关注 / 保存）
```

### AI 辅助批量创作

**一次策划一周的内容**：

```
你是一位短视频内容策划师。为 [品牌/产品] 策划一周 5 条短视频内容。

品牌信息：[品牌描述]
目标平台：TikTok + Instagram Reels
目标用户：[用户画像]
竞品爆款公式：[从拆解中提炼的公式]

输出格式：
每条视频包含：
1. 主题 + Hook 文案（前 3 秒说什么）
2. 完整脚本（口语化，标注画面/字幕）
3. 建议时长
4. 使用场景（教学/展示/UGC/热点/幕后）
5. 建议发布时间
```

## 三、视频制作

### 制作方式选择

| 方式 | 成本 | 速度 | 质量 | 适用场景 |
|------|------|------|------|---------|
| **手机实拍** | 0 | 中 | 取决于拍摄水平 | 产品展示、开箱、Vlog |
| **AI 生成（美图 Seedance）** | ~35 元/条 | 快 | 中高 | 产品带货视频，8 种模板 |
| **数字人口播（HeyGen）** | ~10-15 元/条 | 快 | 中 | 知识类/教程类口播 |
| **Codex + cutcli 混剪** | 主要是时间成本 | 中 | 高 | 跨语言矩阵号批量混剪 |
| **video-use 剪辑** | 0（开源） | 慢 | 取决于 prompt | 自然语言剪辑 |

> 详细工具说明见 `02-自动化工具库/08-短视频工具/短视频工具与自动化流程.md`

### 制作质量标准

| 维度 | 最低标准 | 理想标准 |
|------|---------|---------|
| 分辨率 | 720p | 1080p+ |
| 时长 | TikTok: 15-60s / YouTube Shorts: <60s | 根据内容类型调整 |
| 字幕 | 必须有（85% 用户静音刷视频） | 动态字幕，关键词高亮 |
| 背景音乐 | 使用平台热门 BGM | BGM 节奏与画面切换同步 |
| 前 3 秒 | 有明确钩子 | 钩子 + 视觉冲击 |

## 四、多平台分发策略

### 各平台算法偏好（2026）

| 平台 | 算法核心权重 | 最佳时长 | 最佳发布频率 |
|------|------------|---------|------------|
| **TikTok** | 完播率 > 分享 > 评论 > 点赞 | 15-60s | 每天 1-3 条 |
| **Instagram Reels** | Save > Share > 评论 > 点赞 | 15-30s | 每天 1 条或隔天 |
| **YouTube Shorts** | 观看时长 + 订阅转化 | 30-60s | 每天 1-2 条 |
| **抖音** | 完播率 > 评论 > 分享 | 15-60s | 每天 1-2 条 |

### 同一内容的多平台适配

**不是直接搬运，而是适配**：

| 适配项 | TikTok | Instagram Reels | YouTube Shorts |
|--------|--------|----------------|----------------|
| 画面比例 | 9:16 | 9:16 | 9:16 |
| 字幕风格 | 大字、居中、彩色 | 精致、品牌色 | 简洁、底部 |
| BGM | 平台热门音乐 | 原创音频更优 | 可选 |
| CTA | "Link in bio" | "Link in bio" / "Save this" | "Subscribe" |
| Hashtag | 3-5 个混合标签 | 不影响推荐 | 标题含关键词 |
| 文案 | 简短口语 | 首行 < 125 字符 | 标题优化 |

### 发布排期

```
周一：教育科普类（周一学习意愿高）
周三：产品展示类（周中购买意愿上升）
周五：热点/娱乐类（周五轻松刷视频）
周六：UGC/互动类（周末互动率高）
```

## 五、数据反馈与迭代

### 核心指标

| 指标 | 计算方式 | 健康范围 |
|------|---------|---------|
| 完播率 | 看完人数 / 播放数 | > 30% |
| 互动率 | (点赞+评论+分享) / 播放数 | > 3% |
| 粉丝转化率 | 新增粉丝 / 播放数 | > 0.5% |
| 分享率 | 分享数 / 播放数 | > 0.5% |

### 迭代闭环

```
发布 → 48小时观察数据 → 分析表现 → 调整内容方向 → 下一轮创作
  ↑                                                    |
  └────────────────────────────────────────────────────┘
```

**数据驱动的内容调整**：

| 数据表现 | 诊断 | 调整方向 |
|---------|------|---------|
| 播放量低、完播率高 | 推荐量不够，内容 OK | 优化 Hook、调整发布时间、加热门 Hashtag |
| 播放量高、完播率低 | 推荐量够，内容留不住 | 缩短时长、加快节奏、优化前 3 秒 |
| 互动率低 | 内容没有引发情绪 | 加入争议性观点、提问、投票 |
| 粉丝转化低 | 没有给关注理由 | 加 "Follow for more" CTA、建立内容系列 |
