# 执行指令模板集

> RLM项目各阶段可直接使用的AI执行指令
> 来源：RLM营销方法论 v3.1 | 适用阶段：步骤1B之后（重点竞品拆解）
> 使用方式：复制对应指令，填入`[方括号]`中的信息，发给AI执行

---

## 1. 项目启动指令

```text
请根据以下产品信息，帮我生成RLM项目的00-项目简报.md。

产品：[填写产品]
目标市场：[填写市场]
业务类型：[B2B / B2C / SaaS / 工具站 / 内容站]
我们的身份：[中国供应商 / 出口品牌 / 独立站卖家 / 软件团队]
核心目标：[询盘 / 销售 / 订阅 / 下载 / 联盟变现]
已知关键词：[填写关键词]
已知竞品：[填写竞品]

要求：
1. 明确项目边界
2. 提炼初始关键词
3. 标注待确认问题
4. 不做流量或收益预测
```

---

## 2. 竞品发现指令（步骤1A）

```text
请围绕 [产品 / 关键词]，面向 [目标市场]，帮我批量发现相关竞品。

要求：
1. 使用核心词、类目词、应用场景词、替代词等多组关键词搜索
2. 限定目标市场区域和语言
3. 区分本土品牌、中国出口商、平台、综合电商
4. 目标是找到40家以上相关竞品
5. 输出原始竞品清单（含：名称、网址、类型、初始来源）
6. 不做 P0/P1/P2/P3 分级，分级在步骤1B完成
7. 过滤明显无关的大型平台聚合页
```

> 适用阶段：步骤1A（竞品发现）。产出原始竞品池后，进入步骤1B筛选。

---

## 3. 竞品筛选与优先级指令（步骤1B）

```text
请基于以下原始竞品清单，进行竞品筛选和优先级分级。

输入：竞品清单（40+家）

要求：
1. 清洗重复、无效、低相关网站
2. 补充判断字段（DR/权重、自然流量、核心关键词、页面质量）
3. 按以下标准分级：
   - P0：高相关、高参考价值，必须深度拆解
   - P1：相关性强，用于结构和内容参考
   - P2：局部参考价值（某个页面/类目）
   - P3：相关性弱，暂不研究
4. 输出：更新后的竞品清单 + 竞品优先级说明.md
5. 每个 P0 竞品说明为什么优先研究
6. 没有 DR/流量数据时标注"待验证"
```

> 适用阶段：步骤1B（竞品筛选）。分级完成后，P0 竞品进入深度拆解（使用指令5）。

---

## 4. 竞品深度拆解指令

```text
请根据以下竞品网站信息，按照RLM的竞品深度拆解模板，生成一份Markdown拆解文档。

竞品名称：[填写]
竞品网址：[填写]
目标市场：[填写]
业务类型：[填写]

要求分析：
1. 基本信息
2. 品牌定位
3. 网站结构
4. 产品分类
5. URL结构
6. 内容策略
7. SEO策略观察
8. 转化路径
9. 模仿、超越、差异化机会

没有数据的地方请标注"待验证"，不要编造。
```

> 配合模板：`02-模板库/竞品深度拆解模板.md`

---

## 5. 关键词研究指令

```text
请基于以下竞品URL、类目名和关键词数据，生成RLM关键词研究结果。

输入数据：
[粘贴竞品URL / 关键词CSV / 类目列表]

要求：
1. 提取关键词
2. 按主题集群分组
3. 判断搜索意图
4. 标注适合页面类型
5. 给出优先级
6. 没有搜索量、KD、CPC数据时标注"待验证"
7. 输出关键词库字段表
```

> 配合参考：`01-关键词研究方法论.md`

---

## 6. 网站规划指令

```text
请基于以下RLM输入文件，生成网站规划方案。

输入：
1. 竞品结构分析
2. 关键词库
3. 用户画像
4. 策略清单

要求输出：
1. 网站结构设计
2. URL结构规划
3. 产品类目规划
4. 核心页面规划
5. 产品页面规划
6. 内链建议
7. 每个页面标注目标关键词、页面目标、参考竞品和差异化点
```

---

## 7. 内容策略指令

```text
请基于以下关键词库、竞品内容分析和策略清单，生成RLM内容策略。

输入：
1. 关键词库
2. 内容策略分析
3. 用户画像
4. 模仿 / 超越 / 差异化策略清单
5. 网站结构规划

要求输出：
1. 内容类型规划
2. 内容选题方向
3. 内容清单
4. 每篇内容包含标题、目标关键词、搜索意图、内容类型、优先级、参考竞品、差异化角度
5. 不输出无依据流量预测、收益预测和发布时间表
```

## 8. 内容Brief指令

```text
请为以下选题生成RLM内容Brief。

文章标题：[填写]
目标关键词：[填写]
搜索意图：[填写]
内容类型：[填写]
参考竞品：[填写]
目标用户：[填写]

要求：
1. 分析竞品已经覆盖什么
2. 找出竞品缺口
3. 提出我们的差异化角度
4. 给出推荐大纲
5. 给出必须包含的信息
6. 给出内链建议
7. 给出CTA建议
```

> 配合模板：`02-模板库/内容Brief模板.md`
