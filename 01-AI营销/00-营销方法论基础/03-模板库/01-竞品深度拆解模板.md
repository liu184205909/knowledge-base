# 竞品深度拆解模板

> 适用对象：每个P0竞品单独生成一份Markdown文档
> 文件命名：`01-竞品分析/1D-竞品深度拆解/P0-1-竞品名称.md`
> 来源：RLM营销方法论 - 步骤1D竞品深度拆解（按证据角色组合使用核心输入和辅助输入）
>
> **边界说明**：1D 对单个竞品做完整拆解，输出事实、观察和候选策略点；1E/1F 基于全部 1D 文档做跨竞品汇总对比，不在 1D 中做最终网站结构或内容策略决策。

---

### 核心输入（必须使用）

|------|---------|----------|------|
| 该竞品 SEMrush Top Pages | 是/否 |  | 判断高流量页面、页面类型、流量集中度 |
| 该竞品 SEMrush Top Keywords | 是/否 |  | 判断排名关键词、排名URL、关键词意图和页面承接关系 |
| 该竞品 Sitemap 解析结果 | 是/否 |  | 判断网站结构、页面清单、URL规则、内容类型分布 |
| Seed-Master 关键词主表（1C） | 是/否 |  | 市场需求参照，判断覆盖/遗漏的主题、意图、页面类型 |
| Keyword-Page-Proof 竞品验证视图（1C） | 是/否 |  | 判断哪些关键词-页面组合已被流量验证 |

### 辅助输入（按需使用）

| 数据源 | 是否使用 | 位置/链接 | 用途 |
|------|---------|----------|------|
| 竞品清单汇总（AS、Organic Traffic、月访问量、竞品类型） | 是/否 |  | 仅用于判断竞品权重、参考优先级和背景定位 |
| 页面截图、渠道拆分、站点级流量概览 | 是/否 |  | 仅在分析商业模式、转化路径或内容体验时按需引用 |
| 核心页面轻量复核 | 是/否 |  | 按需访问关键页面验证结构或内容 |

### 数据缺失降级规则

| 缺失数据 | 影响章节 | 降级处理 |
|---------|---------|---------|
| 无 Sitemap | §3/§4/§5 | 基于 Top Pages 推断页面结构 + 核心页面爬取补充，结论标注"初步判断（缺 Sitemap）" |
| Top Pages 稀疏（<10 条有效数据） | §6/§7 | 流量分布结论不可靠，侧重内容类型和关键词意图分析而非流量排序 |
| Top Keywords 稀疏 | §7 | 无法判断关键词布局全貌，侧重已有排名词的意图和页面承接关系 |
| Seed Keywords 无覆盖 | §6 缺口分析 | 无法判断该竞品相对市场需求覆盖/遗漏，只能分析竞品自身内容表现 |
| Keyword-Page-Proof 无记录 | §7 验证 | 无法确认流量证据，关键词排名观察标注为"未验证" |

> 未使用的核心输入必须说明原因。缺少 Top Pages、Top Keywords 或 Sitemap 时，不得输出"完整网站结构/完整内容结构"之类的强结论，只能标注为"初步判断"。辅助输入不使用时无需说明原因。
