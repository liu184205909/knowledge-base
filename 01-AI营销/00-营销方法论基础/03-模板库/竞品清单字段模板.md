# 竞品清单字段模板

> 用于RLM步骤1A竞品发现和1B竞品筛选。所有竞品清单必须包含以下字段，确保后续分析有足够数据。
>
> **数据存储**：每个项目在 Google Drive 中创建独立的竞品清单 Google Sheets，表格链接记录在项目文件夹（`01-竞品分析/`）中。通过 Google Workspace MCP 读写。具体表格链接由各项目的项目简报或竞品分析目录下的文件指定。

---

## 1A 竞品发现 — 原始竞品清单字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 编号 | 自动编号 | 1, 2, 3... |
| 竞品名称 | 品牌名或公司名 | CompetitorA |
| 网址 | 完整URL | https://competitora.com |
| 类型 | 本土品牌/中国出口商/平台/综合电商/垂直内容站 | 本土品牌 |
| 初始来源 | Google搜索/Ahrefs/行业目录/用户指定 | Google搜索 |
| 相关性判断 | 高/中/低（粗判即可，1B阶段再精确分级） | 高 |
| 备注 | 初步观察 | 产品线相似、价格区间 |

## P0/P1/P2/P3 分级标准

| 优先级 | 定义 | 评分要求 | 后续动作 |
|--------|------|---------|---------|
| P0 | 高相关、高参考价值，必须深度拆解 | 业务相关性≥8 且 (SEO表现≥7 或 结构价值≥8) | 单独生成拆解文档 |
| P1 | 相关性强，用于结构和内容参考 | 业务相关性≥6 且 至少一项价值≥6 | 进入对比分析 |
| P2 | 局部参考价值（某个页面/类目） | 某单项突出但综合一般 | 记录参考点 |
| P3 | 相关性弱，暂不研究 | 总分偏低 | 归档 |

**评分维度**（1-10分）：

| 维度 | 评估内容 |
|------|---------|
| 业务相关性 | 产品/服务/市场的重叠度 |
| SEO表现 | DR、自然流量、关键词排名 |
| 网站结构价值 | 导航设计、页面层级、URL结构的参考价值 |
| 内容策略价值 | 博客内容类型、质量、深度的参考价值 |
| 转化路径价值 | CTA设计、表单、询盘/购买路径的参考价值 |
| 差异化机会 | 这个竞品做得不好的地方 = 我们的机会 |
