# RLM项目文件夹结构模板

> 用于新建 RLM 项目目录。执行版只保留核心目录，本模板保留完整结构。

```text
XX产品/
├── 00-项目简报.md
├── 01-竞品分析/
│   ├── 1A-竞品清单.md               ← 1A 输出：Google Sheets 链接 + 筛选口径 + 状态说明
│   ├── 1B-数据采集索引.md           ← 1B 各 Sheet 链接、采集状态、最后更新时间
│   ├── 1B-sitemap解析/               ← 1B 输出之一：主报告+按竞品拆分的详细解析
│   │   ├── 1B-sitemap解析报告.md      ← sitemap总览+横向汇总（索引）
│   │   ├── 02-Energy-Muse.md          ← 编号=1A竞品清单序号
│   │   ├── 04-Crystal-Vaults.md
│   │   └── ...
│   ├── 1C-关键词研究/
│   │   ├── 关键词库（Google Sheets）
│   │   ├── 关键词分组与优先级.md
│   ├── 1D-竞品深度拆解/               ← 1D 输出：按竞品拆分的深度分析
│   │   ├── 竞品A.md
│   │   └── ...
│   ├── 1E-网站结构分析.md
│   ├── 1F-内容策略分析.md
│   ├── 1G-用户画像.md
│   ├── 1H-策略清单.md
│   └── 竞品分析报告.md               ← 步骤1总结报告
├── 02-网站规划/
│   ├── 网站结构设计.md
│   ├── URL结构规划.md
│   ├── 产品类目规划.md
│   ├── 核心页面规划.md
│   ├── 产品页面规划.md
│   ├── 页面决策表.md              ← 2B 输出：做/不做/批次决策
│   ├── pages/                    ← 2C 输出：核心页面构建脚本
│   │   ├── home.js              ← 首页
│   │   ├── about-us.js          ← 关于我们
│   │   ├── contact-us.js        ← 联系我们
│   │   └── ...                 ← 其他核心页面（依页面决策表而定）
│   ├── assets/
│   │   ├── site-images.js       ← 全站图片资产注册表
│   │   └── images/
│   │       ├── [page]-prompts.md ← 2C 输出：逐页图片prompt清单
│   │       └── generated/        ← 2D 图片生成缓存
│   └── templates/               ← 2C 输出：可复用区块模板
│       ├── elementor-utils.js   ← Elementor 工具库
│       └── ...                 ← 其他可复用区块
├── 03-内容策略/
│   ├── 内容类型规划.md
│   ├── 内容选题方向.md
│   ├── 内容清单（Google Sheets）
│   └── 内容Brief/
│       ├── 文章A-Brief.md
│       └── ...
├── 04-商业模式/                    ← 步骤5输出（可选）
│   ├── 商业模式设计.md
│   └── ...
└── 05-社媒分发/                    ← 步骤4输出（可选，内容稳定后再建）
    ├── 分发策略.md               ← 4A：平台选择 + 格式矩阵
    ├── 图文内容/                 ← 4B：Twitter/LinkedIn/Reddit/IG/Pinterest
    │   ├── 文章A-Twitter线程.md
    │   ├── 文章A-LinkedIn.md
    │   └── ...
    ├── 音视频内容/               ← 4C：YouTube/TikTok/Podcast
    │   ├── 文章A-TTS脚本.md
    │   ├── 文章A-音频.mp3
    │   └── ...
    └── 发布排期.md               ← 4D：排期 + 效果追踪
```

不需要商业模式的业务可省略 `04-商业模式/`，不需要社媒分发的业务可省略 `05-社媒分发/`。`04-商业模式/` 是标准项目目录中的固定商业层；不要因为省略社媒分发而把商业模式挪到其他编号。
