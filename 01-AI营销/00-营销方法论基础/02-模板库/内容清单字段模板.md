# 内容清单字段模板

> 用于RLM步骤3内容策略。内容清单是内容创作的核心依据，每篇文章必须绑定完整字段。

---

## 内容清单字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 编号 | 自动编号 | 001, 002... |
| 文章标题 | 暂定标题（后续可调整） | Best Gel Blaster Guns 2025: Top 10 Picks |
| 目标关键词 | 主关键词 | best gel blaster gun |
| 次要关键词 | 2-5个相关关键词 | gel blaster review, top gel blaster |
| 搜索意图 | 信息型/对比型/交易型 | 交易型 |
| 内容类型 | 评测/指南/教程/对比/案例 | 评测榜单 |
| 内容阶段 | 认知/考虑/决策 | 决策 |
| KD难度 | 关键词难度（低/中/高） | 中 |
| 搜索量 | 月搜索量（如有） | 5,400 |
| CPC | 每次点击成本（如有） | $1.2 |
| 优先级 | P0/P1/P2/P3 | P0 |
| 参考竞品 | 可参考的竞品内容 | CompetitorA的同类文章 |
| 差异化角度 | 我们的内容如何做得不同 | 加入实测视频+价格对比表 |
| 目标URL | 规划的URL路径 | /blog/best-gel-blaster-guns |
| Brief状态 | 未开始/进行中/已完成 | 未开始 |

---

## 内容优先级判断

| 优先级 | 条件 | 占比建议 |
|--------|------|---------|
| P0 | 高搜索量 + 低KD + 交易型意图 | 20% |
| P1 | 中等搜索量 + 中KD + 竞品有缺口 | 30% |
| P2 | 信息型 + 长尾词 + 支撑P0内容 | 30% |
| P3 | 低优先级，填充内容矩阵 | 20% |

---

## 内容类型分类

| 类型 | 说明 | 典型标题模式 |
|------|------|-------------|
| 评测/榜单 | 多产品对比评测 | "Best X for Y", "Top 10 X" |
| 对比 | 两个产品/方案直接对比 | "A vs B" |
| 指南 | 完整选购或使用指南 | "How to Choose X", "Ultimate Guide to X" |
| 教程 | 操作步骤类 | "How to X", "X Tutorial" |
| 科普 | 原理解释类 | "What is X", "X vs Y Difference" |
| 案例 | 应用场景案例 | "X in [Industry]" |

---

## 质量检查

- [ ] 每篇文章绑定了目标关键词
- [ ] 每篇文章标注了搜索意图
- [ ] 每篇文章标注了内容类型
- [ ] P0文章标注了参考竞品和差异化角度
- [ ] 没有重复关键词的文章
