## 📊 市场对比

| 维度 | 国内市场 | 海外市场 |
|------|---------|---------|
| **主要平台** | 微信、淘宝、京东、拼多多 | Amazon、eBay、AliExpress |
| **支付方式** | 微信支付、支付宝 | PayPal、信用卡 |
| **物流** | 国内快递（3-5天） | FBA（2-3天）、国际物流（15-30天） |
| **竞争程度** | 激烈（价格战） | 中等（品牌化空间大） |
| **利润空间** | 较低（20-30%） | 较高（40-50%） |
| **认证要求** | 3C认证 | CE、FCC、UL等 |
| **运营难度** | 低（语言文化熟悉） | 高（跨文化、合规） |
| **启动资金** | 5-10万 | 10-30万（含FBA库存） |

## 🔍 案例特征

### AI教育案例
**商业模式**: 免费流量层 + 付费转化层 + B2B增长层

**核心亮点**:
- 混合模式: WaytoAGI（免费）+ Coursiv（付费）
- 微信生态: 飞书知识库 + 小程序会员
- 三层架构: 流量层、转化层、增长层

**市场机会**:
- 中国AI教育市场爆发
- 40+职场人士焦虑驱动
- 线上学习接受度高

### 乐高兼容积木品牌
**商业模式**: MOC玩家零件市场 + 补件服务

**核心亮点**:
- 细分市场: 避开set红海，专注零件蓝海
- 质量定位: 接近Keeppley，超越白牌
- 社区驱动: Rebrickable、Reddit、Facebook

**市场机会**:
- 全球积木市场131亿美元
- 乐高价格高（838亿营收，毛利率70%）
- 零件市场空白（竞争小）

### Replacement产品选品
**商业模式**: 待筛选（根据产品确定）

**可能方向**:
- 汽车配件: 海外DIY文化成熟
- 电子配件: 更新快，需求持续
- 工具配件: 耗材属性强
- 工业零件: B2B客单价高

**筛选标准**:
- Amazon搜索量
- FBA友好度（小而轻）
- 认证门槛（低优先）
- 利润空间（40%+）
