# 博客内容生产 Skill

基于SEO关键词策略，生产高质量的博客文章。

## 📋 执行流程

### 阶段1: 选题研究（SEO关键词驱动）

**关键词分析**：
- 基于content-strategy的关键词清单
- 选择目标关键词
- 分析搜索意图（信息型/导航型/交易型）

**选题维度**：
1. **搜索量**: 月搜索量 > 100
2. **竞争度**: KD < 50（中低竞争）
3. **相关性**: 与产品高度相关
4. **转化潜力**: 能引导到产品页

**选题类型**：
- How-to指南（如何做...）
- What-is解释（什么是...）
- List列表（X个...）
- Comparison对比（A vs B）
- Case Study案例研究

**输出**: 10-20个选题清单

### 阶段2: 大纲生成（结构化框架）

**标准大纲结构**：

```
标题（包含主关键词）

引言（100-200字）
- Hook: 吸引注意的开头
- 问题: 用户痛点
- 预览: 文章将解决什么

第1节 (H2: 包含相关关键词)
  - 第1段 (H3)
  - 第2段 (H3)

第2节 (H2: 包含相关关键词)
  - 第1段 (H3)
  - 第2段 (H3)

第3节 (H2: 包含相关关键词)
  - 第1段 (H3)
  - 第2段 (H3)

结论（100-200字）
- 总结: 核心观点
- CTA: 下一步行动

常见问题 (FAQ)
- Q1: [问题]
- A1: [答案]
```

**大纲要求**：
- H1: 1个（包含主关键词）
- H2: 3-5个（包含相关关键词）
- H3: 8-12个（包含长尾关键词）
- 段落: 每段2-4句话

### 阶段3: 深度撰写（1500-3000字）

**内容质量标准**：

1. **原创性**: 100%原创，无抄袭
2. **深度**: 深入分析，不是表面介绍
3. **准确性**: 基于事实，可验证
4. **实用性**: 提供可操作的建议
5. **可读性**: 清晰易懂，避免术语

**撰写技巧**：

**引言部分**：
- 用数据或故事开头
- 提出用户关心的问题
- 说明文章将提供什么价值

**正文部分**：
- 每个段落有一个核心观点
- 使用例子和数据支持观点
- 加入实用建议和最佳实践
- 使用过渡词连接段落

**结论部分**：
- 总结文章核心观点
- 提供进一步的行动建议
- 引导用户查看相关内容

**内容长度控制**：
- 最短: 1500字
- 推荐: 2000-2500字
- 最长: 3000字

### 阶段4: SEO优化（关键词布局 + 元数据）

**关键词密度**：
- 主关键词: 1-2%
- 相关关键词: 0.5-1%
- 总关键词密度: 1-2%

**关键词位置**：
- 标题（H1）: 包含主关键词
- 前100字: 出现主关键词
- H2/H3: 包含相关关键词
- 正文: 自然分布关键词
- 图片alt: 描述性文字 + 关键词
- URL: 简短 + 关键词

**元数据优化**：

**标题标签（Title Tag）**：
- 长度: 50-60字符
- 格式: [主关键词] | [品牌名]
- 示例: "智能宠物健康监测器选购指南 | PetCare"

**元描述（Meta Description）**：
- 长度: 150-160字符
- 内容: 概括文章 + 吸引点击
- 示例: "如何选择最适合的智能宠物健康监测器？本文详细对比5款主流产品，从功能、价格、用户评价全方位分析，帮你做出明智选择。"

**URL优化**：
- 长度: 短（< 75字符）
- 格式: 小写 + 连字符
- 示例: `/blog/smart-pet-health-monitor-guide`

**图片优化**：
- 文件名: 描述性 + 关键词
- Alt文本: 描述性文字
- 压缩: < 100KB

**内部链接**：
- 链接到相关产品页: 2-3个
- 链接到相关博客: 3-5个
- 使用描述性锚文本

### 阶段5: 质量审核（可读性 + 准确性）

**可读性检查**：

1. **Flesch Reading Ease**: 60-70分
   - 句子长度: 平均15-20词
   - 段落长度: 2-4句话
   - 避免长难句

2. **格式检查**：
   - H1/H2/H3层级清晰
   - 段落之间有空行
   - 使用列表（bullet points）
   - 使用加粗强调重点

3. **语法检查**：
   - 无拼写错误
   - 无语法错误
   - 标点符号正确

**准确性检查**：

1. **事实核查**：
   - 数据可验证
   - 引用有来源
   - 产品信息准确

2. **链接检查**：
   - 所有链接可访问
   - 链接相关性强
   - 无死链

3. **SEO检查**：
   - 关键词密度合适
   - 元数据完整
   - H标签层级正确

**质量评分**：
- 内容深度: ⭐⭐⭐⭐⭐ (1-5分)
- SEO优化: ⭐⭐⭐⭐⭐ (1-5分)
- 可读性: ⭐⭐⭐⭐⭐ (1-5分)
- 准确性: ⭐⭐⭐⭐⭐ (1-5分)
