# 市场分析 Skill

基于RLM递归思想，对产品进行全面的市场分析。

## 📋 执行流程

### 阶段1: 产品评估（100分制）

使用100分制评估系统，从以下4个维度评分：

#### 维度1: 市场分析（30分）
- **市场规模**（10分）：
  - > 100亿（10分）
  - 10-100亿（7分）
  - 1-10亿（4分）
  - < 1亿（0分）

- **市场增长**（10分）：
  - > 50%（10分）
  - 20-50%（7分）
  - 10-20%（4分）
  - < 10%（0分）

- **市场趋势**（10分）：
  - 快速上升（10分）
  - 稳定增长（7分）
  - 饱和（4分）
  - 下降（0分）

#### 维度2: 产品分析（30分）
- **产品独特性**（10分）
- **产品成熟度**（10分）
- **价格竞争力**（10分）

#### 维度3: 竞争分析（20分）
- **竞争激烈度**（10分）
- **进入壁垒**（10分）

#### 维度4: 运营分析（20分）
- **供应链难度**（10分）
- **营销复杂度**（10分）

**评分汇总**：
- 总分 = 市场分析 + 产品分析 + 竞争分析 + 运营分析
- 满分 = 100分

**决策标准**：
- **80-100分**：⭐⭐⭐⭐⭐ 强烈推荐，立即行动
- **60-79分**：⭐⭐⭐⭐ 可以尝试，观察后入
- **40-59分**：⭐⭐⭐ 谨慎考虑，需差异化
- **20-39分**：⭐⭐ 不推荐，风险较大
- **0-19分**：⭐ 放弃，不适合

**一票否决项**（有任何一项立即放弃）：
- ❌ 版权/法律风险高
- ❌ 市场规模 < 1亿
- ❌ 需要囤货 > 10万
- ❌ 违反平台政策

### 阶段2: 竞品发现

使用MCP Web Search Prime搜索竞品：

**搜索策略**：
1. 搜索核心关键词（产品类型 + 主要特征）
2. 搜索品牌名称（如果已知）
3. 搜索电商平台（Amazon、Shopify等）
4. 搜索社交媒体（Instagram、TikTok）

**目标**：
- 发现15+家竞品
- 覆盖不同细分市场
- 包含直接竞品和间接竞品

### 阶段3: 竞品深度分析

从15+竞品中选择5-10家重点分析：

**选择标准**：
- 市场领导者（Top 3）
- 直接竞品（产品相似）
- 创新竞品（有差异化）
- 快速增长者

**分析维度**：
1. **产品定位**
   - 目标用户
   - 核心价值主张
   - 产品特色

2. **价格策略**
   - 价格区间
   - 定价模式
   - 促销策略

3. **内容策略**
   - 网站结构
   - 内容类型
   - 更新频率

4. **流量来源**
   - SEO表现
   - 社媒布局
   - 广告投放

5. **用户评价**
   - 评分分布
   - 常见投诉
   - 用户需求

### 阶段4: 用户洞察

**目标用户画像**：
- **人口统计**：年龄、性别、地域、收入
- **行为特征**：购买习惯、使用场景、决策因素
- **心理特征**：价值观、生活方式、品牌偏好

**用户痛点**：
- 当前方案的不足
- 未被满足的需求
- 愿意支付溢价的地方

**用户需求**：
- 功能需求（核心功能 + 辅助功能）
- 情感需求（安全感、归属感、成就感）
- 社交需求（分享、展示、互动）

### 阶段5: 差异化机会识别

**竞品现状 vs 用户需求**：

| 用户需求 | 竞品现状 | 差异化机会 |
|---------|---------|-----------|
| 需求1 | 竞品A: ... | 机会: ... |
| 需求2 | 竞品B: ... | 机会: ... |
| 需求3 | 竞品C: ... | 机会: ... |

**差异化策略**：
1. **产品差异化**：功能、设计、品质
2. **内容差异化**：信息深度、呈现方式、更新频率
3. **服务差异化**：售前咨询、售后支持、社区运营
4. **品牌差异化**：价值观、故事、个性
