# 内容策略 Skill

基于市场分析结果，制定差异化的内容策略和网站架构。

## 📋 执行流程

### 阶段1: 差异化机会识别

**基于市场分析**：
- 回顾market-analysis的输出
- 竞品内容策略分析
- 用户需求 vs 竞品供给的差距

**识别差异化维度**：
1. **内容主题差异化**
   - 竞品未覆盖的话题
   - 用户搜索但找不到的内容
   - 行业趋势和新兴话题

2. **内容形式差异化**
   - 竞品内容形式分析（文章/视频/指南）
   - 用户偏好内容形式
   - 差异化内容形式建议

3. **内容深度差异化**
   - 竞品内容深度评估
   - 用户对深度的需求
   - 深度 vs 广度的平衡

4. **更新频率差异化**
   - 竞品更新频率
   - 用户期望更新频率
   - 资源vs效果的最佳平衡

**输出**: Top 3-5个差异化机会

### 阶段2: 网站内容结构设计

**网站定位**：
- 目标用户群体
- 核心价值主张
- 与竞品的差异化

**网站架构**：
```
首页
├── 产品/服务
│   ├── 核心产品页
│   ├── 产品对比页
│   └── 使用场景页
├── 解决方案
│   ├── 按用户类型
│   ├── 按使用场景
│   └── 按行业
├── 资源/内容
│   ├── 博客
│   ├── 指南/教程
│   ├── 案例研究
│   └── FAQ
├── 关于我们
│   ├── 品牌故事
│   ├── 团队介绍
│   └── 联系方式
└── 其他页面
    ├── 优惠活动
    ├── 客户评价
    └── 新闻动态
```

**导航结构**：
- 主导航（5-7个）
- 二级导航
- 面包屑导航
- 站内搜索

### 阶段3: 关键页面内容策略

#### 首页策略

**目标**：
- 5秒内传达核心价值
- 引导用户下一步行动
- 建立信任感

**内容元素**：
1. **Hero区域**：
   - 主标题（价值主张）
   - 副标题（补充说明）
   - CTA按钮（主要行动点）

2. **信任元素**：
   - 客户数量/案例
   - 权威认证
   - 媒体报道

3. **核心优势**：
   - 3-5个关键优势
   - 每个优势1-2句话说明

4. **社会证明**：
   - 客户评价
   - 成功案例
   - 使用数据

#### 产品/服务页策略

**每个产品页包含**：
1. **产品概述**（1-2段）
2. **核心功能**（3-5个）
3. **使用场景**（2-3个）
4. **价格信息**
5. **CTA按钮**
6. **相关产品推荐**

#### 博客/资源页策略

**内容分类**：
1. **教育类**（How-to, 指南）
2. **行业洞察**（趋势、分析）
3. **案例研究**（成功故事）
4. **产品更新**（新功能、优化）

**内容规划**：
- 第1个月：核心内容（6-8篇）
- 第2-3个月：扩展内容（20-30篇）
- 第4-6个月：深度内容（50+篇）

### 阶段4: SEO关键词策略

**关键词类型**：

1. **核心关键词**（5-10个）
   - 高搜索量
   - 高相关性
   - 中等竞争度

2. **长尾关键词**（50-100个）
   - 较低搜索量
   - 高转化率
   - 低竞争度

3. **问题型关键词**（20-30个）
   - 用户搜索问题
   - How to, What is, Why
   - 对应FAQ和博客内容

4. **品牌关键词**（10-20个）
   - 品牌词 + 产品词
   - 品牌词 + 功能词
   - 竞品品牌词

**关键词布局**：

| 页面类型 | 主关键词 | 辅助关键词 | 数量 |
|---------|---------|-----------|------|
| 首页 | 品牌词 + 核心词 | 3-5个 | 5-8个 |
| 产品页 | 产品名 + 核心功能 | 5-8个 | 8-12个 |
| 博客 | 长尾关键词 | 3-5个 | 5-10个 |
| 分类页 | 类目词 | 5-10个 | 10-15个 |

**内容-关键词映射**：
- 每个关键词对应1-2个内容
- 优先覆盖高价值关键词
- 建立关键词监控机制
