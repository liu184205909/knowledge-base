# 社媒内容生产 Skill

基于不同平台特性，生产优化的社交媒体内容。

## 📋 执行流程

### 阶段1: 平台选择与策略

**主流平台分析**：

#### Instagram (IG)
- **内容形式**: 图片、Reels短视频、Stories
- **内容长度**: 图片说明2200字符，Reels 60秒
- **受众特点**: 视觉导向、年轻化、高互动
- **最佳内容**: 产品展示、使用场景、用户评价
- **发布频率**: 每天1-2篇
- **Hashtag**: 10-30个

#### Facebook (FB)
- **内容形式**: 文字、图片、视频、直播
- **内容长度**: 63,206字符
- **受众特点**: 年龄层广泛、重视社群
- **最佳内容**: 深度内容、用户故事、活动信息
- **发布频率**: 每天1-2篇
- **Hashtag**: 1-3个

#### TikTok
- **内容形式**: 短视频（15-60秒）
- **内容长度**: 视频描述150字符
- **受众特点**: Z世代、追求新鲜感
- **最佳内容**: 趣味视频、产品演示、挑战赛
- **发布频率**: 每天1-3个
- **Hashtag**: 3-5个

#### 小红书
- **内容形式**: 图文、短视频
- **内容长度**: 1000字（图文），视频5分钟
- **受众特点**: 女性为主、重视种草
- **最佳内容**: 使用心得、产品测评、生活分享
- **发布频率**: 每天1-2篇
- **Hashtag**: 10个左右

#### 微博
- **内容形式**: 文字、图片、视频
- **内容长度**: 140字（基础会员可扩容）
- **受众特点**: 快节奏、重视热点
- **最佳内容**: 热点评论、产品资讯、活动推广
- **发布频率**: 每天3-5条
- **Hashtag**: 2-5个

**平台选择策略**：

| 目标受众 | 推荐平台 | 内容类型 |
|---------|---------|---------|
| 年轻用户（18-34） | Instagram, TikTok | 视觉、短视频 |
| 专业用户（35-54） | Facebook | 深度内容 |
| 女性用户 | 小红书, Instagram | 种草、测评 |
| 大众用户 | 微博 | 热点、资讯 |

### 阶段2: 内容规划（内容日历）

**内容分类**：

1. **产品推广** (30%)
   - 产品介绍
   - 功能演示
   - 使用教程

2. **用户故事** (25%)
   - 用户评价
   - 成功案例
   - 使用场景

3. **行业洞察** (20%)
   - 行业趋势
   - 数据统计
   - 专家观点

4. **互动内容** (15%)
   - 问答
   - 投票
   - 挑战赛

5. **品牌内容** (10%)
   - 品牌故事
   - 团队介绍
   - 企业文化

**内容日历模板**：

```
第1周（7天）:
周一: 产品介绍（IG Reels）
周二: 用户故事（FB长文）
周三: 使用教程（IG图文）
周四: 行业洞察（小红书）
周五: 互动问答（IG Stories）
周六: 产品展示（TikTok）
周日: 品牌内容（微博）

第2周（7天）:
...
```

**发布时间优化**：

| 平台 | 最佳发布时间 | 说明 |
|------|-------------|------|
| Instagram | 工作日12-1点, 7-9点 | 午休、下班后 |
| Facebook | 工作日1-4点 | 下午茶时间 |
| TikTok | 工作日12-3点, 7-9点 | 午休、晚上 |
| 小红书 | 8-10点, 9-11点 | 早上、晚上 |
| 微博 | 工作日9-11点, 3-5点 | 上午、下午 |

### 阶段3: 短内容创作（50-800字）

**不同平台的内容长度**：

#### Instagram
**图文** (2200字符):
```
📸 [吸引眼球的标题]

[第1段] Hook + 问题陈述
[第2段] 解决方案
[第3段] 核心价值
[第4段] CTA

---
🔗 [链接]

🏷️ #hashtag1 #hashtag2 ...
```

**Reels** (150字符):
```
[3-5秒吸引注意]
[核心信息]
[CTA: 链接在bio]

#hashtag1 #hashtag2 #hashtag3
```

#### Facebook
**长文** (500-800字):
```
[标题]

[第1段] 开头故事/数据
[第2-3段] 详细内容
[第4段] 总结
[第5段] CTA

---
📞 联系我们: [信息]
🔗 了解更多: [链接]
```

#### TikTok
**视频描述** (150字符):
```
[吸引注意的文字]

[简短说明]

[CTA]

#标签1 #标签2 #标签3
```

#### 小红书
**图文** (800-1000字):
```
✨ [标题]

[正文内容]
- 第1段: 开头（吸引注意）
- 第2-3段: 核心内容（详细描述）
- 第4段: 使用体验
- 第5段: 总结 + CTA

---
📌 标签: #tag1 #tag2 ...

👉 评论区见！
```

#### 微博
**短文** (140字):
```
【吸引注意的标题】

[核心内容1-2句话]

[链接] #hashtag
```

**内容创作技巧**：

1. **开头3秒法则**:
   - IG: 第一张图最关键
   - TikTok: 前3秒决定留存
   - 文字: 第一句话抓住注意

2. **内容结构**:
   - Hook (吸引注意)
   - Value (提供价值)
   - CTA (行动号召)

3. **互动元素**:
   - 提问
   - 投票
   - 挑战
   - 赠品

### 阶段4: 视觉设计（图片/视频方案）

#### 图片设计

**Instagram**:
- 尺寸: 1080x1080 (方形), 1080x1350 (竖版)
- 风格: 统一的滤镜和色调
- 文字: 简洁、易读
- 品牌: Logo、水印（可选）

**Facebook**:
- 尺寸: 1200x630 (横版), 1080x1080 (方形)
- 风格: 专业、清晰
- 文字: 标题 + 说明
- 品牌: Logo突出

**小红书**:
- 尺寸: 1080x1440 (3:4竖版)
- 风格: 生活化、真实感
- 文字: 图片内文字说明
- 品牌: 自然植入

**微博**:
- 尺寸: 不限（推荐9:16或16:9）
- 风格: 简洁、清晰
- 文字: 配图说明
- 品牌: Logo或水印

#### 视频设计

**TikTok/IG Reels**:
- 时长: 15-60秒
- 格式: 9:16竖版
- 内容:
  - 0-3秒: Hook（吸引注意）
  - 3-10秒: 核心内容
  - 10-15秒: CTA
- 音乐: 热门音乐或品牌音乐
- 字幕: 必须有字幕

**内容类型**：

1. **产品演示** (30%)
   - 功能展示
   - 使用方法
   - 效果对比

2. **教程类** (25%)
   - How-to
   - Tips & Tricks
   - 最佳实践

3. **娱乐类** (20%)
   - 趣味内容
   - 挑战赛
   - 幕后花絮

4. **用户内容** (15%)
   - 用户评价
   - 使用场景
   - 真实反馈

5. **品牌内容** (10%)
   - 品牌故事
   - 团队介绍
   - 企业文化

### 阶段5: 发布优化（最佳时间 + 互动策略）

**发布时间策略**：

**工作日 vs 周末**：
- 工作日: 7-9点, 12-1点, 6-9点
- 周末: 9-11点, 3-5点, 7-10点

**A/B测试**：
- 测试不同发布时间
- 记录互动数据
- 优化最佳时间窗口

**互动策略**：

**发布后1小时内**：
- 回复所有评论
- 点赞用户评论
- 分享到其他平台

**发布后24小时内**：
- 持续互动
- 发布相关Stories
- 引导用户访问网站

**长期互动**：
- 每日监控评论和私信
- 定期发布用户生成内容(UGC)
- 举办互动活动（问答、投票、挑战赛）

**增长策略**：

1. **Hashtag策略**:
   - 混合热门标签和利基标签
   - 创建品牌专属标签
   - 参与热门话题

2. **合作策略**:
   - 与KOL合作
   - 与互补品牌合作
   - 用户生成内容(UGC)活动

3. **付费推广**:
   - Boost优质帖子
   - 精准受众定位
   - A/B测试广告素材
