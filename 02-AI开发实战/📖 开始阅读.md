## 🔄 工作流选择指南

### 决策树

```
你的任务是什么？
│
├── 简单任务（1-3个文件）
│   └── 直接使用 Claude Code（无需特殊工作流）
│
├── 中等复杂度（5-10个文件）
│   ├── 需要严格规范？
│   │   ├── 是 → feature-dev工作流
│   │   └── 否 → Claude Code + 简单流程
│
├── 高复杂度（10+文件）
│   ├── 追求极致自动化？
│   │   ├── 是 → 自动化流水线（需准备好MCP和CSV）
│   │   └── 否 → 继续
│   ├── 团队协作 → feature-dev工作流
│   ├── 独立开发 → Vibe Coding工作流
│   └── 追求深度 → Vibe Coding工作流
│
└── 大规模任务（50+需求，需长期运行）
    └── 自动化流水线（唯一选择）
```
